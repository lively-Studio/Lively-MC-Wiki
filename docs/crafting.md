# 合成

合成是 Minecraft 中最核心的机制之一。通过将材料按照特定图案放入合成方格，可以制作出新物品。

## 基本合成工具

### 工作台

工作台提供 3×3 的合成方格，是制作大多数物品的基础。

**合成配方**：4 个木板（任意木材）

```
[木板] [木板]
[木板] [木板]
```

### 木棍

所有工具的柄，4 个木板可合成。

**合成配方**：2 个木板

```
[木板]
[木板]
```

### 镐

| 材质 | 材料数量 | 合成图案 |

## 盔甲合成

### 头盔

5 个材料排列成倒 U 形（顶排 3 个，中排左右各 1 个）

### 胸甲

8 个材料排成 U 形（除中上外全部填满）

### 护腿

7 个材料排成裤形（左右列满 + 中列上 1 个）

### 靴子

4 个材料（左列上下 + 右列上下）

## 武器与防具

### 弓

```
[木棍] [线] [空]
[木棍] [空] [线]
[木棍] [线] [空]
```
需要：3 木棍 + 3 线

### 箭

```
[燧石]
[木棍]
[羽毛]
```
4 支箭：1 燧石 + 1 木棍 + 1 羽毛

### 盾牌

```
[木板] [铁锭] [木板]
[木板] [木板] [木板]
[空]    [木板] [空]
```
需要：6 木板 + 1 铁锭

### 弩

```
[木棍] [铁锭] [木棍]
[线]   [绊线钩][线]
[空]   [木棍]  [空]
```
需要：3 木棍 + 2 线 + 1 铁锭 + 1 绊线钩
