# 武器

Minecraft Dungeons 中武器分为近战武器和远程武器两大类。

## 近战武器

| 武器 | 类型 | 特点 |
|------|------|------|
| 剑 | 通用 | 中等伤害，连击范围广 |
| 斧 | 慢速 | 高伤害，可击晕敌人 |
| 长柄武器 | 中距离 | 前方扇形攻击 |
| 拳套 | 快速 | 连击积累更快 |
| 双刃斧 | 旋转攻击 | 360° 范围伤害 |
| 镰刀 | 灵魂收集 | 击败敌人获得灵魂能量 |
| 匕首 | 背刺 | 从背后攻击造成额外伤害 |

## 远程武器

| 武器 | 特点 |
|------|------|
| 短弓 | 快速射击，装弹快 |
| 长弓 | 高伤害，强力蓄力射击 |
| 十字弩 | 穿透射击，可击中多个敌人 |
| 重型弩 | 爆炸箭，范围伤害 |
| 散射弩 | 同时发射多支箭矢 |

## 独特武器

独特武器是拥有内置附魔的特殊变体，名称旁有星形标记。例如：钻石剑的独特变体为"钻石剑·独特"，自带等级 I 附魔。
