# Minecraft Dungeons

Minecraft Dungeons 是一款基于 Minecraft 世界观的暗黑破坏神风格动作冒险游戏。

## 基本信息

| 项目 | 详情 |
|------|------|
| 类型 | 动作冒险 / 地下城探索 |
| 开发商 | Mojang Studios / Double Eleven |
| 首发日期 | 2020 年 5 月 26 日 |
| 平台 | Windows, Xbox, PlayStation, Nintendo Switch |
| 终极版 | Minecraft Dungeons: Ultimate Edition（含全部 6 个 DLC） |

## 核心系统

- **装备系统**：近战武器、远程弓弩、盔甲、法器
- **附魔系统**：装备可附魔获得特殊能力，最多 3 个附魔槽位
- **难度等级**：默认 → 冒险 → 天启，共 21 个难度等级
- **DLC 扩展**：隐秘深渊、嚎风峰巅、燃烧下界等 6 个扩展

## 百科分类

- **武器**：近战与远程武器完整图鉴
- **盔甲**：轻甲/中甲/重甲/灵魂甲分类
- **法器**：攻击/防御/辅助三类法器
- **生物**：普通/精英/Boss 全敌人列表
- **附魔**：附魔点数和槽位机制
- **关卡**：主线、隐藏和 DLC 关卡
- **游戏机制**：装备力量、多人合作、难度系统
