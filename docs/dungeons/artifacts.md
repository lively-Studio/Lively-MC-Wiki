# 法器 (Artifacts)

法器是主动使用的特殊道具，消耗灵魂能量后激活。

## 攻击型

| 法器 | 效果 |
|------|------|
| 闪电箭 | 射出灵魂闪电箭 |
| 收割者 | 召唤一把收割灵魂的镰刀 |
| 爆炸弩 | 射出爆炸箭 |
| 腐化种子 | 种下毒藤蔓造成持续伤害 |
| 灵魂灯笼 | 召唤灵魂火焰攻击敌人 |

## 防御型

| 法器 | 效果 |
|------|------|
| 治疗图腾 | 持续恢复生命 |
| 护盾图腾 | 提供护盾 |
| 铁皮护符 | 短时间大幅减伤 |
| 幽灵斗篷 | 短时间隐身并穿墙 |

## 辅助型

| 法器 | 效果 |
|------|------|
| 迅捷之靴 | 短时间大幅加速 |
| 钓鱼竿 | 拉近远处敌人 |
| 美味骨 | 召唤一只狼协助战斗 |
| 魔像工具包 | 召唤一个铁傀儡 |
| 蜜蜂巢 | 召唤蜜蜂攻击敌人 |

## 灵魂机制

击败敌人获得灵魂能量，存储在灵魂槽中。部分法器和武器消耗灵魂来激活特殊效果。
