# 附魔

Dungeons 附魔系统与主游戏不同，通过附魔点数在装备上选择并升级附魔。

## 机制

- 每件装备有 1~3 个附魔槽位
- 每个槽位随机出现 2~3 个可选附魔，选择其一
- 使用附魔点数可升级附魔等级（I → II → III）
- 附魔点数通过升级获得，最高可达到约 240 点

## 通用附魔

| 附魔 | 效果 |
|------|------|
| 锋利 | 增加近战伤害 |
| 多重射击 | 发射更多箭矢 |
| 连锁反应 | 箭矢击中后有概率弹射 |
| 引力 | 将附近敌人拉向攻击点 |
| 灵魂共振 | 增加灵魂收集量 |
| 吸血 | 击杀敌人恢复生命 |

## 盔甲附魔

| 附魔 | 效果 |
|------|------|
| 保护 | 通用减伤 |
| 冷却 | 减少法器冷却 |
| 偏转 | 概率弹开投射物 |
| 荆棘 | 反弹伤害 |
| 雪球 | 定期发射雪球 |

## 独特附魔

独特武器和盔甲拥有专属附魔，不可通过附魔台获得，仅在该物品上出现。
