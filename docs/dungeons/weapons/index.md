# Dungeons 武器

Minecraft Dungeons 的武器系统分为近战武器和远程武器，每件武器可拥有独特变体。

## 近战武器类型

| 类型 | 特点 | 代表武器 |
|------|------|---------|
| 剑 | 通用均衡 | 钻石剑，鹰之剑 |
| 斧 | 慢速高伤 | 双刃斧，火焰之斧 |
| 长柄 | 中距离扇形 | 长矛，三叉戟 |
| 拳套 | 快速连击 | 灵魂之拳，战斗之拳 |
| 镰刀 | 灵魂收集 | 灵魂镰刀，夜魅镰刀 |
| 匕首 | 背刺加成 | 暗影匕首，月光匕首 |

## 远程武器类型

| 类型 | 特点 |
|------|------|
| 短弓 | 快速射击，装弹快 |
| 长弓 | 高伤害，蓄力射击 |
| 十字弩 | 穿透射击，击中多个敌人 |
| 重型弩 | 爆炸箭，范围伤害 |
| 散射弩 | 同时发射多支箭矢 |

## 独特武器

独特武器拥有自带附魔，名称旁有星形标记。例如：
- 钻石剑 → 钻石剑·独特（自带锋利 I）
- 猎人之弓 → 猎人之弓·独特（自带多重射击）
