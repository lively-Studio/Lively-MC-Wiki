# 盔甲

## 盔甲类型

| 类型 | 特点 |
|------|------|
| 轻甲 | 减伤低，移速快，适合远程流派 |
| 重甲 | 减伤高，移速慢，适合坦克流派 |
| 灵魂甲 | 收集灵魂能量，配合灵魂法器 |
| 元素甲 | 提供元素伤害加成或抵抗 |

## 附魔

盔甲可附魔获得以下效果：
- **保护**：通用伤害减免
- **冷却**：法器冷却时间缩短
- **偏转**：概率弹开远程攻击
- **食物储备**：治疗药水效果增强
- **雪球**：定期自动发射雪球眩晕敌人
- **电击**：翻滚后释放电击

## 神器

一些独特盔甲拥有特殊神器词条，如"被击中时爆炸""翻滚后留下火焰轨迹"等。
