# 方块

方块是 Minecraft 中最基本的构成单位。游戏世界几乎完全由各种方块组成，玩家可以采集、放置和使用它们。

## 自然方块

| 安山岩 | 任意生物群系 | 木镐或更好 | 灰色的岩石，可制作磨制安山岩 |
| 深板岩 | Y 0 以下 | 木镐或更好 | 深层的岩石变体，更坚硬 |
| 凝灰岩 | Y 0 以下 | 木镐或更好 | 深层的火山岩方块 |
| 黑石 | 下界玄武岩三角洲 | 木镐或更好 | 下界版的圆石 |
| 玄武岩 | 下界灵魂沙峡谷 | 木镐或更好 | 下界的柱状岩石 |

### 土壤与植被

| 方块 | 说明 |
|------|------|
| 草方块 | 地表最常见的方块，会向相邻泥土扩散 |
| 泥土 | 广泛分布，可用锄头变为耕地 |
| 砂土 | 草不会扩散到上面，用 2 个泥土 + 2 个沙砾合成 |
| 灰化土 | 针叶林生物群系特有，蘑菇可在任意亮度生长 |
| 菌丝体 | 蘑菇岛生物群系特有，蘑菇可在任意亮度生长 |
| 缠根泥土 | 繁茂洞穴生成，下方悬挂垂根 |
| 泥巴 | 红树林沼泽生成，含水 |
| 沙子 | 沙漠和海滩生成，受重力影响 |
| 红沙 | 恶地生物群系特有 |
| 沙砾 | 洞穴和海洋生成，受重力影响，有几率掉落燧石 |
| 黏土 | 河流和沼泽生成，4 个黏土球 → 黏土块 |

## 人造方块

### 木质方块

Minecraft 共有以下木材种类：
- 橡木
- 云杉木
- 白桦木
- 丛林木
- 金合欢木
- 深色橡木
- 红树木
- 樱花木
- 绯红菌柄（下界）
- 诡异菌柄（下界）
- 竹块

每种木材都可以制作：木板、楼梯、台阶、栅栏、栅栏门、门、活板门、按钮、压力板、告示牌。

## 功能性方块

| 方块 | 功能 |
|------|------|
| 工作台 | 提供 3×3 合成网格 |
| 熔炉 | 烧炼物品，将原材料转化为成品 |
| 高炉 | 烧炼矿石速度是熔炉的 2 倍 |
| 烟熏炉 | 烧炼食物速度是熔炉的 2 倍 |
| 附魔台 | 使用经验等级和青金石进行物品附魔 |
| 铁砧 | 修复和重命名物品，组合附魔书 |
| 砂轮 | 移除物品的非诅咒附魔，返还部分经验 |
| 织布机 | 给旗帜添加图案 |
| 切石机 | 将石质方块加工为各种变体 |
| 锻造台 | 将钻石装备升级为下界合金装备 |
| 制图台 | 复制和扩展地图 |
| 酿造台 | 酿造药水 |
| 箱子 | 存储 27 组物品 |
| 木桶 | 存储 27 组物品，上方有方块也可打开 |
| 潜影盒 | 存储 27 组物品，破坏后保留内容物 |
| 末影箱 | 每个玩家独立的 27 格云存储 |

## 红石方块

| 方块 | 功能 |
|------|------|
| 红石粉 | 传输红石信号，最多 15 格 |
| 红石火把 | 持续输出信号或反转信号 |
| 红石块 | 稳定的强红石信号源 |
| 红石中继器 | 延长信号、延迟信号、防止信号倒流 |
| 红石比较器 | 比较信号强度、检测容器状态 |
| 侦测器 | 检测前方方块更新并输出脉冲 |
| 活塞 | 推动最多 12 个方块 |
| 黏性活塞 | 推动和拉回 1 个方块 |
| 投掷器 | 向容器或世界中投掷物品 |
| 发射器 | 发射物品并使用其效果 |
| 漏斗 | 收集和传输物品 |
| 音符盒 | 根据下方方块发出不同音色 |

## 特殊方块

- **信标**：提供正面状态效果，需要矿物金字塔
- **潮涌核心**：提供水下呼吸和水下视野
- **重生锚**：在下界设置重生点
- **磁石**：与指南针绑定，指向固定位置
- **避雷针**：吸引半径 32 格内的闪电
- **锁链**：装饰性方块，可横向或纵向放置
- **光源方块**：创造模式专用，可调节 0-15 亮度
