## 世界生成历史

### 地形生成演变

- **Beta 1.8 前**：极端地形，天域存在
- **Beta 1.8 - 1.6.4**：当前风格地形，海洋风格定型
- **1.7+**：新生物群系，地形更加丰富
- **1.18+**：世界高度翻倍，完全重建的洞穴系统

### 建筑高度演变

| 版本 | 最高高度 | 最低高度 | 矿石分布变化 |
|------|---------|---------|------------|
| Beta 前 | Y=127 | Y=0 | 简单垂直分布 |
| Beta - 1.17 | Y=255 | Y=0 | 三角分布 |
| 1.18+ | Y=319 | Y=-64 | 复杂的深度分布 |

## 重要游戏机制变更

### 战斗系统
- **1.9** (2016)：引入攻击冷却，盾牌取代格挡，双持系统
- **1.14** (2019)：弩加入游戏

### 村民与交易
- **1.8** (2014)：村民繁殖基于意愿
- **1.14** (2019)：村民系统完全重做，职业绑定工作站点
- **1.20.2** (2023)：村民交易的再平衡实验性功能

### 世界生成
- **1.7.2** (2013)：新生物群系系统
- **1.13** (2018)：海洋生物群系全面改造
- **1.16** (2020)：下界生物群系系统
- **1.18** (2021)：洞穴与山地生成完全重建

## 版本号规则

### Java 版
- 格式：`1.MAJOR.MINOR`
- 大版本：增加第二位数字（如 1.16 → 1.17）
- 小版本：增加第三位数字（如 1.20.1 → 1.20.2）

### 快照版本
- 格式：`YYwWWx` 或 `MAJORwYYx`
- 如 `24w21a` 表示 2024 年第 21 周的第一个快照

### 预发布版
- 格式：`1.MAJOR-preREVISION`
- 通常 1-8 个预发布版后发布正式版
