# 角色

## 主角团

### Jesse
玩家可控的主角。性别和外观由玩家在第一集开头选择。性格取决于玩家的对话选择——可以勇敢、善良或幽默。

### Petra
经验丰富的女战士，Jesse 最亲密的盟友之一。擅长战斗，有时过于独立，不愿寻求帮助。

### Axel
Jesse 的老朋友，体型庞大但心地善良。有时过于冲动，但他的忠诚毋庸置疑。害怕失落。

### Olivia
红石技术专家，聪明且务实。她是团队的理性声音，善于分析问题。

### Lukas
原豹猫帮 (Ocelots) 成员，后来加入 Jesse 的团队。作家气质，经常记录团队冒险经历。在团队冲突中扮演调解者角色。

## 反派

### Ivor（第一季）
"方块之序"曾经的成员，携带凋灵风暴制造装置。后来揭示他的动机并非纯粹邪恶。

### The Admin / Romeo（第二季）
拥有创造和改变世界能力的强大存在。作为"管理员"掌管着整个维度。
