# Story Mode 选择与结局

## 第一季关键选择

| 集数 | 关键选择 | 影响 |
|------|---------|------|
| E1 | 救出 Gabriel 还是 Petra | 决定谁更早加入队伍 |
| E4 | 是否信任 Ivor | 影响后续 Ivor 的态度 |
| E5 | 传送时带谁 | 改变队伍配置 |
| E7 | 是否使用命令方块 | 决定最终关卡难度 |
| E8 | 与凋灵风暴的最终决战 | 多种结局分支 |

## 第二季关键选择

| 集数 | 关键选择 | 影响 |
|------|---------|------|
| E1 | 是否帮助 Jack | 决定 Jack 对你的信任 |
| E3 | 监狱中的结盟选择 | 影响后续盟友 |
| E4 | 如何对待 Xara | Xara 可能成为盟友或敌人 |
| E5 | 最终面对管理员 | 决定管理员命运和结局 |

## 结局系统

Telltale 的选择系统意味着：
- 角色存活/死亡取决于你的决定
- 盟友关系影响最终战斗
- 每个玩家可能获得不同的故事体验
- 重玩可探索不同分支

## 跨季影响

第一季的选择会导入第二季（如果平台支持），影响：
- 某些角色的初始态度
- 对话中的回忆和提及
- Jesse 的性格特质
