# 关于

## Lively MC Wiki

这是一个基于 **OWD.K-MD3** 框架构建的 Minecraft 中文维基百科。

## 项目信息

- **框架**：OWD.K-MD3（Open-web-docs.kit）
- **技术栈**：纯静态 HTML + CSS + JavaScript
- **内容驱动**：Markdown
- **渲染引擎**：marked.js + highlight.js
- **设计规范**：Material Design 3

## 内容覆盖

本 Wiki 涵盖以下 Minecraft 内容：

- **方块**：超过 100 种方块的详细说明
- **物品**：工具、武器、盔甲、食物、药水
- **生物**：友好、中立、敌对与 Boss 生物图鉴
- **合成**：常用合成配方与烧炼配方
- **游戏机制**：红石系统、附魔、酿造、战斗、交易
- **版本历史**：Java 版与基岩版历年更新记录
