# 收集品 (Tappables)

Tappables 是 Minecraft Earth 地图上出现的可点击收集物。

## 类型

| 类型 | 地图图标 | 掉落 |
|------|---------|------|
| **树木** | 🟢 绿色圆形 | 原木、木棍、树苗 |
| **岩石** | 🟤 棕色方形 | 圆石、石头、矿物 |
| **宝箱** | 🟡 金色箱形 | 稀有物品、工具、装备 |
| **生物** | 生物图标 | 对应生物及其掉落物 |
| **草地** | 🟩 绿色方形 | 种子、花、食物 |

## 刷新机制

- Tappables 在玩家周围定期刷新
- 稀有度影响刷新概率
- 生物群系影响 Tappable 类型（如水域附近出现更多水生生物）
- 移动距离越大，刷新越多

## 稀有度

- **普通**：树木、岩石、草地（极高刷新率）
- **不常见**：煤炭、铁相关（较高刷新率）
- **稀有**：金、钻石相关（低刷新率）
- **史诗**：附魔装备、独特生物（极低刷新率）
