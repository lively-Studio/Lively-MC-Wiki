# Earth 生物

## 独有生物

| 生物 | 类型 | 特点 |
|------|------|------|
| 泥猪 | 被动 | 喜欢在泥巴中打滚 |
| 彩虹羊 | 被动 | 掉落多彩皮革 |
| 角羊 | 被动 | 头上有独特的角 |
| 蘑菇鸡 | 被动 | 鸡与蘑菇的混合体 |
| 快乐羊驼 | 中立 | 装饰华丽的羊驼 |

## 普通生物

所有 Minecraft 主游戏中的生物在 Earth 中也存在，可通过 Tappables 收集。收集的生物可以放置在 Buildplates 上。

## 敌对生物

冒险模式中出现的敌对生物：
- 骷髅、僵尸、苦力怕
- 洞穴蜘蛛、女巫
- 冒险 Boss（特殊变体）
