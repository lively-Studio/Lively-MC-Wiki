# 溺尸

![Drowned](https://minecraft.wiki/images/Drowned_JE7_BE6.png?format=original)

> **生命值**：20 | **伤害**：2-9（依装备和武器） | **生成**：海洋、河流、僵尸溺水转化
> **掉落**：腐肉、铜锭（基岩版）、三叉戟（稀有）、鹦鹉螺壳（罕见）| **经验**：5

溺尸是水下版的僵尸，在 1.13 水域更新中加入。它可以在水中自然生成，也由僵尸溺水转化而来。部分溺尸手持**三叉戟**，是游戏中唯一掉落三叉戟的生物。

---

## 生成方式

### 自然生成
- 海洋生物群系（亮度 ≤ 7）
- 河流生物群系
- 溺尸可在水底行走，不会受到溺水伤害

### 僵尸转化
- 僵尸头部浸没在水中**超过 30 秒** → 开始转化为溺尸
- 转化过程为抖动动画（15 秒）
- 转化后的溺尸保留僵尸的装备和武器
- 僵尸村民转化后的溺尸无法再被治愈

---

## 战斗行为

### 近战
- 默认手持近战武器（无三叉戟）
- 伤害与同装备的僵尸相同
- 可在水下正常行走和战斗

### 远程（三叉戟溺尸）
- 手持三叉戟的溺尸会投掷三叉戟
- 投掷伤害：8 点
- 投掷冷却约 3 秒
- Java 版：仅 6.25% 溺尸手持三叉戟
- 基岩版：15% 溺尸手持三叉戟
- 持有三叉戟的溺尸有无限弹药

### 陆地行为
- 白天离开水后会在阳光下燃烧（像僵尸一样）
- 在陆地上移动速度很慢

## 三叉戟刷怪场

溺尸农场是获得三叉戟的主要来源：
1. 找到河流生物群系的刷怪点
2. 建造水下平台，加速溺尸生成
3. 使用水电梯收集溺尸
4. 将所有溺尸引导至击杀区
5. 选择带有抢夺 III 剑的玩家手动击杀（自动击杀不会有抢夺加成）

> Java 版中三叉戟掉落率极低（含仅有 6.25% 溺尸会持有三叉戟），拥有抢夺 III 后约 0.58% 概率。建议在基岩版或使用刷怪塔自动收集。

## 你知道吗

1. 溺尸是少数会自然掉落**鹦鹉螺壳**的生物——鹦鹉螺壳是合成潮涌核心的关键材料。
2. 被沉船宝藏室中的溺尸守卫是非常危险的组合——狭窄空间 + 三叉戟远程。
3. 溺尸的攻击在 Java 版中有独特的"水花音效"，与普通僵尸不同。
4. 在早期 1.13 快照中，溺尸曾能在任何水方块中生成——包括村民水井和基地泳池，导致大量玩家抱怨。
