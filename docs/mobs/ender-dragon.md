# 末影龙

![Ender Dragon](https://minecraft.wiki/images/Ender_Dragon_JE6_BE4.png?format=original)

> **类别**：Boss 生物 | **生命值**：200❤ | **生成**：末地（初始有且仅有一只） | **掉落**：龙蛋、大量经验

末影龙是 Minecraft 的最终 Boss。它盘踞在末地主岛上空，是游戏主线流程的终极挑战。击败末影龙是玩家通关的标志，也是访问末地外岛的唯一途径。

---

## 生成

- 玩家首次进入末地时自动生成
- 末地中央有黑曜石柱（带末影水晶）
- 如果末影龙被击败，可以通过在返回传送门四边放置 4 个末影水晶来**复活**

---

## 行为

### 飞行模式
- 在末地主岛上空持续盘旋飞行
- 飞向末影水晶时会受到**治疗**（每半秒 1❤）
- 飞行路径会避开玩家和黑曜石柱

### 攻击方式

| 攻击 | 伤害 | 说明 |
|------|------|------|
| 龙息 | 瞬间 3❤ + 持续伤害 | 紫色粒子云，滞留 3 秒 |
| 翅膀冲撞 | 5❤（普通）/ 10❤（困难） | 将玩家击飞数格 |
| 火球（基岩专属） | 6❤ | Java 版无此攻击 |

### 防御机制
- 末影水晶为其持续回血
- 末影水晶爆炸对其造成 **10❤** 伤害
- 身体各部分有独立的碰撞判定

---

## 战斗攻略

### 第一阶段：摧毁末影水晶
1. 用弓箭射击在铁栏杆保护中的末影水晶
2. 用末影珍珠传送到高处的水晶旁
3. 搭建梯子爬升至水晶高度
4. 破坏所有末影水晶

### 第二阶段：攻击末影龙
1. 末影龙盘旋在传送门上方时，是最佳攻击时机
2. 用**弓箭**射击龙头部（远程最安全）
3. 龙头下降时用**钻石剑/下界合金剑近战**
4. 带有力量 V 的弓需要约 8-12 箭完成任务

### 装备推荐
- 钻石盔甲/下界合金盔甲（保护 IV）
- 力量 V 弓 + 无限箭
- 末影珍珠（用于传送到水晶处）
- 缓降药水（防止摔落伤害）
- 金苹果/治疗药水（应急回血）

## 掉落与奖励

| 奖励 | 说明 |
|------|------|
| 龙蛋 | 落在返回传送门顶部，装饰品 |
| 经验球 | 12000 经验点（首杀） / 500 经验点（复活击杀） |
| 返回传送门 | 可回到主世界 |
| 末地折跃门 | 通往末地外岛的传送门（扔末影珍珠进入） |
| 末地船 | 在外岛可找到鞘翅和龙头 |

## 你知道吗

1. 末影龙是 Notch 亲自设计的生物——但他在完成前就离开了 Mojang。
2. 龙蛋本身没有功能——但用红石比较器可以检测到它的存在。
3. 末影龙的名字来源于"end"（终结）——她是玩家旅程的终点。
4. 复活末影龙会重置所有末影水晶和黑曜石柱。
