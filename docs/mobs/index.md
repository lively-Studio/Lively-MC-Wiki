# 生物图鉴

Minecraft 世界中生活着各种各样的生物（Mob），它们有的友善、有的中立、有的充满敌意。了解每种生物的特性、生成条件和掉落物品是生存的关键。

## 生物分类

### 被动生物（Passive）
不会攻击玩家，即使被攻击也不会反击（除了少数例外）：

| 生物 | 生命值 | 生成环境 | 掉落物 |
|------|--------|---------|--------|
| 村民 | 20❤ | 村庄 | 无 |
| 铁傀儡 | 100❤ | 村庄（玩家建造） | 铁锭、虞美人 |
| 狼 | 8❤ (野生) / 20❤ (驯服) | 森林、针叶林 | 无（可驯服） |
| 猫 | 10❤ | 村庄、沼泽小屋 | 线 |
| 马 | 15-30❤ | 平原、热带草原 | 皮革（可骑乘） |
| 牛 | 10❤ | 草地生物群系 | 皮革、生牛肉 |
| 猪 | 10❤ | 草地生物群系 | 生猪排 |
| 羊 | 8❤ | 草地生物群系 | 羊毛、生羊肉 |
| 鸡 | 4❤ | 草地生物群系 | 羽毛、生鸡肉、鸡蛋 |
| 兔子 | 3❤ | 沙漠、针叶林、雪地 | 兔子皮、生兔肉、兔子脚 |
| 鱿鱼 | 10❤ | 水域 | 墨囊 |
| 发光鱿鱼 | 10❤ | 地下水域 | 荧光墨囊 |
| 海龟 | 30❤ | 沙滩 | 海草、鳞甲 |
| 蜜蜂 | 10❤ | 花林、平原 | 无 |
| 美西螈 | 14❤ | 繁茂洞穴水域 | 无 |

### 中立生物（Neutral）
平时不攻击玩家，但被激怒后会攻击：

| 生物 | 生命值 | 触发条件 | 特殊能力 |
|------|--------|---------|---------|
| 末影人 | 40❤ | 与玩家对视 | 瞬移、搬方块 |
| 猪灵 | 16❤ | 玩家没有金盔甲 | 以物易物 |
| 僵尸猪灵 | 20❤ | 被攻击 | 群体仇恨 |
| 蜘蛛 | 16❤ | 白天变中立 | 夜间主动攻击 |
| 狼 | 8❤ | 被攻击 | 群体狩猎 |
| 北极熊 | 30❤ | 带幼崽被靠近 | 游泳 |
| 海豚 | 10❤ | 被攻击 | 玩家游泳加速 |
| 羊驼 | 15❤ | 被攻击 | 吐口水攻击 |

## 你知道吗

1. "Mob"一词来源于"Mobile Entity"（可移动实体）的缩写。
2. 苦力怕是 Minecraft 最著名的生物——它的诞生源于 Notch 在制作猪模型时把身体和腿的参数搞反了。
3. 村民有 13 种不同的职业，每种都有独特的职业方块和交易类型。
