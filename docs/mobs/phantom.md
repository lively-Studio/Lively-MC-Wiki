# 幻翼

![Phantom](https://minecraft.wiki/images/Phantom_JE3_BE3.png?format=original)

> **生命值**：20 | **伤害**：4-9（依难度） | **生成条件**：玩家超过 3 天不睡觉
> **掉落**：幻翼膜 | **经验**：5

幻翼是一种特殊的飞行敌对生物，仅在玩家超过 **3 个游戏日不睡觉**（约 1 小时实时间）时生成。它从高空俯冲攻击，掉落物可用于修复鞘翅和酿造缓降药水。

---

## 生成条件

| 条件 | 要求 |
|------|------|
| 玩家不睡觉天数 | **≥ 3 天** |
| 玩家上方无实体方块 | 需要直射天空光线 |
| 亮度 | ≤ 7（露天夜间或即将黎明） |
| 时间 | 12000-17750 ticks（夜间至黎明前） |
| 海拔 | Y ≥ 66 |
| 生成概率 | 每 1-2 分钟尝试生成，不睡觉天数越多生成越多 |

> 幻翼每隔一个游戏日生成概率递增——不睡觉 3 天 = 1 只，5 天 = 最多 4 只。

---

## 行为

### 飞行模式
- 在高空（约 Y=150）盘旋
- 寻找不睡觉的玩家
- 不会被方块遮挡限制（能从任何高度俯冲）

### 俯冲攻击
- 发现目标后从高空直冲而下
- 命中后飞回高空准备下一次攻击
- 攻击间隔约 4-8 秒

### 弱点
- **怕猫**：幻翼会避开猫（类似苦力怕）
- **白天燃烧**：日出后暴露在阳光下会燃烧致死
- 幻翼不吃亡灵杀手效果

## 幻翼膜的用途

| 用途 | 配方 |
|------|------|
| 修复鞘翅 | 铁砧：鞘翅 + 幻翼膜（恢复 25% 耐久） |
| 缓降药水 | 粗制药水 + 幻翼膜（酿造台） |

---

## 应对策略

### 最简单：睡觉
只需右键一张床即可重置幻翼生成倒计时——即使只是"躺下起床"不真正睡觉也有效。

### 猫
在基地周围养猫可有效驱赶幻翼。

### 战斗
- 幻翼俯冲时用剑攻击——它俯冲路径为直线，易于命中
- 弓和弩可在远处将其击落
- 利用屋顶/天花板遮挡——幻翼需要直射天空光线来生成

## 你知道吗

1. 幻翼是 2017 年 Minecon Earth 社区投票选出的生物——击败了"深海怪兽"和"火焰怪兽"。
2. 由于幻翼因不睡觉而生成，它常被社区戏称为"Mojang 的睡眠警察"。
3. 幻翼的设计灵感来自现实中的蝠鲼和夜行性昆虫。
4. 许多硬核玩家永久佩戴头盔+不睡觉——幻翼对他们来说只是常驻的夜间威胁。
