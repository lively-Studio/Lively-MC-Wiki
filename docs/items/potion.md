# 药水

![Potion](https://minecraft.wiki/images/Potion_JE1_BE1.png?format=original)

> **类型**：消耗品 | **效果**：各种状态效果 | **可堆叠**：否（每组 1 个） | **可再生**：是
> **酿造工具**：酿造台 | **基础材料**：下界疣 | **稀有度**：常见

药水系统是 Minecraft 中最复杂的制作系统之一。通过酿造台，玩家可以制作具有各种效果的药水——从治疗和力量到隐身和剧毒。

---

## 酿造基础

### 所需设备
- **酿造台**：3 个圆石 + 1 个烈焰棒
- **烈焰粉**：作为酿造台燃料
- **水瓶**：3 个玻璃瓶 + 水源

### 酿造流程
```
水瓶 →（+ 下界疣）→ 粗制药水 →（+ 效果材料）→ 效果药水
                                  →（+ 红石粉）→ 延长版效果药水
                                  →（+ 荧石粉）→ 增强版效果药水
                                  →（+ 火药）   → 喷溅药水
                                  →（+ 龙息）   → 滞留药水
```

## 负面效果药水

| 药水 | 配方 | 效果 |
|------|------|------|
| 剧毒 | 粗制 + 蜘蛛眼 | 中毒 45 秒 |
| 虚弱 | 粗制 + 发酵蛛眼（或直接水瓶 + 发酵蛛眼） | 近战伤害 -4 |
| 迟缓 | 粗制 + 糖 + 发酵蛛眼（或迅捷 + 发酵蛛眼） | 速度降低 15%/级 |
| 伤害 | 粗制 + 闪烁的西瓜片 + 发酵蛛眼（或治疗 + 发酵蛛眼） | 瞬间 6 魔法伤害 |

> 任何正面效果的喷溅药水 + 发酵蛛眼 = 对应负面效果的药水。

## 喷溅药水与滞留药水

### 喷溅药水
- 任何药水 + 火药（酿造台）
- 右键投掷，命中实体或方块后爆炸
- 爆炸半径约 4 格
- 效果持续时间减少（投掷距离影响）

### 滞留药水
- 喷溅药水 + 龙息（酿造台）
- 落地后产生持续 30 秒的粒子云
- 穿过云团的生物获得效果
- 持续时间 = 对应喷溅药水的 1/4

### 药箭
- 8 支箭 + 滞留药水（工作台）→ 8 支药箭
- 箭击中目标时给予对应效果
- 持续时间 = 对应药水的 1/8

## 你知道吗

1. 虚弱药水 + 金苹果 = 治愈僵尸村民——这是获得廉价交易的黄金组合。
2. 滞留伤害药水 II 是少数可以穿透全套下界合金保护的攻击方式之一（无视盔甲减伤）。
3. 酿造台是 Minecraft 中唯一使用化学反响原理的工作方块（现实中的蒸馏/酿造）。
4. 力量 II 药水配合下界合金剑可实现一击击杀穿着全套无保护铁甲的玩家。
