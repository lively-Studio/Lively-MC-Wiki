# 斧

![Axe](https://minecraft.wiki/images/Netherite_Axe_JE3_BE3.png?format=original)

> **类型**：工具/武器 | **耐久度**：59-2031（依材质） | **可堆叠**：否 | **可再生**：是
> **攻击伤害**：7-10（依材质） | **攻击速度**：0.8-1.0 | **稀有度**：常见

斧是 Minecraft 中用于快速砍伐木质方块的工具，同时也是一个强大的近战武器。在 Java 版 1.9+ 中，斧的单次攻击伤害超过同材质剑，但攻击速度较慢。

## 用途

### 砍伐方块
斧可快速破坏所有**木质方块**，包括：
- 原木、去皮原木、木头、去皮木头
- 木板、栅栏、栅栏门、木台阶、木楼梯
- 箱子、工作台、书架、音符盒
- 南瓜、西瓜、南瓜灯
- 可可豆（更快采集）
- 竹块、竹板、竹马赛克

### 给原木去皮
对着原木或木头**右键点击**可将其变为去皮变体。此操作消耗 1 点耐久度。

### 给铜块脱蜡/除锈
对着涂蜡铜块或锈蚀铜块使用斧：
- 脱蜡：移除蜡质保护层
- 除锈：降低一级锈蚀程度

### 武器
斧的单次伤害高于同材质剑（如钻石剑 7 vs 钻石斧 9），可**破盾**（Java 版 1.16+），但攻击速度较慢。

## 你知道吗

1. 在 Java 版战斗测试中，斧的攻击速度曾低至 0.5，但伤害大幅提升。
2. 钻石斧的暴击伤害可一击击杀穿着全套无附魔铁甲的玩家。
3. 用斧给铜块除锈是 1.21 棘巧试炼更新的新功能。
4. 金斧虽然伤害不高，但砍伐速度是所有斧中最快的。
