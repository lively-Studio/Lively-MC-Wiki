# 版本历史

Minecraft 从 2009 年的早期内测版本发展到今天，已经历了超过 15 年的持续更新。2026 年起，版本号采用年份制（26.x），取代了传统的 1.x 格式。

## 你知道吗

1. 版本号 `1.0` 跳过了 `Alpha 1.2.6` 和 `Beta 1.8`——Minecraft 在长达 2 年的公开测试后才正式发布。
2. 1.9 是第一个被玩家社区严重抗议的版本——战斗系统变更导致多年的 PvP 服务器分流。
3. 1.17/1.18 是第一个被拆分为两部分的更新——因为世界生成代码过于复杂。
4. 基岩版最初是手机版（Pocket Edition），后在 1.2 "Better Together" 版本统一了代码库。
5. 2026 年起版本号改为年份制——1.21.11 之后直接跳到 26.1，不再使用 1.x 格式。
6. 从 26.1 开始，每年 3~4 个游戏代（Game Drop）取代了原来的年度大更新模式。
