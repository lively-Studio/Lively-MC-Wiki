## 特点

- **模组支持**：Forge、Fabric、NeoForge 等模组加载器生态
- **服务器**：可自建服务器，支持大量插件
- **快照**：每周发布开发快照预览新特性
- **历史版本**：可自由选择和游玩所有历史版本
- **社区**：全球最大的 Minecraft 修改和创作社区

## 百科分类

- **方块**：87 种方块完整文档（属性、获取、用途、数据值）
- **物品**：13 大类别（工具、武器、盔甲、食物、药水、附魔书等）
- **生物**：21 种生物（被动/中立/敌对/Boss，含战斗策略）
- **合成**：8 大配方类别（工具、武器、盔甲、方块、食物、红石、酿造、烧炼）
- **游戏机制**：14 种核心机制（红石、附魔、交易、战斗、农业、信标等）
- **版本历史**：从 1.0 到 26.2 的完整更新历程
