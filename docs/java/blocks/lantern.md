# 灯笼

![Lantern](../page/images/blocks/lantern/73e414d5.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（镐最快）
> **硬度**：3.5 | **爆炸抗性**：3.5 | **发光**：是（15） | **透明**：否 | **可燃**：否
> **Java ID**：`lantern` / `soul_lantern` | **基岩 ID**：`lantern` (465)
> **地图颜色**：`#545454`

灯笼是 Minecraft 最优雅的照明方块之一，悬挂在上方方块或放置在地面上，亮度 15。灵魂灯笼变体使用蓝色火焰，适合暗黑主题建筑。

---

## 获取

### 合成
```
8 铁粒 + 1 火把 → 灯笼

  铁铁铁
  铁火铁
  铁铁铁
```
- 灵魂火把 → 灵魂灯笼

### 自然生成
- 雪原村庄：路灯装饰
- 堡垒遗迹：走廊照明

### 破坏
- 用镐挖掘 → 掉落自身
- 不用镐 → 无掉落
