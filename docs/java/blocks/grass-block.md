# 草方块

![Grass Block](../page/images/blocks/grass-block/4264a954.png)

> **稀有度**：常见 | **可再生**：否 | **可堆叠**：64 | **工具**：任何（锹最快）
> **硬度**：0.6 | **爆炸抗性**：0.6 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`grass_block` | **基岩 ID**：`grass` (2)
> **音符盒乐器**：竖琴 | **地图颜色**：`#88BB66`（顶）/ `#997744`（侧）

草方块是 Minecraft 最具标志性的方块，与石头并列最早加入的两种方块之一。它的顶部是鲜绿色草皮，侧面有草皮与泥土的渐变纹理。草方块会向相邻泥土方块**蔓延生长**，是主世界地表的"皮肤"。

---

## 获取

### 自然生成
- 主世界几乎所有生物群系的地表层
- 地下洞穴的泥土不会变为草方块（需要阳光）

### 采集
- 精准采集附魔 → 掉落草方块本身
- 无精准采集 → 掉落**泥土**
- 锹挖掘最快

---

## 蔓延机制

草方块会向**相邻 3×3×5 范围内的泥土方块**蔓延：
- 目标必须是**泥土**（普通泥土，非砂土/灰化土）
- 上方有**足够的光照**（亮度 ≥ 9）
- 上方不能有不透明方块遮挡

---

## 外观变化

| 条件 | 改变 |
|------|------|
| 被雪覆盖 | 侧面变为白色（雪草方块） |
| 被实体踩踏 | 变为草径 |
| 上方有不透明方块 | 枯萎→泥土 |
| 绵羊吃草 | 变回泥土（羊吃草行为） |

---

## 生物群系颜色

草方块顶部的绿色随**生物群系温度/湿度**动态染色：
- 热带草原 → 偏黄的橄榄绿
- 丛林 → 鲜艳的翠绿
- 沼泽 → 墨绿色
- 针叶林 → 冷色调绿色
- 平原 → 标准绿色

## 你知道吗

1. 🟢 草方块和石头是 Minecraft **数值 ID 1 和 2**的前两个方块——整个游戏的基础。
2. 🎨 草方块的生物群系着色是 Minecraft 环境美学的核心——同一方块在不同群系呈现不同颜色。
3. 🐑 绵羊吃草行为会消费草方块（变回泥土），然后羊毛重新长出。
4. 🌿 草方块向泥土蔓延是所有自然景观形成的基础——没有它世界就是一片棕色。
