# 红石比较器

![Redstone Comparator](../../assets/images/java/blocks/redstone-comparator/a4c88ba6.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0 | **爆炸抗性**：0 | **发光**：否 | **透明**：是 | **可燃**：否
> **Java ID**：`comparator` | **基岩 ID**：`comparator` (149)
> **地图颜色**：`#FF0000`

红石比较器是红石系统中最精密的元件，能**比较两个信号**的大小（做减法或比较判断），还能**测量容器内容量**并输出对应的信号强度。它是物品分类机、物品检测器和精密红石逻辑电路的核心。

## 获取

### 合成
```
3 石头 + 1 下界石英 + 3 红石火把

  石
  石石
  火下火
```
- 下界石英 = 开采下界石英矿石

### 破坏
- 任何工具均可挖掘，立即掉落

---

## 功能

### 比较模式（默认，前火把不亮）
- 比较背后（A）和侧面（B/C）的信号
- 如果 A ≥ B **且** A ≥ C → 输出 = A
- 否则 → 输出 = 0

### 减法模式（右键切换，前火把亮起）
- 输出 = A - max(B, C)
- 如果 max(B, C) > A → 输出 = 0

### 容器检测
连接到任何容器时：
- 读取容器填充度
- 输出信号 = 0~15
- 一个物品 → 信号 1
- 满容器 → 信号 15

**可检测的容器有**：箱子、熔炉、漏斗、发射器、投掷器、酿造台、音符盒、讲台、木桶、潜影盒、漏斗矿车等

---

## 信号传输

- 比较器 **不会刷新信号**
- 输入 5 → 输出 5（不增强）
- 只做**比较/减法/检测**，不做放大
- 与中继器互补

---

## 经典应用

### 物品分类机
```
漏斗 → 比较器（检测20+个物品）→ 红石 → 锁住上方漏斗
```
- Minecraft 自动存储系统的核心

### 全满检测
```
箱子 → 比较器 → 信号15 → ...
```
- 容器全满时触发动作

### 特定物品数检测
```
漏斗 → 比较器 → 红石（设为特定阈值）
```
- 物品数达到 N 个时触发

### 信号相减
```
比较器(减法模式)：信号A - 信号B = 输出
```

## 你知道吗

1. 🔍 比较器的英文名 "comparator" 来源于电子学——现实中的比较器也是精确测量信号的器件。
2. 📊 比较器测量容器容量的功能让 Minecraft 从"通断开关"级自动化升级到"精确数量"级自动化。
3. 🎯 比较器只会传递较弱信号——输入 7 输出就是 7，不放大。
4. 📚 讲台 + 比较器 = 翻页检测器（每翻一页信号 +1）。
