# 红石中继器

![Redstone Repeater](https://minecraft.wiki/images/Redstone_Repeater_JE3_BE3.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0 | **爆炸抗性**：0 | **发光**：否 | **透明**：是 | **可燃**：否
> **Java ID**：`repeater` | **基岩 ID**：`repeater` (93)
> **地图颜色**：`#FF0000`

红石中继器是红石电路中最常用的元件之一，有三个核心功能：**刷新/延长信号**（将弱的 1 信号刷新为 15）、**引入延迟**（1-4 红石刻可调）、以及作为**二极管**（单向传输）。没有中继器，红石信号最远只能传输 15 格。

---

## 获取

### 合成
```
3 石头 + 2 红石火把 + 1 红石粉

  石石石
  火红火
```
- 石头 = 圆石烧炼

### 破坏
- 任何工具均可挖掘，立即掉落自身

---

## 功能

### 信号刷新
```
输入(1) → 中继器 → 输出(15)
```
- 无论输入信号多弱，输出都是满信号（15）
- 扩展信号传输距离

### 延迟
| 档位 | 延迟 |
|------|------|
| 1 档 | 1 红石刻（0.1 秒） |
| 2 档 | 2 红石刻（0.2 秒） |
| 3 档 | 3 红石刻（0.3 秒） |
| 4 档 | 4 红石刻（0.4 秒） |
- 右键循环切换档位
- 两个拨杆距离越远 = 延迟越长

### 二极管（单向）
- 信号只能从**后面输入、前面输出**
- 侧面无信号交互
- 实现信号方向控制

---

## 经典应用

### 脉冲延长器
```
输入 → 中继器(4档) → 回路 → 中继器 → 输出
```
- 将短暂脉冲延长为持续信号

### 信号延迟链
```
→ 中继器(4) → 中继器(4) → 中继器(4) →
```
- 精确延迟信号（每次 +0.4 秒）

### 锁存器
- 两个中继器互相指向 → 无法改变状态 → 信号被"锁定"
- 侧面中继器激活时锁死

---

## 锁定机制

- 侧面有**另一个激活的中继器**指向本中继器
- 本中继器状态被"锁死"
- 锁定的中继器输出端出现**基岩纹理条**
- 解除侧面信号 → 解锁

## 你知道吗

1. ⏱️ 1 红石刻 = 0.1 秒，游戏中的最小时间单位之一。
2. 🔒 中继器锁定功能（1.4.2 加入）开启了复杂的红石记忆电路时代。
3. 📐 中继器的两个拨杆间距直观表示延迟等级——无需记忆。
4. 🔴 中继器的输出端有一个固定的红石火把——表明它总是输出信号。
