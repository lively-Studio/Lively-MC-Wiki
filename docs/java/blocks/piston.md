# 活塞

![Piston](../../assets/images/java/blocks/piston/c8b8dc40.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0.5（推动时 1.5） | **爆炸抗性**：0.5 | **发光**：否 | **透明**：是 | **可燃**：否
> **Java ID**：`piston` | **基岩 ID**：`piston` (33)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#8B6B4A`

活塞是红石机械运动的基础方块，能在红石信号激活时**推出一个方块**，信号消失后**收回**（但不带回方块）。它是自动门、隐藏楼梯、农场收割机等所有红石运动机械的基础。

- 任何工具均可挖掘，掉落自身

## 工作原理

### 推动规则
- 激活时**推出 1 格**（活塞臂会推动前方方块）
- 最多推动 **12 个方块**（呈直线排列）
- 信号消失 → 活塞臂缩回（不拉回方块）

### 不可推动的方块
| 类别 | 示例 |
|------|------|
| 高硬度方块 | 黑曜石、哭泣的黑曜石、远古残骸 |
| 容器类 | 箱子、熔炉、发射器等（基岩版可推） |
| 固定方块 | 基岩、末地传送门框架、命令方块 |
| 延长活塞 | 已伸出的活塞 |

### 推动限制
- 最多同时推动 **12 个相连方块**
- 任何被推方块不能超出世界边界
- 被粘液块粘连的方块有额外限制

## 黏性活塞

![Sticky Piston](../../assets/images/java/blocks/piston/622ed952.png)

### 合成
```
1 粘液球 + 1 活塞
```
- 粘液球：击败史莱姆掉落

### 特性
- 激活时推出方块（与普通活塞相同）
- **信号消失时拉回方块**（与普通活塞的关键区别）
- 1 红石刻 脉冲 → 推出方块但不拉回（方块留在推出位置）
  - 这是 Java 版中的 **"0 tick" 方块的转移法**（方块瞬移）

## 经典应用

### 自动门
```
压力板/按钮 → 红石 → 活塞 ← 门方块
```

### 隐藏楼梯
```
活塞 → 收回方块 → 露出楼梯

→ 活塞 → 推出方块 → 隐藏楼梯
```

### 自动农场
```
侦测器（检测作物成熟）→ 活塞（推出水流收割）
```

### TNT 大炮
```
TNT（推进药）+ 活塞 ← 红石 → TNT（弹药）

活塞推出弹药 TNT → 飞行
```

## 你知道吗

1. 🔧 活塞由 Minecraft 玩家 Hippoplatimus 提出——Mojang 采纳了他的 Mod 创意。
2. 🎯 1 红石刻脉冲 + 黏性活塞 = 方块瞬移（0-tick 方块的转移法）——Java 版特色。
3. 🏗️ 活塞是 Minecraft 中"体积最大"的技术方块——Piston Head（活塞头）是一个独立的技术实体。
4. 💡 活塞不能推动自身或另一个激活的活塞——防止无限循环。
