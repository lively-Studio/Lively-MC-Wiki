# 闪长岩

![Diorite](../images/blocks/diorite/9b3fea94.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：1.5 | **爆炸抗性**：6 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`diorite` | **基岩 ID**：`stone` (数据值: 3)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#BABABA`

闪长岩是 1.8 Bountiful Update 加入的三种岩石变体之一，与花岗岩和安山岩并列。其灰白色带深色斑点的纹理灵感来源于现实世界中的闪长岩矿物——主要由斜长石和角闪石构成。在 MC 社区中，闪长岩因其"鸟粪石"的外号而颇具争议——爱好者认为它是最优雅的石材，反对者则认为它看起来像鸟的排泄物。

---

## 获取方式

### 自然生成
闪长岩在主世界以矿脉形式生成：
- **高度范围**：Y=0 到 Y=79
- **矿脉规模**：与花岗岩和安山岩相似，每矿脉含十至数十个方块
- **分布特点**：在所有主世界生物群系均有分布，无特定偏好——不像花岗岩倾向于山地
- 少量生成于**海底遗迹**和**古迹废墟**

### 合成
通过工作台合成（可再生）：
- 2 圆石 + 2 下界石英 → **2 闪长岩**（无序合成，任意位置均可）

这个配方非常实用——圆石是无限的，下界石英可通过猪灵以物易物或挖掘下界石英矿石获得。

### 战利品
| 来源 | 结构 | 容器 | 数量 | 概率 |
|------|------|------|------|------|
| 磨制闪长岩 | 村庄 | 石匠箱子 | 1 | 37.7% |

### 破坏
- 用镐（无精准采集）→ 掉落**闪长岩本身**（不同于石头掉圆石）
- 用带精准采集的镐 → 同样掉落闪长岩
- 不用镐 → 无掉落
- 金镐挖掘最快：0.2 秒

## 用途

### 基础合成
- 4× 闪长岩（2×2）→ 4× **磨制闪长岩**

### 合成其他岩石
闪长岩是"岩石三角"的核心：
- 闪长岩 + 圆石 → **2 安山岩**
- 闪长岩 + 下界石英 → **2 花岗岩**

这意味着只要有足够多的闪长岩 + 圆石或下界石英，就可以**无限产出**安山岩和花岗岩。

### 切石机加工
- 磨制闪长岩
- 磨制闪长岩台阶
- 磨制闪长岩楼梯
- 磨制闪长岩墙
- 磨制闪长岩压力板
- 磨制闪长岩按钮

### 村民交易
- 石匠学徒 → 出售磨制闪长岩
- 石匠大师 → 16 闪长岩换 1 绿宝石

## 你知道吗

1. ⚪ 磨制闪长岩是 Minecraft 中**最白的砖块之一**（与石英块和白色混凝土竞争），在现代极简风格建筑中非常受欢迎。
2. 🐦 "鸟粪石"的外号起源于社区对闪长岩斑驳纹理的调侃，但实际上磨制闪长岩的纯白光滑质感完全消除了这个问题——许多人不知道闪长岩的"素颜"和"精装"差距如此之大。
3. 🔺 闪长岩是"岩石三角"的关键中枢：它可以合成花岗岩（+下界石英）和安山岩（+圆石），是唯一能合成另外两种岩石的材料。
4. 🌍 在现实世界中，闪长岩是一种中性深成岩，颜色介于花岗岩和辉长岩之间——Mojang 的设计非常贴合地质学定义。
5. 📦 石匠村民同时出售磨制花岗岩和磨制闪长岩，使建筑玩家可以稳定获取这两种精加工石材。
