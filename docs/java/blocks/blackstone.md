# 黑石

![Blackstone](../images/blocks/blackstone/844cc69a.png)

> **稀有度**：常见（下界）| **可再生**：是 | **可堆叠**：64 | **工具**：木镐+ | **硬度**：1.5 | **爆炸抗性**：6
> **Java ID**：`blackstone` | **基岩 ID**：`blackstone` (528)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#1A1A1A`

黑石是 1.16 下界更新加入的"下界版圆石"，可以完全替代圆石的所有合成配方。其深黑带灰色脉络的外观非常适合暗色系和哥特风格建筑。

---

## 获取方式

### 自然生成
- **玄武岩三角洲**大量生成——是整个下界最集中的黑石来源
- 下界堡垒中少量出现
- 废弃传送门（下界侧）中生成

### 以物易物
- 与猪灵用金锭交易有几率获得黑石（可再生来源）

### 破坏
- 任何镐 → 掉落黑石本身，不需要精准采集

---

## 用途

### 圆石替代
黑石可**完全替代圆石**用于以下配方：
- 熔炉（8 黑石围成方形）
- 酿造台（3 黑石 + 1 烈焰棒）
- 投掷器、发射器
- 黑石工具（石镐、石剑等，属性与石质工具相同）

### 建筑变体
- 磨制黑石（4 黑石 2×2）
- 黑石砖（4 磨制黑石 2×2）
- 裂纹黑石砖（黑石砖烧炼）
- 錾制磨制黑石
- 上述所有变体的**台阶、楼梯、墙**

### 镀金黑石
下界堡垒中自然生成的特殊变体，挖掘时有几率掉落**金粒**（时运附魔提高概率）。
