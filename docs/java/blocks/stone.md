# 石头

![Stone](../page/images/blocks/stone/99f0154b.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：1.5 | **爆炸抗性**：6 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`stone` | **基岩 ID**：`stone` / 数字ID `1`
> **音符盒乐器**：低音鼓 | **地图颜色**：`#707070`

石头是 Minecraft 最早加入的两个方块之一（与草方块并列），是主世界 Y=0 以上最丰富的固体方块。它与草方块一起象征着这个游戏的基本构成。

---

## 获取方式

### 自然生成
石头在主世界大量生成于草方块、泥土和沙砾等图层以下 1~5 层处，构成地壳的主体。在 Y=8 开始逐渐过渡为深板岩，到 Y=0 以下则完全被深板岩取代。

石头也会作为以下结构的组成部分自然生成：
- **雪屋地下室**：墙壁和天花板的一部分
- **废弃传送门（主世界）**：部分传送门框架
- **古迹废墟**：结构墙体
- **试炼密室**：与蜘蛛刷怪笼相连的房间中
- **虚空超平坦预设起始平台**：33×33 的石头平台

### 作为战利品
| 来源 | 结构 | 容器 | 数量 | 概率 |
|------|------|------|------|------|
| 石头 | 村庄 | 石匠箱子 | 1 | 37.7% |

### 可再生生成（刷石机）
当**熔岩流到水源上方**时，水会被替换为石头。这是可再生石头的主要来源。

![Stone generator](../page/images/blocks/stone/7fa63938.gif)

### 破坏
- 用镐（无精准采集）→ 掉落**圆石**
- 用带有精准采集附魔的镐 → 掉落**石头本身**
- 不用镐挖掘 → 无掉落

### 烧炼
- 圆石 + 任何燃料 → 石头

## 用途

### 合成
- 4× 石头 → 4× 石砖
- 3× 石头 → 6× 石台阶
- 6× 石头（L 形）→ 4× 石楼梯
- 1× 石头 → 石按钮
- 2× 石头 → 石压力板

**红石组件**：
- 石头 + 红石粉 + 红石火把 → 红石中继器
- 石头 + 下界石英 + 红石火把 → 红石比较器
- 石头 + 铁锭 → 切石机

### 切石机加工
石头可通过切石机精确加工为：
- 石台阶、石楼梯、石砖台阶、石砖楼梯、石砖墙
- 錾制石砖、圆石、圆石变体

### 烧炼
- 石头 + 燃料 → **平滑石头**

### 蠹虫
蠹虫可以钻入石头，将其变为**被虫蚀的石头**。

### 酿造
- 石头 + 水瓶 → **平凡药水**
- 石头 + 粗制药水 → **寄生药水**（1.21.5+，Java版独有）

### 交易
石匠村民（学徒等级）：
- 20 个石头 → 1 个绿宝石
- 概率 100%，库存最多 16 次

## 历史

### pre-Classic
- **Cave game tech test**：石头首次加入，内部代号 "rock"。纹理来自 Markus Persson (Notch) 的前作 *RubyDung*，底部纹理垂直镜像。

### Java 版经典时期
- **rd-20090515**：更改纹理，旧纹理转为圆石使用。
- **0.26 SURVIVAL TEST**：苦力怕爆炸不再破坏石头；挖掘石头掉落圆石。

### Indev 至 Beta
- **Indev 0.31**：圆石可烧炼为石头。
- **Infdev 20100617**：熔岩与水接触可再生生成石头。
- **Beta 1.3**：石台阶配方切换为使用石头；可合成红石中继器。
- **Beta 1.8**：可合成石砖。

### Java 版正式版
- **1.0.0**：精准采集可获得石头本身。
- **1.5**：加入红石比较器。
- **1.8**：纹理添加随机旋转和镜像；花岗岩等变种使用独立 ID。
- **1.14**（村庄与掠夺）：纹理更新；可烧为平滑石头；村庄石匠箱概率出现。
- **1.16**（下界更新）：主世界废弃传送门中生成。
- **1.20**（足迹与故事）：生成于古迹废墟。
- **1.21**（试炼密室）：生成于试炼密室；可酿造寄生药水。

### 基岩版
- **Alpha 0.1.0**：加入石头（旧纹理）。
- **Alpha 0.2.0**：纹理更新。
- **Alpha 0.3.0**：圆石可烧为石头。
- **1.10.0**：纹理更新为当前样式；生成平滑石头配方。

## 你知道吗

1. 石头和草方块是 Minecraft 最早的两种方块（数值 ID 分别为 1 和 2）。这个排序反映在最早的 `terrain.png` 纹理图集上。
2. 最初的石头纹理由 Markus "Notch" Persson 从他的前一款游戏 *RubyDung* 中直接搬来。
3. 挖掘石头掉落圆石而非石头本身，暗示了游戏中"工艺"的概念——你需要精准采集技术才能完整获取原石。
4. 石头是极少数可以通过可再生方式（熔岩+水）无限生产的固体方块之一。
5. 石头在试炼密室中与蜘蛛网一起出现，暗示附近有蜘蛛刷怪笼。
6. 自 1.14 纹理更新以来，石头的纹理明显变得更柔和、更圆润。
