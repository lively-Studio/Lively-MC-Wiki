# 灰化土

![Podzol](../../assets/images/java/blocks/podzol/4400483c.png)

> **稀有度**：不常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（锹最快）
> **硬度**：0.5 | **爆炸抗性**：0.5 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`podzol` | **地图颜色**：`#72582A`

灰化土是针叶林独有的土壤变体，覆盖一层褐灰/橙色的腐殖质层。蘑菇可在任意亮度下生长在灰化土上，无需菌丝体。

---

## 获取

### 自然生成
- 巨木针叶林、原始针叶林
- 竹林

### 可再生
- 巨型云杉长大后 → 周围泥土变为灰化土

---

## 特性

- 蘑菇可在任何光照下的灰化土生长
- 精准采集才能获取（否则掉泥土）
