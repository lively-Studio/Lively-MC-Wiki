# 末影箱

![Ender Chest](../page/images/blocks/ender-chest/360e911c.png)

> **稀有度**：稀有 | **可再生**：否 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：22.5 | **爆炸抗性**：600 | **发光**：是（7） | **透明**：否 | **可燃**：否
> **Java ID**：`ender_chest` | **基岩 ID**：`ender_chest` (130)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#111111`

末影箱是 Minecraft 最"魔法"的存储方块——所有末影箱**共享同一个存储空间**，每个玩家拥有独立的 **27 格云存储**。你在地图东端的末影箱放入物品，走到西端打开另一个末影箱就能取出。物品在玩家间不可见。

---

## 获取

### 合成
```
8 黑曜石 + 1 末影之眼

  黑黑黑
  黑眼黑
  黑黑黑
```
- **末影之眼** = 烈焰粉 + 末影珍珠
- 共需 8 个黑曜石

### 破坏
- 必须用**精准采集附魔的镐**挖掘 → 掉落末影箱本身
- 无精准采集 → **掉落 8 个黑曜石**（末影之眼不返还）
- 不用镐 → 无掉落

---

## 工作原理

### 共享空间
```
末影箱 A → 玩家甲的 27 格空间
   ↓
末影箱 B → 同一个 27 格空间（玩家甲看到的是同一批物品）
```
- 所有维度（主世界、下界、末地）的末影箱共享
- 每个玩家拥有**独立的 27 格空间**
- 玩家甲放入的物品 → 玩家乙看不到

---

## 特性

### 优点
- **跨维度云存储**：全地图即时同步
- 安全性：其他玩家看不到你的物品
- 可作为"无线传输机"：放入东西 → 走到目的地取出

### 限制
- 仅 27 格（不能像潜影盒那样扩展）
- 需要精准采集才能回收（否则损失末影之眼）
- 不能自动用漏斗输入/输出

---

## 粒子效果

- 末影箱周围发出**末影粒子**
- 粒子颜色：深紫色，类似末影人传送效果
- 打开时盖子升起（和普通箱子一样的动画）

## 你知道吗

1. 🌌 "末影"（Ender）主题贯穿末影珍珠、末影之眼、末影人（Enderman）和末影箱——都是紫色+传送相关。
2. 📡 末影箱是 Minecraft 中最接近"云存储"的概念——在任何地方就能访问同一批物品。
3. 💎 破坏末影箱必须精准采集——否则末影之眼会消失（相当于损失一个末影珍珠 + 一个烈焰粉）。
4. 🚫 漏斗不能与末影箱交互——防止自动化滥用。
