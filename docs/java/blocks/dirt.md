# 泥土

![Dirt](../page/images/blocks/dirt/8428ff1e.png)

> **稀有度**：极其常见 | **可再生**：是（砂土+锄头） | **可堆叠**：64 | **工具**：任何（锹最快）
> **硬度**：0.5 | **爆炸抗性**：0.5 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`dirt` | **基岩 ID**：`dirt` (3)
> **音符盒乐器**：竖琴 | **地图颜色**：`#997744`

泥土是主世界最常见的方块之一，充斥于草皮层以下的浅层地质。它是农业的基础（耕地由泥土制成），也是草方块、菌丝和灰化土的"基材"。

---

## 获取

### 自然生成
- 主世界 Y>0 的绝大多数区域
- 草地下方 1-5 层即开始出现

### 采集
- 锹挖掘最快
- 任何工具均掉落自身

---

## 变体

泥土有多个密切相关的变体：

| 变体 | 特性 |
|------|------|
| 草方块 | 泥土顶部有草皮（会蔓延） |
| 砂土 | 草不会蔓延到上面 |
| 灰化土 | 针叶林特有（巨木针叶林） |
| 菌丝体 | 蘑菇岛特有（紫色菌丝） |
| 缠根泥土 | 苍郁洞穴生成 |
| 泥巴 | 泥土+水 → 泥巴 |
| 耕地 | 泥土+锄头 → 耕地 |

---

## 用途

- **耕地**：用锄头右键 → 种植作物
- **草径**：锹右键草方块 → 草径
- 合成泥巴（水+泥土）
- 填充/平整地面

## 你知道吗

1. 🧱 泥土是 Minecraft 存储量最多的方块——每个玩家都有一座"泥土山"（挖矿副产品）。
2. 🌱 用锄头把泥土变为耕地后，如果附近 4 格内无水，耕地会变回泥土。
3. 📦 2 泥土 + 2 沙砾 → 4 砂土——一种不需要光照也不长草的泥土。
