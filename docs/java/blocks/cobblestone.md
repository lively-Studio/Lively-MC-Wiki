# 圆石

![Cobblestone](../images/blocks/cobblestone/4e10bf92.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：2 | **爆炸抗性**：6 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`cobblestone` | **基岩 ID**：`cobblestone` (数字ID: 4)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#707070`

圆石是 Minecraft 中使用最广泛的基础建材之一。它通过开采石头获得，是刷石机的直接产物。圆石最具辨识度的纹理——斑驳的灰色碎石外观——自游戏最早的 pre-Classic 阶段就存在了。

---

## 获取方式

### 自然生成
圆石在大量结构中自然生成：
- **地牢**：墙壁和地面由圆石构成，是玩家遇到的第一种结构方块之一
- **要塞**：大部分墙壁和支柱都由圆石构建（图书馆除外）
- **丛林神庙**：整体结构的主要材料
- **林地府邸**：基础结构材料
- **掠夺者前哨站**：瞭望塔主体采用圆石
- **村庄**：铁匠铺墙壁、教堂、部分小屋使用圆石
- **废弃矿井**：少量出现
- **海底遗迹**：极少量

### 挖掘石头
- 用镐（无精准采集附魔）挖掘石头 → 掉落圆石
- 这是圆石最主要、最基本的获取方式

### 刷石机
流动的熔岩触碰水源时有两种情况：
- **熔岩流到水源上方** → 生成**石头**
- **熔岩流触碰流动的水** → 生成**圆石**

通用的刷石机设计：
```
熔岩         水
  ↓          ↓
[熔岩] [圆石] [水]
```

在 1 格宽的水道中间让熔岩与水在侧面接触即可无限产出圆石。

## 用途

### 基础合成
圆石是 Minecraft 中最重要的合成材料之一：

**功能性方块**：
- 8 圆石（围成方形）→ **熔炉**
- 3 圆石 + 1 烈焰棒（竖排中间）→ **酿造台**
- 7 圆石（U 形）+ 1 红石粉（中间）→ **投掷器**
- 7 圆石（U 形）+ 1 弓（中间）+ 1 红石粉（中间下方）→ **发射器**
- 6 圆石 + 2 红石粉 + 1 下界石英 → **侦测器**

**石质工具**：
| 工具 | 配方 (顶排 + 两根木棍) |
|------|------------------------|
| 石镐 | 3 圆石 |
| 石斧 | 3 圆石（右上角） |
| 石锹 | 1 圆石（正中） |
| 石锄 | 2 圆石（左上+左中） |
| 石剑 | 2 圆石（中上+中中）+ 木棍 |

**其他**：
- 2 圆石 + 2 下界石英 → 2 闪长岩
- 1 圆石 + 1 闪长岩 → 2 安山岩
- 6 圆石 + 1 铁锭 → 切石机（1.21+ 配方）
- 3 圆石 + 6 铁锭 → 活塞
- 2 圆石 + 2 木棍 → 拉杆

### 烧炼
- 圆石 + 燃料 → **石头**

### 切石机加工
- 1 圆石 → 圆石台阶
- 1 圆石 → 圆石楼梯
- 1 圆石 → 圆石墙
- 1 圆石 → 石台阶（1.21+）

### 蠹虫
- 蠹虫钻入圆石 → **被虫蚀的圆石**

## 你知道吗

1. 🎨 圆石的纹理是 Minecraft 最早的纹理之一——可以追溯到 2009 年的 pre-Classic 阶段，比很多后来加入的方块都要古老。
2. ♻️ 圆石是最容易**无限可再生**的方块：不需要复杂的装置，最简单的刷石机只需 1 桶熔岩 + 1 桶水就能无限产出。
3. 🏰 地牢是玩家在生存模式中第一次遇到自然生成的圆石结构，让无数新手玩家感到惊叹和恐惧。
4. ⚡ 圆石的硬度 (2.0) 高于石头 (1.5)，这意外地使圆石成为比石头更耐爆破的建材——虽然两者爆炸抗性相同。在旧版本中，圆石曾是防苦力怕爆炸的首选快建材料。
5. 🔧 圆石几乎是所有进阶机械（熔炉、酿造台、投掷器、发射器、活塞）的基础材料，可以说是"科技树的基石"。
6. 🐛 被虫蚀的圆石外观与普通圆石完全相同，但挖掘速度明显更快，且会释放蠹虫——在要塞探索时需要格外小心。
