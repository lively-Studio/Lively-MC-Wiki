# 砂轮

![Grindstone](../page/images/blocks/grindstone/7bf91268.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：2 | **爆炸抗性**：2 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`grindstone` | **基岩 ID**：`grindstone` (450)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#545454`

砂轮是 1.14 加入的**去附魔**工具，能移除物品上的所有附魔（返还部分经验），以及**修复物品**（两个同类损坏物品合并）。武器匠村民的工作站点方块。

---

## 获取

### 合成
```
2 木棍 + 1 石台阶 + 2 木板

  棍  棍
    台
  板  板
```

### 自然生成
- **村庄武器匠小屋**：固定生成 1 个

### 破坏
- 用镐挖掘 → 掉落自身
- 不用镐 → 无掉落

---

## 功能

### 去附魔
- 将有附魔的物品放入砂轮
- 移除**所有附魔**（包括诅咒！）
- 返还部分经验

### 修复
- 两个同类损坏物品 → 1 个物品（耐久相加 + 5% 加成）
- 移除所有附魔（不同于铁砧，不保留附魔）

### 经验返还
- 去附魔时按附魔等级返还经验
- 不给诅咒附魔返还经验

---

## 与铁砧的对比

| 特性 | 砂轮 | 铁砧 |
|------|------|------|
| 去附魔 | ✓（移除所有） | ✗ |
| 修复 | ✓（但移除附魔） | ✓（保留附魔） |
| 附魔合并 | ✗ | ✓ |
| 重命名 | ✗ | ✓ |
| 经验消耗 | 无（去附魔返还） | 有 |
| 损坏 | 不损坏 | 会损坏 |

## 你知道吗

1. 🔄 砂轮可以移除**所有附魔**——包括铁砧都去不掉的诅咒附魔（消失诅咒、绑定诅咒）。
2. 🛠️ 砂轮修复不消耗经验反而返还经验——让它比铁砧早期修复更实用。
3. ⚔️ 砂轮的外形是一块磨刀石（grindstone），历史上用于磨砺刀剑——与其修复武器的功能完美吻合。
