# 投掷器

![Dropper](../page/images/blocks/dropper/5775caae.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：3.5 | **爆炸抗性**：3.5 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`dropper` | **基岩 ID**：`dropper` (125)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#545454`

投掷器是发射器的"纯投掷"版本——只能**抛出物品实体**或将物品**射入前方容器**，没有发射器的特殊功能。它是物品传输链和垂直物品电梯的关键组件。

---

## 获取

### 合成
```
7 圆石 + 1 红石粉

  圆圆圆
  圆圆圆
  圆红圆
```
- 配方**不需要弓**（与发射器的关键区别）

### 破坏
- 用镐挖掘 → 掉落自身及内部物品

---

## 功能

### 投掷模式
- 红石激活时，**随机投出**内部 1 个物品为掉落物
- 物品在前方 1 格出现

### 传输模式
- 前方有容器（箱子、漏斗、另一个投掷器）→ 将物品**射入该容器**
- 这是物品垂直运输的主力机制

---

## 物品传输链

### 垂直物品电梯
```
投掷器 ↑（指向上方投掷器）
投掷器 ↑
投掷器 ↑
  ...
箱子 ← 输出
```
- 投掷器链 + 红石时钟 = 垂直物品传输
- 配合漏斗实现任意方向物品运输

### 水流传输
```
投掷器 → 水流 → 漏斗/箱子
```
- 投掷器投出物品到水流中
- 水流将物品运至目标位置

## 你知道吗

1. 🔗 投掷器链是 Minecraft 自动化中最优雅的物品传输方案——不需要水、冰或漏斗链。
2. 📐 投掷器没有发射器的"口部"动画——它通过简单的开口来投出物品。
3. 🔄 投掷器 + 侦测器时钟 = 每 2 红石刻（0.2 秒）传输一个物品，速度为 5 个/秒。
