# 锻造台

![Smithing Table](../page/images/blocks/smithing-table/329df707.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（斧最快）
> **硬度**：2.5 | **爆炸抗性**：2.5 | **发光**：否 | **透明**：否 | **可燃**：是
> **Java ID**：`smithing_table` | **基岩 ID**：`smithing_table` (457)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#8B6B4A`

锻造台是升级装备的核心方块，用于将**钻石装备升级为下界合金装备**（1.16 加入），以及使用锻造模板为盔甲添加**纹饰**（1.20 加入）。工具匠村民的工作站点方块。

## 功能

### 下界合金升级（1.16+）
```
钻石装备 + 下界合金锭 + 升级模板 → 下界合金装备
```
- 保留所有附魔、耐久度和名称
- **1.20 后必须使用"下界合金升级"锻造模板**

### 盔甲纹饰（1.20+）
```
盔甲 + 纹饰模板 + 矿物/宝石 → 带纹饰的盔甲
```
- 纯装饰效果，不影响属性
- 10 种模板 × 10 种颜色材料 = 数百种组合
