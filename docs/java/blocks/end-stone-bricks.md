# 末地石砖

![End Stone Bricks](../page/images/blocks/end-stone-bricks/433169e3.png)

> **稀有度**：不常见 | **可再生**：否 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：3 | **爆炸抗性**：9 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`end_stone_bricks` | **地图颜色**：`#E0E090`

末地石砖是末地石的加工变体，由末地石合成。末地城大量使用作为建筑材料。与石砖不同，末地石砖没有苔藓或裂纹变体。

---

## 获取

### 合成
- 4 末地石 → 4 末地石砖

### 自然生成
- 末地城（墙体、塔楼）
