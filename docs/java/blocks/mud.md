# 泥巴

![Mud](../page/images/blocks/mud/d701bb31.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（锹最快）
> **硬度**：0.5 | **爆炸抗性**：0.5 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`mud` | **地图颜色**：`#52493A`

泥巴是 1.19 加入的沼泽变体方块，通过水+泥土（水瓶右键）获得。泥巴上方实体移动速度略慢。可自然脱水为黏土。

---

## 获取

### 制作
- 水瓶右键泥土 → 泥巴

### 自然生成
- 红树林沼泽

### 用途
- 泥巴 + 滴水石锥下方 → 黏土（可再生黏土来源）
- 合成泥砖（泥巴烧炼为泥砖）
