# 漏斗

![Hopper](../images/blocks/hopper/8db9c56d.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：3 | **爆炸抗性**：4.8 | **发光**：否 | **透明**：是 | **可燃**：否
> **Java ID**：`hopper` | **基岩 ID**：`hopper` (154)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#545454`

漏斗是 Minecraft 自动化系统的**核心组件**，能从上方收集掉落物并从底部输出到容器。它是物品分类机、自动农场、经验农场等所有自动化设施的基石。没有漏斗，就没有工业化的 Minecraft。

---

## 获取

### 合成
```
5 铁锭 + 1 箱子

  铁  铁
  铁箱铁
    铁
```
- 铁锭排列成 V 形，箱子在中间

### 破坏
- 用镐挖掘 → 掉落自身（含内部物品）
- 不用镐 → 无掉落

---

## 工作原理

### 物品传输
```
    输入方向
       ↓
  ┌─────────┐
  │  漏斗   │  → 输出方向
  └─────────┘
```

| 方向 | 功能 |
|------|------|
| **上方** | 从上方容器（箱子、熔炉等）吸取物品 |
| **下方** | 将物品推入下方容器 |
| **侧面** | 可将物品推入侧面容器 |
| **对着顶部** | 收集任意掉落的物品实体 |

### 传输速度
- **每 8 游戏刻（0.4 秒）传输 1 个物品**
- 即 **2.5 个/秒** 或 **150 个/分钟**

---

## 关键用法

### 物品分类机
```
箱子 → 漏斗 → 漏斗（含过滤物品）→ 红石比较器 → 红石 → 下方漏斗锁
```
- Minecraft 自动存储系统的核心技术

### 自动熔炉
```
输入漏斗（上方） → 熔炉 → 输出漏斗（下方）
燃料漏斗（侧面） ↗
```
- 全自动烧炼系统

### 物品电梯
```
漏斗链 → 投掷器链 → 水电梯 → ...
```
- 垂直物品运输

### 漏斗矿车
- 漏斗 + 矿车 → 漏斗矿车
- 可沿轨道收集掉落物

---

## 红石控制

- **红石信号激活** → 漏斗**锁死**（停止传输）
- 常用于物品分类机中的精确控制
- 比较器读取漏斗内容量（信号强度 0~15）

## 你知道吗

1. 🔧 漏斗是 Minecraft 自动化中**使用最频繁**的红石组件——没有漏斗就没有"工业化"。
2. ⏱️ 漏斗的传输速度（2.5 个/秒）是 Minecraft 自动化设计中最关键的参数之一。
3. 🔒 红石信号锁死漏斗的机制允许精确控制物品流动——物品分类机依赖于此。
4. 📦 漏斗内部也有 5 格物品槽，可暂时缓存物品。
