# 海晶灯

![Sea Lantern](../page/images/blocks/sea-lantern/11782bb3.png)

> **稀有度**：不常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0.3 | **爆炸抗性**：0.3 | **发光**：是（15） | **透明**：否 | **可燃**：否
> **Java ID**：`sea_lantern` | **基岩 ID**：`sealantern` (169)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#00AAAA`

海晶灯是亮度 15 的水下光源，天然生成于海底遗迹和海底废墟。破坏掉落海晶碎片，精准采集可获取自身。独特的淡蓝色水下发光效果使其成为海洋建筑的首选照明。

---

## 获取

### 合成
```
4 海晶碎片 + 5 海晶砂粒

  沙沙沙
  沙片沙
  沙沙沙
```

### 自然生成
- 海底遗迹（海洋神殿）：大量镶嵌在结构中
- 海底废墟：偶尔出现

### 破坏
- 无精准采集 → 掉落 **2-3 海晶碎片**
- 精准采集 → 掉落海晶灯本身
