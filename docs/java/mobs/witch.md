# 女巫

![Witch](https://minecraft.wiki/images/Witch_JE3_BE3.png?format=original)

> **生命值**：26 | **伤害**：药水效果 | **生成**：主世界亮度 ≤ 7、沼泽小屋、被闪电击中村民
> **掉落**：多种药水材料 | **经验**：5

女巫是 Minecraft 中的魔法型敌对生物，使用药水进行攻击和防御。她是最有战术价值的敌对生物之一，也是获得**红石粉、萤石粉、火药**等困难的酿造材料的可再生来源。

---

## 生成

| 生成方式 | 概率/条件 |
|---------|----------|
| 主世界夜间 | 亮度 ≤ 7，单独生成（不与群族同时出现） |
| 沼泽小屋 | 世界生成时固定 1 只，之后不断刷新 |
| 袭击事件 | 第 3-5 波中出现 |
| 村民被闪电击中 | 立即转变（不可逆） |

---

## 战斗行为

### 进攻（玩家近距离）
女巫会向玩家投掷负面药水：
- **迟缓药水**：减速玩家
- **剧毒药水**：持续中毒伤害
- **虚弱药水**：降低近战伤害
- **伤害药水**：直接魔法伤害

### 防御（女巫受伤时）
女巫喝药水自愈：
- **治疗药水**：恢复生命值
- **再生药水**：持续回血
- **抗火药水**：免疫火焰伤害
- **水肺药水**：水下生存

### 药水抗性
女巫对大部分药水效果有天然抗性：
- 85% 魔法伤害抗性
- 免疫中毒、迟缓、虚弱效果

## 女巫农场

女巫是可再生萤石粉和红石粉的少数来源之一（另一个是牧师村民交易）：

1. 找到或建造**沼泽小屋**
2. 在刷怪点周围建造刷怪平台
3. 建造杀死机制（坠落 + 收集系统）
4. 每小时可产出数百个萤石粉和红石粉

> 女巫农场也是**火药**的可再生来源——虽然不如苦力怕农场高效，但兼产其他稀缺物资。

## 你知道吗

1. 女巫原本是 1.4.2 骇人更新中作为村民变种加入的（当时女巫是村民被闪电击中的唯一结果）。
2. 女巫喝抗火药水意味着她可以在熔岩中生存——玩家也一样（喝抗火药水站在熔岩中）。
3. 女巫是游戏中最"独立"的生物——她不与任何其他生物配合，所有的攻击和防御都靠自己的药水。
4. 女巫的小屋（沼泽小屋）包含一个炼药锅和一个工作台——暗示女巫懂得酿造和合成。
