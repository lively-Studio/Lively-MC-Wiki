# 凋灵骷髅

![Wither Skeleton](https://minecraft.wiki/images/Wither_Skeleton_JE4_BE3.png?format=original)

> **类别**：敌对生物 | **生命值**：20❤ | **生成**：下界要塞 | **掉落**：煤炭、骨头、凋灵骷髅头颅（2.5%）

凋灵骷髅是下界要塞中出现的危险敌对生物。它身高 3 格（比普通骷髅高 1 格），携带石剑，攻击附带凋零效果。其头颅是召唤凋灵 Boss 的必需材料。

---

## 生成

- **位置**：下界要塞内部
- **条件**：亮度 ≤ 7
- 在灵魂沙峡谷的下界要塞中生成概率更高
- 可生成在下界砖上，无论是否有屋顶

---

## 行为

### 攻击模式
- 移动速度与玩家行走速度相当
- 近战攻击，石剑伤害 **4❤（简单）/ 7❤（普通）/ 10❤（困难）**
- **凋零效果**：击中玩家后施加持续 10 秒的凋零状态
  - 凋零每 2 秒造成 1❤ 伤害
  - 生命条变黑，难以判断剩余血量

### 其他特性
- 免疫火焰伤害（在下界自然生成的结果）
- 阳光下不会燃烧（与普通骷髅不同）
- 对凋零效果免疫
- 被狼攻击时会反击

## 凋灵骷髅头颅

### 获取方法
- 凋灵骷髅 2.5% 基础掉落率
- 抢夺 III：约 5.5% 掉落率
- 需要 **3 个头颅** 来召唤凋灵

### 最佳刷头策略
- **抢夺 III 钻石剑/下界合金剑**
- 在灵魂沙峡谷的下界要塞建刷怪塔
- 使用狼群协助击杀（抢夺附魔对狼无效）

## 你知道吗

1. 凋灵骷髅是唯一在创造模式拾取栏中也显示不同纹理的物品——凋灵骷髅头颅。
2. 凋灵骷髅的碰撞箱比其他 3 格高生物窄——无法通过 2 格高的门。
3. 凋零效果是整个游戏中唯一会使生命条变黑的效果。
4. 如果在主世界生成凋灵骷髅（使用刷怪蛋），阳光不会对其造成伤害——它们天生不惧日光。
