# 铁傀儡

![Iron Golem](https://minecraft.wiki/images/Iron_Golem_JE4_BE4.png?format=original)

> **类别**：中立（村庄）/ 被动（玩家建造） | **生命值**：100❤ | **攻击力**：7.5-21.5❤ | **掉落**：铁锭、虞美人

铁傀儡是 Minecraft 中最强大的中立生物之一。这具高大的铁制守护者保护村民免受怪物侵害，也可以由玩家建造作为忠诚的守卫。

---

## 生成

### 自然生成
- 在村庄中，当满足以下条件时自动生成：
  - 至少 10 个村民（Java）或 20 个村民（BE）
  - 至少 20 张床
  - 村民曾"闲谈"过（表示他们感到受到威胁）
  - 上一个铁傀儡生成超过一定时间

### 玩家建造
将 4 个铁块摆成 **T 形**，再在顶部中央放置一个**雕刻南瓜**：

```
     [南瓜]
[铁块][铁块][铁块]
     [铁块]
```

- 南瓜必须最后放置
- 雕刻南瓜可以用南瓜灯或南瓜代替（仅 Java 版）
- 玩家建造的铁傀儡永远不会攻击玩家

---

## 行为

### 对村民
- 在村庄中巡逻
- 给村民赠送虞美人（表示友好）
- 检测到威胁时优先攻击

### 对怪物
- 主动攻击大多数敌对生物（苦力怕除外）
- **攻击力超强**：7.5❤（简单） / 15❤（普通） / 22.5❤（困难）
- 攻击时将目标抛向 **3-4 格高空**
- 可无视墙壁检测追踪目标

### 弱点
- 受到高处摔落伤害
- 水中会下沉并溺水

## 你知道吗

1. 铁傀儡的设计致敬了《天空之城》中的机器人。
2. 铁傀儡的碰撞箱为 1.4×2.7×1.4 格——它们无法通过 1.5 格高的空间。
3. 铁傀儡农场是游戏中最有效率的铁来源——利用村民的恐慌机制无限生成。
4. 给铁傀儡喂食铁锭可以修复 25❤ 生命值。
