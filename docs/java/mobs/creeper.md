# 苦力怕

![Creeper](https://minecraft.wiki/images/Creeper_JE4_BE2.png?format=original)

> **生命值**：20 | **最大爆炸伤害**：49（普通）| **生成**：主世界亮度 ≤ 0
> **掉落**：火药、音乐唱片（被骷髅杀死时）| **经验**：5

苦力怕是 Minecraft 最具标志性的生物——一个无臂、四足的绿色生物，靠近玩家后自爆。它最初是 Notch 编程猪模型时水平反转产生的意外，后来被命名为"爬行者/苦力怕"。

## 爆炸机制

### 引爆条件
- 与玩家的距离 ≤ 3 格（Java）/ ≤ 1.85 格（基岩）
- 引信启动后发出嘶嘶声
- 引信过程中玩家跑出爆炸范围 → 苦力怕取消爆炸并重新追击

### 爆炸伤害计算
```
伤害 = 爆炸威力 × 暴露比例 × (1 - 距离衰减)
```

- 零距离：约 49 点伤害（足以一击击杀无保护的玩家）
- 3 格距离：约 10 点伤害
- 全套无保护下界合金（20 盔甲值）：零距离仍受到约 **10 点伤害**

---

## 变种

### 闪电苦力怕
当苦力怕被**闪电**击中时变为闪电苦力怕：
- 爆炸半径翻倍（6 格）
- 爆炸威力翻倍（最高 97 伤害）
- 外观增加蓝色电弧
- 被闪电苦力怕炸死的生物掉落**生物头颅**（僵尸、骷髅、苦力怕）

> 收集生物头颅的主要方法：建一个避雷针 + 苦力怕陷阱，在雷暴天气中激活。

## 掉落物

| 掉落物 | 概率 | 说明 |
|--------|------|------|
| 火药 | 100% | 0-2 个（抢夺 +1/级） |
| 音乐唱片 | 100% | 仅当被**骷髅**的箭射死时掉落 |

### 音乐唱片掉落机制
骷髅射中苦力怕并击杀 → 掉落随机音乐唱片（共 16 种）：
- 这是获得唱片 "11"（破损唱片）和 "Pigstep" 的少数方式之一
- 需要精确操作：先让骷髅伤害苦力怕，再由骷髅完成最后一击

## 应对策略

### 盾牌
举盾可完全格挡爆炸伤害（正面），但爆炸仍会摧毁周围方块。

### 打火石
对着跑来的苦力怕使用打火石可立即引爆它（有准备时间）。

### 猫
**猫/豹猫**会驱赶苦力怕——在基地附近养猫是有效的防御措施。

### 远程
用弓在 16 格外即可安全击杀——苦力怕不会在远程伤害下引爆。

## 你知道吗

1. 苦力怕是世界上辨识度最高的游戏角色之一——即使没玩过 Minecraft 的人也认识它。
2. 苦力怕的原始模型是一个失败的猪模型——Notch 搞反了长度和高度。
3. "That's a very nice [everything you have] you have there. It would be a shame if something happened to it." 是社区关于苦力怕的经典台词。
4. 苦力怕怕猫的原因从未在游戏中有官方解释——是纯粹的社区梗。
5. 2011 年 Mojang 发布了苦力怕的官方毛绒玩具，至今仍是最畅销的 Minecraft 周边之一。
