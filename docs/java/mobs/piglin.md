# 猪灵

![Piglin](https://minecraft.wiki/images/Piglin_JE3_BE2.png?format=original)

> **类别**：中立生物 | **生命值**：16❤ | **生成**：下界（绯红森林、下界荒地、堡垒遗迹） | **掉落**：金剑、弩、金质盔甲

猪灵是 1.16 下界更新加入的中立生物。它们生活在下界，对黄金有着极端的痴迷——只要佩戴金质盔甲，它们就会保持中立。独特的"以物易物"系统使它们成为下界最重要的资源来源之一。

---

## 生成

- **自然生成**：绯红森林、下界荒地、堡垒遗迹
- 亮度 ≤ 11 时生成
- 在非和平模式下生成
- 部分猪灵在生成时即装备金剑或弩

---

## 行为规则

### 中立条件
只要玩家满足以下**任一**条件，猪灵保持中立：
- 穿戴任何**金质盔甲**（金头盔/胸甲/护腿/靴子）
- 与猪灵有以物易物进行中
- 等级 ≥ 有一定声望

### 敌对条件（会触发猪灵攻击）
- 玩家没有穿金质盔甲
- 玩家挖掘以下方块：
  - 金矿石、下界金矿石
  - 镶金黑石
  - 金块、下界合金块
  - 末影箱、潜影盒
  - 金钟
- 打开或破坏箱子、陷阱箱、木桶、潜影盒
- 攻击猪灵或猪灵蛮兵

### 以物易物
- 向猪灵**丢出金锭**（按 Q 或右键）
- 猪灵捡起金锭后，短暂观察后返回互动物品
- 以物易物可能出现以下物品：

| 物品 | 概率 | 说明 |
|------|------|------|
| 黑曜石 | 8.79% | 下界传送门材料 |
| 哭泣的黑曜石 | 8.79% | 重生锚材料 |
| 灵魂沙 | 8.79% | 灵魂奔跑者 + 凋灵召唤 |
| 火焰弹 | 8.79% | 类似打火石 |
| 铁粒 | 8.79% | 可合成铁锭 |
| 抗火药水 | 8.79% | 完全免疫火焰伤害 |
| 线 | 7.53% | 可制羊毛 + 钓竿 |
| 下界石英 | 7.53% | 红石比较器 + 建材 |
| 萤石粉 | 7.53% | 照明 + 药水 |
| 岩浆膏 | 5.72% | 抗火药水酿造 |
| 末影珍珠 | 5.72% | 去末地的敲门砖 |
| 下界砖 | 4.39% | 下界砖块 |
| 附魔书（灵魂疾行）| 1.13% | 最珍贵的以物易物奖励 |

### 僵尸化
- 猪灵被带到主世界或末地后，约 15 秒后开始抖颤
- 完全僵尸化后变为**僵尸猪灵**（不再以物易物）

---

## 你知道吗

1. 猪灵是 Minecraft 中第一种可在两个维度间转化的生物（猪灵 → 僵尸猪灵）。
2. 在猪灵被攻击时，周围的其他猪灵会加入战斗——类似于狼。
3. 堡垒遗迹中的宝箱会吸引猪灵守卫——即使玩家有金盔甲，打开箱子仍会激怒它们。
4. 猪灵蛮兵（50❤ 生命值）不会以物易物，且免疫任何形式的贿赂——它们只为保护堡垒而存在。
