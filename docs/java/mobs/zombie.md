# 僵尸

![Zombie](https://minecraft.wiki/images/Zombie_JE4_BE4.png?format=original)

> **生命值**：20 | **伤害**：2.5-4.5（依难度） | **生成**：主世界亮度 ≤ 0
> **掉落**：腐肉、铁锭（稀有）、胡萝卜/马铃薯（稀有）| **经验**：5 + 装备加成

僵尸是 Minecraft 中最具代表性的敌对生物，也是新手玩家遇到的第一个真正威胁。它们行动缓慢但数量众多，在夜晚成群结队地出现。

---

## 生成

- 主世界**亮度 ≤ 0** 的固体方块上
- 生成上限极高——夜晚平原上可同时存在数十只
- 群体生成：4 只为一组

### 变种生成
| 变种 | 生成条件 |
|------|---------|
| 尸壳 | 沙漠生物群系（不会在阳光下燃烧） |
| 溺尸 | 水下生成；僵尸溺水 30 秒后转化 |
| 小僵尸 | 5% 概率；速度极快，白天不燃烧 |
| 僵尸村民 | 5% 概率；可用虚弱药水+金苹果治愈 |

---

## 行为

### 追踪与攻击
- 白天阳光直射下**燃烧**（戴头盔/南瓜可延缓）
- 检测玩家范围：40 格
- 能通过听觉感知玩家
- 攻击会触发周围的僵尸也加入追踪（群体仇恨）

### 特殊行为
- 可以**打破木门**（困难模式）
- 可以**捡起掉落物**和装备
- 小僵尸不会在阳光下燃烧，且移动极快
- 被攻击会召唤周围僵尸协助（类似僵尸猪灵）

|--------|------|------|
| 腐肉 | 100% | 0-2 个（抢夺 +1/级） |

| 马铃薯 | 0.83% | 同上概率 |

## 僵尸村民治愈

僵尸村民可被治愈为普通村民：
1. 对僵尸村民投掷**虚弱喷溅药水**
2. 手持**金苹果**右键点击
3. 等待 3-5 分钟（铁栏杆和床可加速）
4. 治愈后村民保留随机职业

> 治愈的村民会给予玩家永久交易折扣！反复治愈可使交易价格降至最低 1 绿宝石。

## 你知道吗

1. 僵尸是 2009 年 Minecraft 最早的敌对生物之一，原型来自经典丧尸电影。
2. 小僵尸骑鸡的生成组合被称为"鸡骑士"——是游戏中最稀有的自然生成之一。
3. 僵尸可以装备任何工具和盔甲——包括钻石和下界合金，虽然概率极低。
4. 被闪电击中的僵尸村民会直接变为女巫，而非复活。
