# 村民

![Villager](https://minecraft.wiki/images/Plains_Villager_Base_JE6_BE5.png?format=original)

> **类别**：被动生物 | **生命值**：20❤ | **生成**：村庄 | **掉落**：无

村民是 Minecraft 中最重要的非玩家角色（NPC）。他们居住在村庄中，各有职业，提供交易系统——这是游戏中获取附魔书、经验瓶、钻石工具等高级物品的主要途径。

---

## 生成

- **自然生成**：在村庄中与建筑同时生成
- **繁殖**：两个村民 + 足够的床 + 食物
- **治愈僵尸村民**：投掷虚弱药水 + 喂食金苹果
- **职业分配**：失业村民会自动认领附近的职业方块

## 交易系统

### 交易等级
每个职业有 5 个交易等级：
- 新手（Novice）→ 学徒（Apprentice）→ 老手（Journeyman）→ 专家（Expert）→ 大师（Master）
- 每次完成任意交易获得 2-12 点经验
- 升级解锁新的交易选项

### 价格浮动
- **村庄英雄**：打劫掠后获得，交易价格大幅折扣
- **声望**：正面声望让村民降价
- **僵尸化后治愈**：治愈的村民提供最便宜的价格（永久折扣）
- **供需机制**：购买太多某物品会暂时涨价

### 关键交易推荐
1. **图书管理员 → 经验修补附魔书**（最珍贵）
2. **牧师 → 末影珍珠**（去末地的替代方案）
3. **武器匠/工具匠/盔甲匠 → 钻石装备**（可再生钻石来源）
4. **农民 → 金胡萝卜**（最佳食物之一）

## 你知道吗

1. 村民的鼻子（"大鼻子"）是 Notch 有意为之的设计——他认为这让人觉得更友善。
2. 将村民多次感染为僵尸再治愈，可以将交易价格压至 1 绿宝石。
3. 傻子村民是唯一穿绿色长袍的村民——作为没有功能的"彩蛋"角色。
4. 村民交易大厅是大多数 Minecraft 技术服务器的核心设施。
