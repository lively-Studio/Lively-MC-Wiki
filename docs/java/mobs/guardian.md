# 守卫者

![Guardian](https://minecraft.wiki/images/Guardian_JE3_BE3.png?format=original)

> **生命值**：30 | **伤害**：4-9（激光）| **生成**：海底遗迹
> **掉落**：海晶碎片、海晶砂粒、生鱼 | **经验**：10

守卫者是守护海底遗迹的水下敌对生物。它能发射不可躲避的激光攻击，尖刺外壳可反弹近战伤害。远古守卫者是它的巨大变种。

---

## 生成

- 仅在**海底遗迹**内及其周围生成
- 每个海底遗迹周围有 22 个固定生成点
- 从不自然重生成——只有生成点的守卫者被杀死后会在原地重生

---

## 攻击机制

### 激光攻击
守卫者的激光是游戏中独特的攻击方式：
1. **充能阶段**：2 秒（激光颜色从浅紫 → 亮白）
2. **伤害阶段**：4-9 点魔法伤害
3. 激光**不可被盾牌完全格挡**（伤害减半）
4. **不可躲避**——一旦锁定必然命中（除非退出射程）

### 尖刺防御
守卫者受到伤害时，尖刺外壳会反弹伤害：
- 每次反弹：**2 点伤害**（无视盔甲）
- 近战攻击者受到反弹
- 远程攻击不受反弹

### 游泳速度
- 守卫者可在水中快速游泳
- 追击玩家时速度极快
- 离开水后会扑腾

## 远古守卫者

每个海底遗迹生成 **3 只远古守卫者**（正厅 1 只，翼楼各 1 只）：

| 属性 | 守卫者 | 远古守卫者 |
|------|--------|-----------|
| 生命值 | 30 | 80 |
| 激光伤害 | 4-9 | 5-12 |
| 挖掘疲劳 | 无 | 施加 III 级（5 分钟） |
| 大小 | 普通 | 巨大（约 2 倍） |
| 掉落物 | 同下 | + 湿海绵 |

> 远古守卫者的**挖掘疲劳 III** 使玩家几乎无法挖掘任何方块（挖掘速度降低 99.7%），需要喝牛奶才能解除。

## 海底遗迹攻略

1. **准备装备**：全套钻石/铁质盔甲（保护 IV）、水肺药水、夜视药水
2. **找到入口**：遗迹正面大门通常朝向海洋中心
3. **优先击杀远古守卫者**：进入后第一时间定位并击杀 3 只远古守卫者
4. **清除挖掘疲劳**：每击杀一只远古守卫者就喝牛奶
5. **海绵房**：击败所有远古守卫者后，挖出海绵房的湿海绵
6. **金块**：遗迹中心有 8 个金块（藏在暗海晶石中）

## 你知道吗

1. 守卫者的激光伤害类型是**魔法伤害**——保护附魔有效，但常规盔甲减伤无效。
2. 海底遗迹是游戏中为数不多的"无法用镐挖掘进入"的结构——远古守卫者的挖掘疲劳迫使玩家在入口处正面进入。
3. 守卫者的音效（激光和叫声）由 C418 设计，混合了电子音和水下气泡的效果。
4. 海晶碎片和海晶砂粒是制作**海晶石系列方块**的必需材料——没有守卫者农场就无法批量获取。
