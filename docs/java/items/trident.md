# 三叉戟

![Trident](https://minecraft.wiki/images/Trident_JE2_BE2.png?format=original)

> **类型**：武器 | **耐久度**：250 | **可堆叠**：否 | **可再生**：否（仅溺尸掉落）
> **近战伤害**：9 | **投掷伤害**：8 | **攻击速度**：1.1 | **稀有度**：罕见

三叉戟是 Minecraft 中唯一可以同时作为**近战武器**和**投掷武器**的物品，也是唯一拥有专属附魔体系的武器。它不能被合成，只能从溺尸掉落获得。

---

## 获取方式

### 溺尸掉落
- 持有三叉戟的溺尸死亡时掉落：
  - Java 版：**6.25%** 概率（抢夺每级 +1%，抢夺 III = 9.25%）
  - 基岩版：**15%** 概率（抢夺每级 +1%，抢夺 III = 18%）
- 三叉戟无法从非持有它的溺尸获得

> **注意**：只有手持三叉戟的溺尸才会掉落三叉戟。目前 Java 版仅有 6.25% 的溺尸手持三叉戟——这意味着总掉落率约为 **0.39%**，使其成为最难获得的武器之一。

---

## 几种模式

### 近战模式
- 攻击伤害：**9 点**
- 攻击速度：1.1
- 类似于剑的使用方式

### 投掷模式
- 投掷伤害：**8 点**
- 投掷距离：约 24 格（随后下落）
- 投掷后有冷却时间（武器槽的白色覆盖动画）
- 投掷后必须捡回（除非有忠诚附魔）

---

## 专属附魔

三叉戟拥有 4 种独特附魔，这是任何其他物品都没有的：

| 附魔 | 最高等级 | 效果 |
|------|---------|------|
| 忠诚 III | 3 | 投掷后自动飞回，等级越高返回越快 |
| 激流 III | 3 | 在水中/雨中投掷时带着玩家飞行前进 |
| 引雷 I | 1 | 雷暴天气中击中生物引发雷电 |
| 穿刺 V | 5 | 对水生生物每级 +2.5 伤害 |

> **冲突**：忠诚与激流不可共存。激流与引雷不可共存。

### 引雷的特殊用途
- 将苦力怕变为**闪电苦力怕**（爆炸威力×2）
- 将村民变为**女巫**
- 将猪变为**僵尸猪灵**
- 将红色哞菇变为**棕色哞菇**（反之亦然）
- 获取"**非常**恐怖"成就

---

## 最佳附魔组合

| 玩法 | 推荐附魔 |
|------|---------|
| 投掷流 | 忠诚 III + 穿刺 V + 耐久 III + 经验修补 |
| 飞行流 | 激流 III + 穿刺 V + 耐久 III + 经验修补 |
| 近战流 | 穿刺 V + 耐久 III + 经验修补 |

## 你知道吗

1. 三叉戟是唯一不能通过合成或交易获得的武器。
2. 配合鞘翅 + 激流 III + 下雨天气 = 几乎无限的自由飞行能力。
3. 激流 III 的突进速度可在雨中达到极快速度——比创造模式飞行更快。
4. 用忠诚 III 的三叉戟投掷时，如果玩家传送得太远（如下界传送门），三叉戟将永远失去。
