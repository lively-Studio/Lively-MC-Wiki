# 弓

![Bow](https://minecraft.wiki/images/Bow_JE2_BE2.png?format=original)

> **类型**：武器 | **耐久度**：384 | **可堆叠**：否 | **可再生**：是
> **拉满伤害**：9-10（暴击） | **射速**：约 1 秒/箭 | **稀有度**：常见

弓是 Minecraft 中最经典的远程武器，不需要弹药材料消耗（配合无限附魔），在 PvE 和 PvP 中都有着不可替代的地位。

---

## 获取方式

### 合成
```
[木棍] [线] [空]
[木棍] [空] [线]
[木棍] [线] [空]
```
需要：3 根木棍 + 3 根线

### 生物掉落
- 骷髅死亡时有 8.5% 概率掉落弓
- 抢夺附魔可提高掉落概率（抢夺 III 为 11.5%）
- 掉落的弓可能已附魔，耐久度随机

### 钓鱼
弓可作为宝藏类钓鱼战利品获得，概率约 0.8%。

### 交易
- 制箭师村民（专家等级）：2-3 绿宝石出售 1 把弓
- 制箭师村民（大师等级）：有概率出售附魔弓

## 使用机制

### 拉弓过程
- 按住右键拉弓，箭的伤害随拉扯时间增加
- 拉弓时间 0-1 秒：伤害 1-9（线性增长）
- 拉满（1 秒以上）：造成 9 点基础伤害
- 拉满 + 暴击（弓处于"完全拉满"的粒子效果）：额外伤害

### 箭轨
- 箭以抛物线轨迹飞行
- 初始速度约 60 m/s（拉满时）
- 受重力影响，远程射击需要抬高角度

## 最佳附魔

| 附魔 | 最高等级 | 说明 |
|------|---------|------|
| 力量 V | 5 | 每级 +25% 伤害（V 级 = +150%，总计 250%） |
| 冲击 II | 2 | 增加击退距离 |
| 火矢 I | 1 | 点燃目标和 TNT |
| 无限 I | 1 | 消耗 1 支箭即可无限射击 |
| 耐久 III | 3 | 延长使用寿命 |

> **冲突**：无限与经验修补不可共存。推荐：力量 V + 冲击 II + 火矢 I + 无限 I + 耐久 III。

## 弓 vs 弩

| 特性 | 弓 | 弩 |
|------|----|-----|
| 射速 | 约 1 秒/箭 | 约 1.25 秒装填 |
| 蓄力 | 需持续拉弓 | 装填后可保留 |
| 伤害（无附魔） | 1-10 | 6-11 |
| 特殊弹药 | 药箭 | 烟花火箭 |
| 多重射击 | 不支持 | 支持（3 箭齐发） |
| 水下射击 | 箭速降低 | 正常射击 |

## 你知道吗

1. 骷髅使用弓时会自动计算弹道——如果玩家横向移动，骷髅的箭落点会预判偏移。
2. 力量 V 的弓在拉满时一箭伤害可达 25 点——足以击杀一只苦力怕。
3. 在 Beta 1.2 之前，弓像机关枪一样无需拉弓——右键点击即可以极高速率射出箭矢。
4. 弓的材质中，弦颜色的深浅反映拉弓程度——全白表示完全拉满。
