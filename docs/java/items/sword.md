# 剑

![Sword](https://minecraft.wiki/images/Netherite_Sword_JE3_BE3.png?format=original)

> **类型**：武器 | **耐久度**：59-2031（依材质） | **可堆叠**：否 | **可再生**：是
> **攻击伤害**：4-8（依材质） | **攻击速度**：1.6 | **稀有度**：常见

剑是 Minecraft 中最常用的近战武器，也可用于快速破坏某些方块（蜘蛛网、竹子等）。

## 合成配方

```
[材料]
[材料]
[木棍]
```

材料：木板 → 木剑；圆石 → 石剑；铁锭 → 铁剑；金锭 → 金剑；钻石 → 钻石剑
下界合金剑：钻石剑 + 下界合金锭（锻造台）

## 攻击机制

### 攻击冷却
Java 版 1.9+ 引入攻击冷却（剑冷却时间 0.625 秒）：
- **满冷却（200%）**：造成全额伤害 + 横扫攻击
- **部分冷却**：伤害按比例降低
- **冷却指示器**：准星旁的剑形图标

### 横扫攻击
满冷却时，剑会对站立在攻击范围内的所有敌对生物造成**1 点横扫伤害**（可被横扫之刃附魔提升至最高 75% 主伤害）。

## 你知道吗

1. 在 Beta 1.5 之前，剑的伤害与工具材质无关——木剑和钻石剑伤害相同。
2. 剑可以比空手更快地破坏蜘蛛网（用剑 0.4 秒 vs 空手 20 秒）。
3. 下界合金剑的材质保留了原钻石剑上的部分蓝色宝石装饰。
4. 在 1.9 战斗更新之前，剑没有攻击冷却，可以连续快速点击。
5. 火焰附加 II 的剑是烤动物肉的高效工具——杀死燃烧的生物直接掉落熟肉。
