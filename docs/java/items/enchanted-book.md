# 附魔书

![Enchanted Book](https://minecraft.wiki/images/Enchanted_Book_JE2_BE2.png?format=original)

> **类型**：物品 | **可堆叠**：是（64） | **可再生**：是 | **稀有度**：罕见-非常稀有
> **用途**：通过铁砧将附魔转移到物品上

附魔书是 Minecraft 中唯一可以"储存"附魔的物品。一本书可以储存 1 个或多个附魔，然后通过铁砧将特定附魔应用到工具、武器或盔甲上。

---

## 获取方式

### 附魔台
- 放置一本书在附魔台中进行附魔
- 与直接附魔工具不同，附魔书的附魔选择更加随机
- 30 级附魔台 + 15 个书架 = 最佳结果

### 村民交易
图书管理员村民是附魔书的最稳定来源：
- 每个图书管理员出售 1-3 种附魔书
- 价格范围：5-64 绿宝石 + 1 本书
- 可反复破坏/放置讲台刷新交易（Java 版）

### 战利品
附魔书可在以下结构的箱子中找到：
- 末地城
- 要塞图书馆
- 沙漠神殿
- 地牢
- 林地府邸
- 沉船（宝藏室）
- 钓鱼（宝藏类别，概率约 1.6%）

---

## 使用方式 (铁砧)

1. 将**附魔书**放在铁砧左格
2. 将**目标物品**放在铁砧中格
3. 铁砧自动计算附魔兼容性
4. 消耗**经验等级**完成附魔转移

### 附魔费用规则
- 第一次操作：1-5 级
- 每次后续操作费用递增
- 累计费用 **≥ 40 级**时显示"过于昂贵！"，无法继续操作
- 通过砂轮移除所有附魔可重置费用

### 合并附魔书
两本同附魔的附魔书可以合并：
- 效率 III + 效率 III = 效率 IV
- 消耗经验等级递增
- 无法超出该附魔的最高等级（效率最高 V）

## 附魔兼容性总表

以下附魔**互斥**（不能在同一物品上共存）：

| 互斥组 | 包含附魔 |
|--------|---------|
| 伤害组 | 锋利、亡灵杀手、节肢杀手 |
| 保护组 | 保护、弹射物保护、爆炸保护、火焰保护 |
| 采集组 | 精准采集、时运 |
| 无限/修补 | 无限、经验修补 |
| 三叉戟组 | 激流、忠诚（激流也与引雷冲突） |
| 弩组 | 多重射击、穿透 |

## 最珍贵的附魔书

| 附魔 | 最高等级 | 获取难度 | 价值 |
|------|---------|---------|------|
| 经验修补 I | 1 | 只能通过交易或战利品获得（附魔台无法出） | 极高 |
| 冰霜行者 II | 2 | 仅通过交易/战利品获得 | 极高 |
| 灵魂疾行 III | 3 | 仅猪灵交易获取 | 极高 |
| 迅捷潜行 III | 3 | 仅远古城市战利品箱 | 极高 |
| 精准采集 I | 1 | 附魔台低概率，热门交易 | 高 |
| 时运 III | 3 | 附魔台中等概率 | 高 |
| 耐久 III | 3 | 附魔台高概率 | 中 |
| 效率 V | 5 | 需合并 2 本效率 IV | 中 |

> **经验修补**是最重要的附魔书——没有它，任何物品都会因耐久耗尽而永久损毁。

## 你知道吗

1. 一本附魔书可以包含多个附魔——从战利品或钓鱼获得的附魔书通常有 2-3 个附魔。
2. 图书管理员是唯一出售经验修补附魔书的稳定来源——这使"村民繁殖场 + 图书管理员大厅"成为几乎所有 Minecraft 技术玩家的重要工程。
3. 附魔书的闪烁光泽动画是一种独特的视觉效果，代表内容未读的"魔法书"概念。
4. 在基岩版中，将附魔书放在砂轮上可以移除所有附魔并返还部分经验。
