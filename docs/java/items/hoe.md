# 锄

![Hoe](https://minecraft.wiki/images/Netherite_Hoe_JE3_BE3.png?format=original)

> **类型**：工具 | **耐久度**：59-2031（依材质） | **可堆叠**：否 | **可再生**：是
> **攻击伤害**：1 | **攻击速度**：1.0-4.0（依材质） | **稀有度**：常见

锄在 Minecraft 早期被认为是最无用的工具——不能用于挖掘，攻击伤害仅有 1 点。但在后续更新中，锄逐渐获得了更多功能，特别是下界合金锄具有最快的攻击速度（4.0），成为独特的高速武器。

---

## 材质等级

| 材质 | 耐久度 | 攻击速度 | 说明 |
|------|--------|---------|------|
| 木质 | 59 | 1.0 | 基础锄 |
| 石质 | 131 | 2.0 | 略微提升攻击速度 |
| 铁质 | 250 | 3.0 | 平衡选择 |
| 金质 | 32 | 1.0 | 附魔性能好 |
| 钻石质 | 1561 | 4.0 | 后期选择 |
| 下界合金质 | 2031 | 4.0 | 最高耐久的锄 |

> 下界合金锄是唯一推荐升级到最高材质的锄——高耐久 + 高攻速使其成为特殊战斗工具。

## 用途

### 耕地
对着泥土、草方块或土径**右键点击**可将其变为耕地。耕地用于种植以下作物：
- 小麦种子（小麦）
- 马铃薯
- 胡萝卜
- 甜菜根种子（甜菜根）
- 南瓜种子（南瓜）
- 西瓜种子（西瓜）
- 火把花种子
- 瓶子草荚果

耕地在水源 4 格范围内会保持湿润（深色），超出范围会逐渐退化为泥土。

### 破坏方块
锄是最快速破坏以下方块的合适工具：
- 干草块
- 标靶
- 干海带块
- 树叶
- 海绵、湿海绵
- 下界疣块、诡异疣块
- 幽匿脉络、幽匿块、幽匿尖啸体、幽匿感测体
- 苔藓块

## 你知道吗

1. 直到 1.16 下界更新之前，不同材质的锄之间没有实质区别——都只能耕地且只有 1 点伤害。
2. 下界合金锄的 4.0 攻速是游戏中最快的近战攻击速度，可实现每秒 4 次 1 点伤害的攻击。
3. 在 Java 版中，锄被归类为"挖掘工具"而非"武器"——这意味着刷怪笼中生成的僵尸拿着锄无法攻击玩家。
4. 干草块的采集工具居然是锄——这被许多玩家认为是反直觉的设计。
