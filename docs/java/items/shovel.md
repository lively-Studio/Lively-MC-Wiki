# 锹

![Shovel](https://minecraft.wiki/images/Netherite_Shovel_JE3_BE3.png?format=original)

> **类型**：工具 | **耐久度**：59-2031（依材质） | **可堆叠**：否 | **可再生**：是
> **攻击伤害**：2.5-6.5（依材质） | **攻击速度**：1.0 | **稀有度**：常见

锹是用于快速挖掘泥土、沙子、沙砾等松散方块的必备工具。在生存初期，石锹通常是最早合成的工具之一。

## 可加速挖掘的方块

锹可加速挖掘以下方块：
- 泥土、砂土、缠根泥土
- 草方块、菌丝体、灰化土
- 沙子、红沙、灵魂沙
- 沙砾、可疑的沙砾
- 黏土块
- 雪、雪块、细雪（虽然细雪无法掉落）

---

## 特殊用途

### 创造草径
对着草方块、泥土、砂土、菌丝体、灰化土或缠根泥土**右键点击**可将其变为土径。此操作消耗 1 点耐久度。

### 船
在基岩版中，对着船使用锹可使船变为任意种类的木板。

## 你知道吗

1. 锹是唯一不能直接用于攻击多个目标的工具（剑有横扫，斧有高伤害，镐有群伤）。
2. 金锹虽然脆弱，但挖掘速度极快，适合短时间大量挖掘沙子。
3. 在早期 Minecraft 中，沙砾有 10% 概率掉落燧石——现在仍可通过时运附魔达到最高 100% 概率。
4. 土径比普通泥土低 1/16 格高度，行走时有独特音效。
