# 食物

![Food](https://minecraft.wiki/images/Golden_Apple_JE4_BE2.png?format=original)

> **类型**：消耗品 | **效果**：恢复饥饿值 + 饱和度 | **可堆叠**：大部分 64 | **可再生**：大部分是

食物是 Minecraft 生存模式的核心要素之一。玩家的饥饿值会随时间消耗（奔跑、跳跃、战斗加速消耗），需要通过进食来维持。饥饿值归零时玩家开始受到伤害（简单/普通模式降至 10、困难模式可致命）。

---

## 饥饿值系统

- 饥饿值范围：0-20（10 个鸡腿图标）
- **饱和度**：隐藏数值，进食时与饥饿值同时恢复
- 饱和度先消耗，耗尽后才开始消耗饥饿值
- 饥饿值 ≥ 18 时缓慢自然回血

## 高效食物来源

### 前期（第一天）
- 杀死猪/牛/羊/鸡获取生肉，用熔炉烧熟
- 打草获取小麦种子 → 种小麦 → 做面包

### 中期（建立农场后）
- 养牛场：小麦繁殖牛，牛排 = 游戏最佳性价比食物
- 自动烤鸡机：鸡下蛋 + 岩浆灼烧成年鸡 → 无限熟鸡肉
- 村民交易：农民村民出售金胡萝卜

### 后期（自动化）
- 猪灵交易农场：金锭 → 无限猪灵交易（包含大量可食用的生猪排）
- 疣猪兽农场：下界中可建立疣猪兽陷阱获取熟猪排

## 你知道吗

1. 附魔金苹果是生存模式中最珍贵的食物，但被 Mojang 标记为不可合成（1.9 后），只能从战利品箱获得。
2. 河豚的负面效果如此严重，以至于吃河豚的唯一"用途"是获得进度"均衡饮食"。
3. 金胡萝卜虽然恢复的饥饿值不如牛排，但提供的饱和度最高——适合需要长时间不进食的场景（如长时间 PvP 战斗）。
4. 蛋糕是唯一需要放在方块上才能吃的食物——每右键一次吃掉一片（共 7 片）。
