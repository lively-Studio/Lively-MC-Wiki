# 盾牌

![Shield](https://minecraft.wiki/images/Shield_JE3_BE2.png?format=original)

> **类型**：防具 | **耐久度**：336 | **可堆叠**：否 | **可再生**：是
> **防御**：100% 正面伤害减免 | **冷却**：被斧击中后 5 秒禁用 | **稀有度**：常见

盾牌是 1.9 战斗更新中加入的防御装备，装备在副手槽中，右键使用可完全格挡来自正面的近战和远程攻击。

---

## 合成配方

```
[木板] [铁锭] [木板]
[木板] [木板] [木板]
[空]   [木板] [空]
```

需要：6 块木板（任意木材种类）+ 1 个铁锭

---

## 防御机制

### 格挡效果
- 完全格挡来自**正面**的最多约 180° 范围内的：
  - 近战攻击（生物和玩家）
  - 箭矢
  - 火球（恶魂、烈焰人）
  - 雪球、鸡蛋
  - 三叉戟
  - 烟花火箭冲击
- 被格挡的攻击不会消耗盾牌耐久度（箭矢除外）
- 箭矢被格挡后弹开，盾牌消耗 1 点耐久度

### 无法格挡
- 苦力怕爆炸伤害
- TNT 爆炸伤害
- 守卫者激光（伤害减少但无法完全格挡）
- 监守者音波攻击
- 来自背后和侧面的攻击
- 穿透附魔的弩箭

### 盾牌禁用
被**斧**攻击（生物或玩家）命中后，盾牌进入 **5 秒禁用状态**（无法格挡）。禁用期间盾牌图标上出现裂痕动画。

---

## 旗帜图案

盾牌可与旗帜结合，将旗帜图案印在盾牌正面：
- 1 个盾牌 + 1 个任意旗帜 → 带图案的盾牌
- 图案盾牌可被炼药锅清洗恢复为普通盾牌
- 旗帜图案不影响盾牌功能

## 你知道吗

1. 在 1.9 之前，玩家用**剑右键格挡**来减少 50% 伤害。1.9 后盾牌取代了这个机制。
2. 盾牌可以格挡凋灵之首的直接命中，但无法完全阻止爆炸伤害。
3. 一群掠夺者的弩箭齐射可以迅速耗尽无附魔盾牌的 336 耐久度。
4. 在 Java 版中，蹲下时举盾可以减少被击退距离。
5. 盾牌的格挡角度是约 180° 的半球形——正面和左右约 90° 范围内的攻击都可被格挡。
