# 弩

![Crossbow](https://minecraft.wiki/images/Crossbow_JE2_BE2.png?format=original)

> **类型**：武器 | **耐久度**：465 | **可堆叠**：否 | **可再生**：是
> **伤害**：6-11（随机） | **装填时间**：1.25 秒（1.0 含快速装填 III） | **稀有度**：常见

弩在 1.14 村庄与掠夺更新中加入，是弓的替代方案。与弓不同，弩可以**预装填**——将装好箭矢的弩放在物品栏中，需要时切出来即可立即发射。

---

## 获取方式

### 合成
```
[木棍] [铁锭] [木棍]
[线]   [绊线钩] [线]
[空]   [木棍]   [空]
```
需要：3 根木棍 + 2 根线 + 1 个铁锭 + 1 个绊线钩

### 生物掉落
- 掠夺者死亡时掉落弩（概率 8.5%）
- 猪灵死亡时掉落弩（概率 8.5%）

### 掠夺者前哨站
前哨站箱子里有概率生成弩。

## 弩 vs 弓

| 特性 | 弩 | 弓 |
|------|----|----|
| 装填方式 | 装填后保留（可预装 9 个） | 即拉即射 |
| 伤害（无附魔） | 6-11 | 1-10 |
| 射速（持续） | 约 1.25 秒/箭 | 约 1 秒/箭 |
| 首次射击速度 | 即刻（预装填） | 需拉 1 秒 |
| 力量附魔 | 不支持 | 最高 +150% |
| 特殊弹药 | 烟花火箭（范围伤害） | 药箭 |
| 多重射击 | 支持（3 箭齐发） | 不支持 |
| 穿透射击 | 支持（穿透敌人） | 不支持 |

---

## 最佳附魔

| 附魔 | 最高等级 | 说明 |
|------|---------|------|
| 快速装填 III | 3 | 装填时间-0.75 秒（仅需 0.5 秒） |
| 多重射击 I | 1 | 一次射出 3 支箭（仅消耗 1 支） |
| 穿透 IV | 4 | 箭矢穿透实体数量 +1（最高穿透 5 个目标） |
| 耐久 III | 3 | 延长寿命 |
| 经验修补 I | 1 | 经验修复 |

> **冲突**：多重射击与穿透不可共存。

## 烟花火箭弹药

弩可装填**烟花火箭**作为弹药：
- 无烟火之星：造成 5-6 点伤害（无爆炸效果）
- 有烟火之星：造成 5-6 点伤害 + 每个烟火之星额外 1-2 点伤害
- 最多 7 个烟火之星 → 最高约 **18 点伤害**
- 配合多重射击 → 3 枚烟花齐发
- 范围爆炸伤害，类似火箭炮

> 烟花弩是极其强大的 PvP 武器，三枚满烟火之星烟花配合多重射击可瞬间造成巨大伤害。

## 你知道吗

1. 掠夺者使用弩时会发射**烟花火箭**庆祝袭击胜利。
2. 在恶魂农场中，弩是击杀恶魂的安全选择——预装填后在恶魂飞来时立即发射。
3. 弩在真实历史中是中世纪最具革命性的武器之一，训练时间远小于长弓。
4. 水下使用弩不会有射速惩罚（与弓不同）。
