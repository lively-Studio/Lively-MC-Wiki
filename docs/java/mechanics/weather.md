# 天气与睡眠

![Weather](https://minecraft.wiki/images/Thunder_JE4_BE3.png?format=original)

> **类别**：游戏机制 | **核心元素**：雨、雷暴、阳光、睡眠 | **影响**：生物生成、红石信号、游戏难度

Minecraft 拥有动态天气系统——阳光、降雨和雷暴三种状态之间的切换。天气不仅影响视觉效果，还直接影响游戏机制：从生物生成条件到红石信号，从村民行为到玩家安全。

---

## 天气类型

### 晴天
- 默认天气状态
- 亮度 15（正午最大）
- 僵尸和骷髅在阳光下燃烧
- 所有作物正常生长
- 村民正常工作

### 降雨
- 随机出现，持续 **0.5-1 MC 日**（10-20 分钟）
- 亮度轻微降低（最大 12）
- 以下生物会在白天生成在表面：
  - 僵尸、骷髅、女巫
  - 史莱姆（沼泽中）
- 鱼类上钩等待时间减少
- 耕地湿润（无需水源）
- 炼药锅缓慢注水

### 雷暴
- 降雨的加强版，出现概率更低
- 亮度降至 **10**（等同于夜间昏暗）
- **闪电**：随机击中地面
  - 击中猪 → 僵尸猪灵
  - 击中村民 → 女巫
  - 击中苦力怕 → **高压苦力怕**（2 倍爆炸半径）
  - 击中红石粉 → 激活
  - 击中避雷针 → 吸引周围闪电
- **三叉戟引雷**：引雷附魔的三叉戟在雷暴中必定召唤闪电

## 睡眠机制

### 床的使用
- 右键睡觉，跳过夜晚
- **必须条件**：
  - 夜晚或雷暴
  - 床上方 2 格空间
  - 附近没有怪物（Java 1.14 后可有时睡觉）
- **多人模式**：需要超过半数玩家同时睡觉

### 重生锚
- 下界专用"床"
- 合成：6 哭泣的黑曜石 + 3 荧石
- 需要用荧石充能（最多 4 级）
- 每次使用消耗 1 级
- 在主世界和末地使用 → 爆炸！

## 天气限制

### 会受天气影响的事物
| 事物 | 效果 |
|------|------|
| 末影人 | 雨中持续受到伤害 + 频繁瞬移 |
| 雪傀儡 | 雨中/炎热的生物群系中持续受伤害 |
| 烈焰人 | 雨中持续受伤害 |
| 骷髅/僵尸 | 白天不燃烧 |
| 阳光探测器 | 输出信号改变 |
| 钓竿等待时间 | 降雨中明显缩短 |

## 你知道吗

1. `/weather clear 999999` 可以让世界永远保持晴天——许多技术服务器这样禁用天气。
2. 高压苦力怕的爆炸半径是普通苦力怕的 **2 倍**——破坏力约为 TNT 的 4 倍。
3. 避雷针是 1.17 附魔洞穴与山崖更新加入的——它可以保护木质建筑不自然着火。
4. 雷暴中的闪电可以将红色哞菇变为棕色——这是获得谜之炖菜的稀有途径。
