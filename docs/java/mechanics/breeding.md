# 动物养殖与繁殖

![Breeding](https://minecraft.wiki/images/Cow_JE6_BE5.png?format=original)

> **类别**：游戏机制 | **核心资源**：动物 + 食物 | **用途**：可再生食物和材料

动物繁殖是 Minecraft 中最基础的可再生资源系统之一。通过喂食特定食物，玩家可以让两只同种动物进入"爱心模式"并生育后代，从而建立持久的产品供应链。

---

## 繁殖基本规则

### 通用规则
1. 两只同种成年动物
2. 喂食合适的繁殖食物
3. 它们进入"爱心模式"（持续 30 秒）
4. 碰在一起后，幼体出现
5. 冷却时间：约 5 分钟（Java）/ 1 分钟（BE）

### 幼体特征
- 体型较小
- 掉落物减少（或无掉落）
- 生长时间：20 分钟（约 1 个 MC 日）
- 可用该动物的食物加速生长（每份食物减少 10% 生长时间）

## 高效养殖场设计

### 基础设计
- 围墙区域（至少 8×8）
- 门口使用栅栏门
- 分开成年和幼体区
- 使用漏斗矿车收集掉落物

### 自动化方案
- **自动喂食器**：投掷器定时投放食物
- **自动屠宰**：利用熔岩或摔落伤害
- **收集系统**：漏斗 + 箱子
- 注意：幼体不能被自动屠宰

## 你知道吗

1. 骡是 Minecraft 中唯一的杂交动物——马和驴的后代，但它本身不能繁殖。
2. 鸡是唯一会主动"生蛋"的动物——鸡蛋可用于制作蛋糕和南瓜派。
3. 用繁殖食物喂养幼体可加速生长——这在短缺时非常有用。
4. 海龟是唯一在繁殖后会"返回出生地"产卵的动物——玩家可以建造海龟孵化场。
