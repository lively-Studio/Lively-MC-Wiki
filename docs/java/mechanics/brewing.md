# 酿造系统

![Brewing Stand](https://minecraft.wiki/images/Brewing_Stand_%28empty%29_JE2_BE2.png?format=original)

> **类别**：游戏机制 | **核心方块**：酿造台 | **消耗**：烈焰粉 + 水瓶 + 材料

酿造系统允许玩家制作具有各种效果的神秘药水。从治疗到隐身，从抗火到伤害增强，药水在战斗中、探险中和日常生存中都提供决定性的优势。

---

## 酿造台

### 合成配方
```
[        ][        ][        ]
[烈焰棒  ][        ][烈焰棒  ]
[        ][烈焰棒  ][        ]
```

### 界面结构
- **左侧槽**：放置水瓶（3 格）
- **顶部槽**：放入酿造材料
- **底部槽**：放入烈焰粉（燃料）
- 每份烈焰粉可酿造 **20 次**

---

## 酿造流程

### 基本流程
1. 将水瓶放入底部 3 格
2. 在顶部放入材料
3. 等待 20 秒（酿造完成）
4. 取出成品药水

### 阶段说明

| 阶段 | 输入 | 输出 | 效果 |
|------|------|------|------|
| 基础 | 水瓶 + 下界疣 | 粗制的药水 | 无效果，药水底液 |
| 效果 | 粗制药水 + 效果材料 | 效果药水 | 添加指定效果 |
| 强化 | 效果药水 + 改性材料 | 加强版药水 | 改变效果等级/时长 |

## 关键药水配方

### 核心战斗药水
| 药水 | 配方 | 持续时间 |
|------|------|---------|
| 力量 II (3:00) | 基液 + 烈焰粉 + 荧石粉 | 3:00 |
| 再生 II (1:30) | 基液 + 恶魂之泪 + 荧石粉 | 1:30 |
| 迅捷 II (1:30) | 基液 + 糖 + 荧石粉 | 1:30 |
| 抗火 (8:00) | 基液 + 岩浆膏 + 红石粉 | 8:00 |
| 治疗 II | 基液 + 闪烁的西瓜片 + 荧石粉 | 瞬间 |

### 喷溅工艺
- 基液 → 效果药水 → + **火药** → 喷溅药水
- 投掷距离约 8 格
- 爆裂范围 4 格
- 附魔效果直接作用于范围内的生物

## 你知道吗

1. 龙息是唯一一种只能从末影龙获得的酿造材料。
2. 发酵蛛眼是游戏中唯一用于"腐化"药水效果的物品——将正效果变为负效果。
3. 酿造台是最早引入 1.0 正式版的方块之一。
4. 不祥之兆 + 村庄 = 袭击事件——虽然不能通过酿造获得，但可以用牛奶清除。
