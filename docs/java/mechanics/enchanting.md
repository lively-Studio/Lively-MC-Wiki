# 附魔系统

![Enchanting Table](https://minecraft.wiki/images/Enchanting_Table_JE4_BE3.png?format=original)

> **类别**：游戏机制 | **核心方块**：附魔台、铁砧 | **消耗**：经验等级 + 青金石

附魔系统允许玩家在工具、武器和盔甲上添加特殊效果。它是游戏中后期的核心机制，显著影响玩家的战斗和采集能力。

---

## 附魔台机制

### 放置与条件
- 附魔台需要**书架**提升附魔等级
- 书架与附魔台之间**必须空 1 格**（空气或可穿过方块）
- **最多 15 个书架**有效
- 书架布置："口"字形围绕附魔台，1 格距离

### 附魔等级
附魔台有 3 个选择（由上至下）：
- 顶行：基础成本最低（1-5 级）
- 中行：中等成本（6-18 级）
- 底行：最高成本（15-30 级）
- **30 级**附魔台提供最佳附魔结果

### 附魔流程
1. 放置物品（左格）
2. 放置青金石（中格，1-3 个）
3. 选择附魔等级
4. 获得附魔物品

## 你知道吗

1. 附魔台周围的符文文字来自"标准银河字母"——与 Commander Keen 游戏中的文字相同。
2. 经验修补是唯一不能通过附魔台获得的附魔——只能通过村民交易或战利品取得。
3. 在 1.14 之前，铁砧不会因过度使用而"过于昂贵"——玩家可在获得所有附魔后再无限修复物品。
