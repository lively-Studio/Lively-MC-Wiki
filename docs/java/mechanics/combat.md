# 战斗系统

![Combat](https://minecraft.wiki/images/Combat_Update.png?format=original)

> **类别**：游戏机制 | **核心概念**：攻击冷却、横扫攻击、盾牌防御、暴击 | **版本**：Java 1.9+ 战斗更新

Minecraft 的战斗系统在 1.9 战斗更新中被完全重写。从简单的点击连打演变为基于时机的精准攻击系统，引入了攻击冷却、横扫攻击、盾牌和双持等机制。

---

## Java 版战斗机制

### 攻击冷却
- 每次攻击后，武器进入冷却状态
- 冷却期间攻击伤害为 **基础值的 20-30%**
- 冷却指示器：
  - 准星星的十字图标（默认显示）
  - 物品栏中的透明度变化
  - 快捷栏上的冷却条

### 各武器冷却时间

| 武器 | 冷却时间 | 最佳攻击间隔 |
|------|---------|-------------|
| 剑 | 0.625 秒 | 1.6 次/秒 |
| 斧 | 1.0 秒（木/石）/ 1.1 秒（铁/钻石/下界合金） | 1.0 次/秒 |
| 三叉戟 | 0.9 秒 | 1.1 次/秒 |
| 锄 | 0.25-1.0 秒 | 1.0-4.0 次/秒 |
| 空手 | 0.25 秒 | 4.0 次/秒 |

### 横扫攻击（Sweeping Edge）
- 攻击冷却 **100% 恢复**时触发
- 对正前方 3 格内的所有生物造成群体伤害
- 基础横扫伤害：1❤
- **横扫之刃 III**：横扫伤害提升至基础攻击的 75%

### 暴击（Critical Hit）
- **下落时攻击**触发暴击
- 不能在下落加速时（如使用鞘翅、飞行）
- 暴击伤害 = 基础伤害 × 1.5
- 暴击时出现星星粒子效果

## 防御机制

### 盾牌
- 右键举盾，格挡正前方攻击
- 格挡成功后：
  - 完全吸收物理伤害
  - 箭矢被弹开
  - 消耗耐久度
- **不能格挡**：
  - 爆炸伤害（苦力怕、TNT）
  - 魔法伤害（药水）
  - 声波攻击（循声守卫）
  - 穿透附魔的弩箭

### 盔甲减伤公式
每点盔甲提供 **4%** 伤害减免，上限 **80%**（20 点盔甲）。

## 基岩版战斗

基岩版使用不同的战斗系统：
- **无攻击冷却**（连续点击有效）
- 无横扫攻击
- 双持系统不同（副手只能放地图、箭、不死图腾等少数物品）
- 盾牌需要潜行键举盾

## PvP 策略

### 剑 vs 斧
| 属性 | 剑 | 斧 |
|------|----|----|
| DPS | 更高 | 较低 |
| 单次伤害 | 较低 | 更高 |
| 特殊能力 | 横扫攻击 | 破盾 |
| 最佳用途 | PvE 群怪 | PvP 盾牌对决 |

### 核心技巧
1. 保持移动——静止的玩家是弓箭的活靶子
2. 合理使用盾牌——不要一直举盾（移速减慢 50%）
3. 水晶 PvP——利用末影水晶爆炸造成伤害
4. 利用环境——高处攻击 = 暴击
5. 药水是决定性因素——力量 II = 秒杀效率翻倍

## 你知道吗

1. Java 1.9 战斗更新是 Minecraft 历史上最具争议的更新——许多 PvP 服务器至今仍运行 1.8 战斗系统。
2. 盾牌在 1.9 之前不存在——在此之前玩家只能用剑右键"格挡"（50% 伤害减免）。
3. 横扫攻击是唯一可以攻击到多个实体却不会伤害宠物和村民的近战攻击。
4. 斧可以在一击之内摧毁盾牌——这是 PvP 中的关键战术。
