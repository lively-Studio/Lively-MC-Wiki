# 工具合成配方

![Crafting Tools](https://minecraft.wiki/images/Crafting_a_Pickaxe_JE2_BE2.gif?format=original)

> **类别**：合成系统 | **核心内容**：镐、斧、锹、锄、钓竿 | **合成方式**：工作台 3×3

工具是 Minecraft 中最重要的物品类别之一。每种工具都有对应的最佳挖掘方块类型，使用正确的工具可以大幅提高效率。

---

## 镐（Pickaxe）

挖掘石头、矿石、金属方块的最基本工具。

### 合成配方
```
[材料][材料][材料]
[    ][木棍][    ]
[    ][木棍][    ]
```

### 等级数据

| 等级 | 材料 | 耐久 | 挖掘速度倍数 | 可挖掘 |
|------|------|------|-------------|--------|
| 木 | 木板 | 59 | ×1 | 石头、煤矿石 |
| 石 | 圆石 | 131 | ×1 | 铁矿、青金石矿 |
| 铁 | 铁锭 | 250 | ×1 | 钻石矿、金矿、红石矿 |
| 钻石 | 钻石 | 1561 | ×1 | 黑曜石、远古残骸 |
| 下界合金 | 下界合金锭 | 2031 | ×1 | 所有（最快） |
| 金 | 金锭 | 32 | ×1 | ⚠ 耐久极低但速度最快 |

---

## 斧（Axe）

砍伐木质方块的最佳工具。

### 合成配方
```
[材料][材料][    ]
[材料][木棍][    ]
[    ][木棍][    ]
```
*（可镜像）*

### 等级数据

| 等级 | 材料 | 耐久 | 攻击伤害（Java） |
|------|------|------|-----------------|
| 木 | 木板 | 59 | 7❤ |
| 石 | 圆石 | 131 | 9❤ |
| 铁 | 铁锭 | 250 | 9❤ |
| 钻石 | 钻石 | 1561 | 9❤ |
| 下界合金 | 下界合金锭 | 2031 | 10❤ |
| 金 | 金锭 | 32 | 7❤ |

---

## 锹（Shovel）

挖掘泥土、沙子、沙砾和雪的最佳工具。

### 合成配方
```
[材料]
[木棍]
[木棍]
```

### 特殊用途
- 右键**草方块** → 草径
- 右键**泥土** → 土径（Java 1.17+）

---

## 锄（Hoe）

将泥土/草方块变为耕地的工具。

### 合成配方
```
[材料][材料]
[    ][木棍]
[    ][木棍]
```

### 特殊用途
- 右键**泥土/草方块** → 耕地
- 右键**粗泥** → 泥土
- 1.16+ 获得专用附魔支持（效率、耐久、精准采集等）

---

## 钓竿

### 合成配方
```
[    ][    ][木棍]
[    ][木棍][线  ]
[木棍][    ][线  ]
```

### 数据
- **耐久**：64（木材质）
- 钓鱼：抛竿 + 等待鱼上钩
- 可附魔：饵钓、海之眷顾、经验修补

---

## 剪刀

### 合成配方
```
[    ]
[铁锭]
[    ][铁锭]
```
*（无定形合成）*

- 用于剪羊毛、采集树叶、破坏蜘蛛网
- 耐久：238

---

## 盾牌

### 合成配方
```
[木板][铁锭][木板]
[木板][木板][木板]
[    ][木板][    ]
```

- 格挡正面攻击（右键）
- 耐久：336
- 可用旗帜自定义图案（Java 版）
