# 合成指南

合成是 Minecraft 最核心的游戏机制。通过将原材料按照特定图案放入合成方格中，玩家可以制作出工具、武器、盔甲、食物、建材和红石设备。

## 合成系统分类

### 物品栏合成（2×2）
玩家物品栏自带的 2×2 合成格，用于制作简单的物品：
- 木板（原木 → 4 木板）
- 木棍（2 木板 → 4 木棍）
- 工作台（4 木板）

### 工作台合成（3×3）
工作台提供的 3×3 合成方格，用于制作绝大多数物品：
- 所有工具、武器、盔甲
- 所有方块压缩和建材
- 所有红石组件

### 锻造台合成
1.14 村庄与掠夺更新的新方块，用于：
- 钻石装备 → 下界合金装备升级
- 盔甲纹饰锻造

### 切石机合成
精确的石材加工：
- 单个石头加工为台阶/楼梯/墙
- 比工作台更省材料（1 个石头 = 1 个台阶 vs 3 个石头 = 6 个台阶）

### 自动合成
通过自动工作台（投掷器 + 漏斗 + 工作台）实现全自动合成（Java 版 1.20+）。

## 合成图案类型

| 类型 | 示例 | 说明 |
|------|------|------|
| 填充型 | 木板→工作台 | 特定位置必须有材料 |
| 无定形 | 蘑菇煲 | 材料在任意位置都有效 |
| 镜像型 | 斧的合成 | 左右颠倒图案同样有效 |
