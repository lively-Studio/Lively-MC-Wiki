# 盔甲合成配方

![Armor](https://minecraft.wiki/images/Diamond_Armor_JE5_BE4.png?format=original)

> **类别**：合成系统 | **核心内容**：头盔、胸甲、护腿、靴子 | **合成方式**：工作台 3×3

盔甲是 Minecraft 中最重要的防御装备。每个等级的材料提供全套 4 件装备，提供不同级别的伤害减免。

## 特殊材质盔甲

### 皮革盔甲染色
- 在炼药锅中混合染料
- 可染成超过 **1200 万种**颜色（16,777,216 种组合）
- 颜色可以混合，创造独特外观

### 下界合金升级
在锻造台中将钻石盔甲升级为下界合金：
1. 放入钻石装备
2. 放入 1 个下界合金锭
3. 获得下界合金装备（保留所有附魔）

## 你知道吗

1. 下界合金盔甲不会在熔岩中烧毁——掉入熔岩后，它会浮在表面等待拾取。
2. 皮革是最弱的盔甲材料，但皮革靴子可以在细雪上行走而不会沉入。
3. 金质盔甲是猪灵识别"友方"的标志——如果不穿金装，猪灵会攻击玩家。
4. 全套钻石盔甲提供的 80% 减伤，加上保护 IV 可以达到近 **92%** 伤害减免。
