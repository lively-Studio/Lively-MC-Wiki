# 中国版特色功能

我的世界中国版由网易代理运营，拥有大量独特的本地化功能和内容。

## 免费游玩

与国际版不同，中国版 Java 版和基岩版的客户端均完全免费下载和使用。

## 组件中心

类似于基岩版的 Marketplace，但大部分内容免费：
- **地图**：冒险、跑酷、解谜、PVP 等数千张玩家创作地图
- **皮肤**：超过数千款免费皮肤，涵盖动漫、游戏、影视 IP
- **材质包**：光影、PBR 包、简约包等多种风格
- **Add-Ons / 模组**：行为包、功能模组
- **小游戏**：内置各类小游戏组件

## 好友系统

| 功能 | 说明 |
|------|------|
| 好友列表 | 添加好友后可跨服聊天、查看在线状态 |
| 好友动态 | 查看好友的游戏成就和动态 |
| 邀请系统 | 一键邀请好友加入游戏 |
| 黑名单 | 屏蔽不良玩家 |

## 网络游戏大厅

启动器内置服务器浏览器，可直接加入各类社区服务器：
- **生存服**：纯净生存、地皮生存、空岛生存
- **创造服**：大型建筑项目
- **小游戏服**：起床战争、空岛战争、TNTRun、The Walls
- **RPG 服**：自定剧情、职业系统、副本系统
- **模组服**：科技、魔法、探险主题模组包

## 租赁服务器

网易官方提供服务器租赁服务：
- **10 人服**：约 15 元/月
- **20 人服**：约 30 元/月
- **40 人服**：约 60 元/月
- 支持模组、插件、自定义设置
- 24 小时稳定运行，无需自建服务器

## 经济系统

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| 绿宝石 | 活动、签到、任务 | 购买部分组件 |
| 钻石 | 充值（1元=10钻） | 购买高级组件、会员 |
| 会员 | 订阅制 | 专属皮肤、服务器优先 |

## 防沉迷系统

根据中国法律要求：
- 18 岁以下玩家需实名认证并受防沉迷限制
- 未成年人游戏时长限制
- 未成年人充值金额限制
- 22:00 - 次日 8:00 禁止未成年人登录

## 特殊 IP 合作

中国版与众多国内 IP 进行联动合作：
- 故宫、长城主题地图
- 三星堆考古主题
- 各类动漫 IP 皮肤
- 与教育机构合作的教育版内容
