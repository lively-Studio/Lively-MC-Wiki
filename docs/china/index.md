# 我的世界中国版

由网易代理运营的中国大陆特供版本，包含免费的 Java 版和基岩版客户端。

## 基本信息

| 项目 | 详情 |
|------|------|
| 运营方 | 网易 |
| 收费模式 | 免费下载和游玩 |
| 平台 | Windows (Java + 基岩)、iOS/Android (基岩) |
| 官网 | mc.163.com |

## 特色功能

- **免费游玩**：Java 版和基岩版均完全免费
- **组件中心**：内置大量免费地图、皮肤、模组
- **好友系统**：跨服务器好友管理
- **网络游戏**：内置社区服务器大厅
- **租赁服**：官方提供服务器租赁
- **防沉迷系统**：符合中国法律要求的实名认证和时长限制
