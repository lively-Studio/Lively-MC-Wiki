# 中国版新手指南

## 第一步：下载与安装

1. 访问官方网站 [mc.163.com](https://mc.163.com)
2. 点击"下载游戏"按钮
3. 选择对应的平台（Windows PC / iOS / Android）
4. 下载安装包并安装

## 第二步：注册账号

1. 打开网易启动器
2. 点击"注册"创建网易通行证
   - 支持手机号注册
   - 支持邮箱注册
3. 设置游戏昵称（不可更改，请慎重）
4. 完成实名认证

## 第三步：选择版本

启动器中可选择两个版本：

| 版本 | 说明 | 内存需求 |
|------|------|---------|
| Java 版 | 经典 PC 版 | 4GB+ |
| 基岩版 | 跨平台版 | 2GB+ |

> 中国版 Java 版与国际 Java 版存档兼容，可直接导入国际版存档。

## 第四步：进入游戏

### Java 版
1. 选择你想要的游戏版本（通常选择最新版）
2. 点击"启动游戏"
3. 进入游戏后可选择：
   - **单人游戏**：本地世界
   - **多人游戏**：加入服务器
   - **网络游戏**：浏览服务器大厅

### 基岩版
1. 直接点击启动
2. 登录 Xbox Live 账号（首次需要）
3. 进入游戏大厅

## 常见问题

| 问题 | 解答 |
|------|------|
| 中国版能和国际版联机吗？ | 不能，使用不同的服务器系统 |
| 能装 Forge/Fabric 吗？ | Java 版可以，基岩版不支持 |
| 能导入国际版存档吗？ | Java 版可以（.minecraft/saves），基岩版可以 |
| 存档在哪？ | 启动器内右键游戏 → 游戏目录 |
| Java 版能装模组吗？ | 可以，在组件中心安装或手动导入 |
| 基岩版能装光影吗？ | 可以，通过组件中心安装 RTX 或光影包 |
| 如何切换 Java 版本？ | 启动器 → Java 版 → 版本选择 |
| 如何更换皮肤？ | 组件中心 → 皮肤 → 选择后应用 |
