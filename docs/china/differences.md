# 与国际版差异

## 核心差异

| 方面 | 中国版 | 国际版 |
|------|--------|--------|
| 收费模式 | 免费 | Java 版收费，基岩版收费 |
| 启动器 | 网易启动器 | 官方启动器 |
| 账号 | 网易通行证 | Microsoft 账户 |
| 服务器 | 中国境内服务器 | 全球服务器 |
| 内容获取 | 组件中心 | 市场/手动安装 |

## 游戏内容差异

- **翻译**：中国版使用独立的中文本地化
- **审核**：部分用户生成内容需经审核
- **更新节奏**：版本更新通常晚于国际版数周

## 跨版本兼容

- 中国版 Java 版与国际 Java 版的存档格式兼容
- 中国版基岩版与国际基岩版存档兼容
- 无法跨区联机（中国版与国际版之间）
