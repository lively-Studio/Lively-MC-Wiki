# 启动器

中国版使用网易自主开发的启动器，与国际版完全不同。

## 功能

- **一体化管理**：在一个启动器中管理 Java 版和基岩版
- **版本选择**：可选择安装多个游戏版本
- **组件管理**：在启动器内直接安装和管理组件（地图、皮肤、模组）
- **账号管理**：网易通行证登录

## 与官方启动器的差异

| 特性 | 中国版启动器 | 官方启动器 |
|------|------------|-----------|
| 费用 | 免费 | Java 版需购买 |
| 平台支持 | Windows | Windows/Mac/Linux |
| 版本选择 | 需官方发布 | 可自由选择所有历史版本 |
| 模组 | 组件中心 | 需手动安装 |

## 下载方式

通过 [MC.163.com](https://mc.163.com) 下载启动器，注册网易通行证后即可开始游戏。
