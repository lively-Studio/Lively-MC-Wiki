# 特色功能

我的世界中国版包含多项与国际版不同的特色功能。

## 免费游玩

与国际版不同，中国版 Java 版和基岩版客户端均完全免费下载和游玩。

## 好友与联机

- **好友系统**：内置于启动器中，跨服务器添加好友
- **租用服务器**：网易官方提供服务器租赁服务
- **局域网联机**：支持本地网络联机

## 网络游戏大厅

启动器内集成了"网络游戏"，可浏览和加入由社区运营的各类服务器：
- 生存服、创造服
- 小游戏服（起床战争、空岛战争等）
- RPG 服

## 组件中心

类似于基岩版市场，提供免费的资源内容：
- 地图、玩法
- 皮肤、材质包
- 模组、Add-Ons
- 光影包

## 独特的社交功能

- **好友动态**：查看好友的游戏动态
- **新鲜事**：社区内容推送
- **开发组入驻**：部分知名服务器开发组直接入驻

## 实名认证

根据中国法规，中国版需要实名认证后才能游玩。未满 18 周岁的玩家受防沉迷系统限制。
