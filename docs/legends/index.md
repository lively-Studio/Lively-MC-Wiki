# Minecraft Legends

Minecraft Legends 是一款即时战略（RTS）游戏。玩家在方块世界中带领盟友军队对抗猪灵入侵。

## 基本信息

| 项目 | 详情 |
|------|------|
| 类型 | 即时战略 (RTS) |
| 开发商 | Mojang Studios / Blackbird Interactive |
| 首发日期 | 2023 年 4 月 18 日 |
| 平台 | Windows, Xbox, PlayStation, Nintendo Switch |
| 模式 | 战役模式 + PvP 对战 |

## 核心系统

- **指挥系统**：指挥苦力怕、僵尸、骷髅、铁傀儡等盟友单位
- **建造系统**：建造生成器、箭塔、城墙等防御工事
- **资源系统**：收集木材、石头、铁矿、钻石等资源
- **战役模式**：循序渐进的故事关卡
- **PvP 对战**：1v1 至 4v4 多人对战

## 百科分类

- **单位与生物**：主世界盟友与猪灵敌人图鉴
- **建筑与结构**：生产、防御、功能建筑图鉴
- **战役模式**：完整关卡指南
- **资源系统**：经济与资源管理
- **PvP 对战**：策略与技巧
