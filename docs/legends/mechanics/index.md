# Legends 资源

## 资源类型

| 资源 | 用途 | 来源 |
|------|------|------|
| 木材 | 基础建造、基础单位 | 树木、资源点 |
| 石头 | 中级建造、高级单位 | 岩石、资源点 |
| 铁矿 | 高级建造、精英单位 | 铁矿脉 |
| 钻石 | 终极单位、终极建筑 | 稀有矿脉 |
| 红石 | 特殊建筑和升级 | 红石矿脉 |
| 青金石 | 附魔升级 | 青金石矿脉 |

## 资源管理

1. **早期扩张**：快速占领最近的资源点
2. **平衡投入**：不要把所有资源投入单一单位类型
3. **保护采集线**：确保资源从采集点到基地的运输路线安全
4. **抢占关键资源**：铁矿和钻石决定后期胜负

## 资源箱

- 放置在资源点附近自动收集
- 可升级扩大收集范围
- 需防御保护，被摧毁后失去收集的资源
