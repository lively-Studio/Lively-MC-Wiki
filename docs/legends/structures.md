# 建筑与结构

## 生成器 (Spawners)

生成器是生产单位的关键建筑：

| 生成器 | 生产单位 |
|--------|---------|
| 苦力怕生成器 | 苦力怕 |
| 僵尸生成器 | 僵尸 |
| 骷髅生成器 | 骷髅 |
| 铁傀儡生成器 | 铁傀儡 |
| 苔藓骷髅生成器 | 苔藓骷髅 |

升级生成器可加快生产速度和单位强度。

## 防御建筑

| 建筑 | 效果 |
|------|------|
| **箭塔** | 自动攻击附近敌人 |
| **城墙** | 阻挡敌人前进 |
| **木刺陷阱** | 对经过敌人造成伤害 |
| **爆炸陷阱** | 触发后范围爆炸 |
| **治疗泉** | 范围内友方单位持续恢复 |

## 功能建筑

| 建筑 | 用途 |
|------|------|
| **资源箱** | 收集和储存资源 |
| **进阶建筑** | 解锁高级单位和建筑 |
| **桥梁** | 跨越地形障碍 |
| **传送门** | 快速旅行 |

## 村庄防御

保护盟友村庄是战役的关键目标。玩家需在村庄周围建造防御工事，防止猪灵摧毁村庄。
