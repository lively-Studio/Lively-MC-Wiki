# PvP 模式

## 模式概述

Legends PvP 是 1v1 至 4v4 的多人对战模式。双方在随机生成的地图上竞争，目标是摧毁对方的基地。

## 地图要素

- **基地**：双方各有一个主基地
- **资源点**：分布在野外的资源生成点
- **猪灵前哨**：中立的猪灵据点，击败后可获得奖励
- **高地**：有利的防御地形

## 资源经济

- **木材**：基础资源，建造和训练低成本单位
- **石头**：中级资源，建造防御建筑和高级单位
- **铁矿**：高级资源，解锁最强力的单位和升级
- **钻石**：稀有资源，终极单位和建筑

## 策略要点

1. **早期扩张**：快速占领资源点并建造防御
2. **单位搭配**：苦力怕攻城 + 骷髅防守 + 僵尸推进
3. **时机把握**：利用猪灵 Boss 出现时机突击对手
4. **情报侦察**：派遣快速单位侦察对手动向
