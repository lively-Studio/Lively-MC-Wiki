# 木桶

![Barrel](../images/blocks/barrel/7d9629b7.png)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：2.5 | **爆炸抗性**：2.5 | **发光**：否 | **透明**：否 | **可燃**：是
> **Java ID**：`barrel` | **基岩 ID**：`barrel` (458)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#7F5B2A`

木桶是 Minecraft 1.14 加入的低成本存储方块，拥有和箱子相同的 27 格容量，但**上方有实心方块时仍可打开**——这是它相比箱子的最大优势。渔夫村民的工作站点方块。

---

## 获取

### 合成
```
6 木板 + 2 木台阶
  板板板
  板  板
  板板板
```
- 木板和台阶需同种木材

### 自然生成
- **村庄渔夫小屋**：固定生成 1-3 个

### 破坏
- 任何工具均可挖掘（斧最快），掉落自身及内部物品

## 与箱子的对比

| 特性 | 木桶 | 箱子 | 陷阱箱 |
|------|------|------|--------|
| 容量 | 27 | 27 | 27 |
| 上方可被遮挡 | ✓ 是 | ✗ 否 | ✗ 否 |
| 可组合为大型容器 | ✗ | ✓（大箱子 54） | ✓（大陷阱箱 54） |
| 合成成本 | 8 木板 | 8 木板 | 箱子+绊线钩 |
| 可燃 | 是 | 是 | 是 |
| 村民工作站点 | 渔夫 | — | — |

## 你知道吗

1. 🎣 木桶和渔夫职业绑定——即使在沙漠村庄，渔夫也需要木桶。
2. 📦 木桶是唯一可以在完全被包围的情况下打开的存储方块，非常适合紧凑储藏室设计。
3. 🎭 放在音符盒下方会产生低音提琴音色。
