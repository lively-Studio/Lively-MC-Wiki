# 深色橡木

![Dark Oak Wood](https://minecraft.wiki/images/Dark_Oak_Wood_JE2_BE2.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（斧最快）
> **硬度**：2 | **爆炸抗性**：2 | **可燃**：是
> **Java ID**：`dark_oak_wood` | **地图颜色**：`#553800`

深色橡木来自深色橡树（黑森林群系），是颜色最深的木材。深色橡树只能以 **2×2 树苗**形式种植，产量极高。林地府邸大量使用深色橡木建造。

---

## 获取

- 黑森林群系（唯一来源）
- 2×2 树苗种植（单个树苗无法生长）
- 林地府邸（大量）

---

## 特性

| 特性 | 描述 |
|------|------|
| 颜色 | 深棕色（最深木材） |
| 种植 | 必须 2×2 树苗 |
| 产出 | 极高（每棵约 60-100 原木） |
| 特别适合 | 暗色建筑、中世纪风格、贵族内饰 |
