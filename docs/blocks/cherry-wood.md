# 樱花木

![Cherry Wood](https://minecraft.wiki/images/Cherry_Wood_JE1_BE1.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（斧最快）
> **硬度**：2 | **爆炸抗性**：2 | **可燃**：是
> **Java ID**：`cherry_wood` | **地图颜色**：`#C68C7C`

樱花木来自樱花树（1.20 加入），树皮为深粉褐色。樱花树林是 Minecraft 最浪漫的生物群系——粉色花瓣从树叶持续飘落。

---

## 获取

### 自然生成
- 樱花树林群系（1.20+）
- 此群系仅在山地生成

### 合成与使用
- 4 樱花原木 → 3 樱花木
- 合成樱花木板、楼梯、台阶等

---

## 特性

| 特性 | 描述 |
|------|------|
| 颜色 | 深粉褐色树皮，粉色内部 |
| 特色 | 樱花树叶持续飘落花瓣粒子 |
| 建筑风格 | 粉色系建筑、花园 |
| 蜂巢 | 樱花树生成时可能带蜂巢 |

## 你知道吗

1. 🌸 樱花树叶会持续飘落粉色花瓣粒子——是 Minecraft 最美的视觉效果之一。
2. 🐝 樱花树天然带蜂巢的概率与其他花树相同——樱花树下的蜜蜂农场特别美丽。
3. 🏔️ 樱花树林仅在山地群系生成——和草甸一样，你得爬山去找。
4. 樱花木是唯一粉色系的木材——适合少女风格建筑。
