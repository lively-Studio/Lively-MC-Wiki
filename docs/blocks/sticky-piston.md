# 黏性活塞

![Sticky Piston](https://minecraft.wiki/images/Sticky_Piston_JE4_BE4.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0.5 | **爆炸抗性**：0.5 | **发光**：否 | **透明**：是 | **可燃**：否
> **Java ID**：`sticky_piston` | **基岩 ID**：`sticky_piston` (29)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#8B6B4A`

黏性活塞是活塞的升级版，用**粘液球**涂抹活塞头部，使其激活时不仅能推出方块，信号消失后**还能拉回方块**。这是红石机械中运动控制的关键组件——从隐藏门到飞行器引擎，都离不开它。

---

## 获取

### 合成
```
1 粘液球 + 1 活塞
```
- **粘液球**：击败史莱姆掉落（沼泽或史莱姆区块）

### 破坏
- 任何工具均可挖掘，掉落自身

---

## 特性

### 拉回方块
- 激活 → 推出前方方块
- 信号消失 → **拉回方块**（普通活塞做不到）
- 拉回的方块可以是任何可推动的方块

### 1 红石刻脉冲（GC 脉冲）
- 给黏性活塞一个**极短脉冲**（1 红石刻 = 0.1 秒）
- 活塞推出方块但**来不及拉回**
- 方块永久留在新位置
- 这被称为 **"0-tick 方块的转移法"**（Java 版特性）

### 方块转移（Block Swapping）
```
1 脉冲 → 推出 A   → A 留在前方
         尝试拉回 B → 失败（因为 A 在前方挡住了）
```
- 常用于快速交换两个相邻方块

---

## 经典应用

### 隐藏门
```
黏性活塞 → 拉回墙壁方块 → 露出通道
         → 推出墙壁方块 → 隐藏通道
```
- 基地安全系统的核心

### 红石飞行器
```
侦测器 → 黏性活塞 → 侦测器 → ...
```
- 黏性活塞拉回整个结构前进
- Minecraft 飞行器的标准引擎

### 方块切换器
```
黏性活塞 A + 黏性活塞 B → 交换前方两个方块
```
- 可切换地板/天花板

### 可调节农场
```
黏性活塞 → 推出/收回水源 → 控制水流收割
```

## 与普通活塞对比

| 特性 | 黏性活塞 | 普通活塞 |
|------|----------|----------|
| 拉回方块 | ✓ | ✗ |
| 合成 | 活塞 + 粘液球 | 基础合成 |
| 推出方块 | ✓ | ✓ |
| 1 脉冲 | 方块留在新位置 | 方块留下（本来就如此） |
| 飞行器 | 必需 | 不能 |
| 隐藏结构 | ✓ | ✗ |

## 你知道吗

1. 🟢 粘液球让活塞头变成绿色——这是唯一通过合成改变外观的活塞变体。
2. ⚡ 1 红石刻脉冲是 Java 版黏性活塞的专属技巧——基岩版不具备这个行为。
3. 🚀 所有红石飞行器都必须使用黏性活塞——普通活塞无法拉回结构。
