# 音符盒

![Note Block](https://minecraft.wiki/images/Note_Block_JE4_BE3.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（斧最快）
> **硬度**：0.8 | **爆炸抗性**：0.8 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`note_block` | **基岩 ID**：`note_block` (25)
> **音符盒乐器**：多个（见下方） | **地图颜色**：`#8B6B4A`

音符盒是 Minecraft 音乐系统的核心方块，能在红石信号激活时发出乐器声音。音符盒下方的方块材质决定乐器类型——从木吉他的竖琴到沙子的沙锤，从石头的低音鼓到金块的小铃铛。2 个八度 × 16 种乐器 = 理论上可制作任何音乐。

---

## 获取

### 合成
```
8 木板 + 1 红石粉

  板板板
  板红板
  板板板
```

### 破坏
- 任何工具均可挖掘，掉落自身

---

## 使用

### 调音
- **右键**音符盒 = 升高半音（共 25 个音高 = 2 个八度）
- 每次点击音高 +1，音高轮转（F♯3 → G3 → G♯3 → ... → F♯5 → F♯3）
- 破坏并重新放置 → 重置为 F♯3

### 激活
- 红石信号激活 → 发出一次声音
- 每次激活都会发出粒子效果（音符♫）
- 音符盒上方**必须是空气**才能发出声音（被方块覆盖则静音）

---

## 乐器类型

音符盒下方的方块决定乐器音色：

| 下方方块 | 乐器 | 音色描述 |
|----------|------|----------|
| 木板、工作台、箱子等木质方块 | 竖琴/钢琴 | 清亮柔和的拨弦音 |
| 圆石、石头、矿石、黑曜石 | 低音鼓 | 低沉有力的鼓声 |
| 沙子、沙砾、混凝土粉末 | 沙锤 | 沙沙作响 |
| 玻璃、荧石、海晶灯 | 叮当/钟琴 | 清脆的敲击声 |
| 黏土块、蜂巢 | 长笛 | 悠扬的吹奏音 |
| 羊毛 | 吉他 | 柔和的弹拨 |
| 冰块 | 钟琴 | 冰冷清脆 |
| 金块 | 铃铛/小钟 | 悦耳的金属铃声 |
| 骨块 | 木琴 | 轻快的打击乐 |
| 灵魂沙 | 牛铃 | 沉闷的金属撞击 |
| 铁块 | 颤音琴 | 持续震动的金属音 |
| 南瓜 | 迪吉里杜管 | 低沉的嗡鸣 |
| 绿宝石块 | 电子音/方块音 | 8-bit 芯片音 |
| 干草块 | 班卓琴 | 乡村风格的弹拨 |

## 红石特性

- 比较器读取音高（0~24 = 音符编号）
- 可实现自动演奏序列

## 你知道吗

1. 🎵 音符盒在 Beta 1.2 加入，是 Minecraft 最古老的音乐方块。
2. 🎹 音符盒有完整的 2 个八度 + 一个半音（F♯3 到 F♯5），共 25 个音高。
3. 🥁 基岩版有独有的 16 种乐器（Java 版某些乐器不同）。
4. 🎶 玩家用音符盒制作过《致爱丽丝》、《荒野乱斗》主题曲乃至整首《波西米亚狂想曲》！
5. 🐄 音符盒的声音可以被鱿鱼听到——会使鱿鱼上升（虽然没什么实际用途）。
