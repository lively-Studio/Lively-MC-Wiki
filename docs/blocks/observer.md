# 侦测器

![Observer](https://minecraft.wiki/images/Observer_JE4_BE3.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：3 | **爆炸抗性**：3 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`observer` | **基岩 ID**：`observer` (251)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#545454`

侦测器是 Minecraft 的"方块更新监测器"，能检测前方方块的任何状态变化（放置、破坏、作物生长、红石信号变化等），并输出一个短暂的**红石脉冲**。它是自动化农场、飞行器引擎和精密红石电路的核心组件。

---

## 获取

### 合成
```
6 圆石 + 2 红石粉 + 1 下界石英

  圆圆圆
  红红石
  圆圆圆
```
- 下界石英需从下界石英矿石获得

### 破坏
- 用镐挖掘 → 掉落自身
- 不用镐 → 无掉落

---

## 工作原理

### 侦测面
```
┌──────────┐
│  ⊕  脸  │  ← 侦测面（监测前方变化）
└──────────┘
    ↓
   输出面 → 红石信号输出
```
- 侦测面（脸）：监测该方向的方块更新
- 输出面（背面红点）：输出红石信号

### 检测内容
| 类别 | 示例 |
|------|------|
| 方块放置/破坏 | 任何方块 |
| 方块状态变化 | 门开/关、水流方向、活塞伸缩 |
| 作物生长 | 小麦、胡萝卜等阶段变化 |
| 红石信号变化 | 红石粉信号、中继器 |
| 容器状态 | 箱子打开、酿造台完成 |
| 音符盒 | 调音变化 |

---

## 输出特性

- **强充能脉冲**：持续 **2 红石刻（0.1 秒）**
- 输出信号强度固定为 15
- Java 版：侦测器本身变化不算（防止自激）
- 基岩版：1 红石刻（0.05 秒）

---

## 经典应用

### 作物自动收割
```
侦测器（监测作物生长）→ 活塞（收割作物）
```

### 飞行器引擎
```
侦测器 → 黏性活塞 → 侦测器 → ...
```
- Minecraft 飞行器的标准推进引擎

### 高频时钟
```
侦测器 → 红石 → 侦测器前
```
- 每 2 红石刻触发一次

### 自动甘蔗/竹子农场
```
侦测器 → 活塞 → 甘蔗/竹子
```
- 作物长高时自动收割

## 你知道吗

1. 👁️ 侦测器的脸部纹理看起来像一张人脸——这并非巧合，Mojang 故意设计了一个"惊恐表情"。
2. 🚀 侦测器是 Minecraft 飞行器技术的核心——没有它就没有可自我推进的红石飞行器。
3. ⚡ 侦测器的 0.1 秒脉冲是红石电路中最短的信号之一，常用于精密计时。
4. 🎃 把侦测器放在南瓜前方 → 侦测器像戴了南瓜头一样（视觉玩笑）。
