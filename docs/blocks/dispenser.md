# 发射器

![Dispenser](https://minecraft.wiki/images/Dispenser_JE3_BE3.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：3.5 | **爆炸抗性**：3.5 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`dispenser` | **基岩 ID**：`dispenser` (23)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#545454`

发射器是投掷器的升级版，不仅能**投出物品**，还能**激活特定物品的特殊功能**——射箭、发射火焰弹、放置水/熔岩、生成生物、装备盔甲等。它是红石自动化的多功能核心组件。

---

## 获取

### 合成
```
7 圆石 + 1 弓 + 1 红石粉

  圆圆圆
  圆弓圆
  圆红圆
```
- 弓必须**未损坏**（满耐久）

### 破坏
- 用镐挖掘 → 掉落自身及内部物品
- 不用镐 → 无掉落

---

## 功能

### 投掷模式（默认）
- 红线信号激活时，**随机投出**内部 1 个物品
- 物品以实体形式抛出（类似玩家按 Q）

### 激活模式（特殊物品）
发射器对特殊物品有独特行为：

| 物品 | 行为 |
|------|------|
| 箭/光灵箭/药箭 | 射出（伤害约 3❤） |
| 火焰弹 | 发射小火球 |
| 雪球/鸡蛋 | 投掷 |
| 喷溅/滞留药水 | 投掷药水 |
| 水桶 | **放置水源** |
| 熔岩桶 | **放置熔岩源** |
| 打火石 | 点燃前方方块 |
| TNT | **放置并点燃 TNT** |
| 船 | 放置船（前方有水时） |
| 盔甲 | 给正前方实体装备 |
| 生物头颅 | 给正前方实体装备 |
| 生物刷怪蛋 | **生成生物** |
| 骨粉 | 催熟作物/植物 |

| 射入容器 | ✗ | ✓ |

---

## 红石行为

- 每次红石脉冲 → 激活 1 次（投出/激活 1 个物品）
- 高速脉冲（如侦测器时钟）→ 连续激活
- 激活时口部短暂张开动画

## 你知道吗

1. 🏹 发射器射出的箭伤害低于玩家射箭——因为拉弓蓄力有伤害加成。
2. 🔥 发射器 + 火焰弹 = 远程火焰攻击系统，是陷阱的经典选择。
3. 💧 发射器可以无限次放置/收回水和熔岩——不需要补充。
4. 🎯 发射器射出的箭不受重力影响，直线飞行。
