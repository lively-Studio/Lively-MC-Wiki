# 红石粉

![Redstone Dust](https://minecraft.wiki/images/Redstone_Dust_JE2_BE2.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0 | **爆炸抗性**：0 | **发光**：是（动态） | **透明**：是 | **可燃**：否
> **Java ID**：`redstone_wire` | **基岩 ID**：`redstone_wire` (55)
> **地图颜色**：`#FF0000`

红石粉是 Minecraft 电子电路的"电线"，能将红石信号从一个位置传输到另一个位置。它是整个红石系统的基础——从简单的按钮+门到复杂的计算机，一切红色电路都始于红石粉。

---

## 获取

### 破坏红石矿石
- 开采红石矿石 → **4-5 个红石粉**（时运 III 最多 8 个）
- 深层红石矿石也会掉落

### 其他来源
- **女巫**死亡掉落：0-6 个
- 丛林神庙：15 个红石粉（陷阱中）
- 村庄牧师交易：1 绿宝石 → 2 红石粉

---

## 信号传输

### 基本规则
- 红石粉铺设在地面形成"红石线"
- 信号强度从源头 **15 → 0** 每格递减 1
- 最远传输 **15 格**后信号消失
- 爬墙和下墙也需要另外放置红石粉

### 连接形状
| 形状 | 信号方向 |
|------|----------|
| 直线 | 双向传输 |
| L 形拐角 | 双向传输 |
| T 形 | 三个方向 |
| 十字形 | 四个方向 |
| 点状（孤立） | 不传输 |

### 信号强度
```
源头(15) → ○(14) → ○(13) → ... → ○(1) → ○(0)
```

---

## 充能与激活

### 强充能（Hard Power）
- 红石粉向**它指向**的实心方块输出信号
- 被充能的方块可以：
  - 激活相邻红石粉
  - 激活红石元件（中继器、比较器）
  - 激活相邻门/活板门/TNT

### 弱充能（Soft Power）
- 红石粉只充能它**直接连接**的方块
- 不充能相邻方块

---

## 应用

### 基础电路
```
按钮 → 红石粉 → 红石灯
```
- 最简单的电路

### 延长传输
```
红石粉(15格) → 中继器 → 红石粉(15格) → ...
```
- 中继器刷新信号为 15

### 逻辑门
- 红石粉 + 红石火把 → NOT 门（非门）
- 多路红石粉汇聚 → OR 门（或门）

## 你知道吗

1. 🔴 红石是 Minecraft 中专有名词的缩写——由 Red（红）和 Stone（石）组成，但实际上并不来源于红色石头。
2. ⚡ 红石粉会在激活时**闪烁发光**（亮度动态变化，0~7），形成可视化信号流。
3. 🧮 红石系统是图灵完备的——理论上你可以在 Minecraft 中建造任何通用计算机。
4. 📡 红石粉在 Java 版 1.0.0 之前叫 "Redstone"，1.0 后改为 "Redstone Dust"。
