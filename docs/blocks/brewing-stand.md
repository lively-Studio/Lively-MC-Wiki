# 酿造台

![Brewing Stand](../images/blocks/brewing-stand/160a1a02.png)

> **稀有度**：不常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何
> **硬度**：0.5 | **爆炸抗性**：0.5 | **亮度**：1 | **透明**：否 | **可燃**：否
> **Java ID**：`brewing_stand` | **基岩 ID**：`brewing_stand` (117)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#545454`

酿造台是 Minecraft **药水系统的核心**，允许玩家通过烈焰粉提供燃料，将水瓶与各种材料结合，酿造出功能各异的神奇药水。从基础的粗制药水到隐身药水，药水系统是游戏中后期战斗和探索的基石。

---

## 获取

### 合成
```
3 圆石 + 1 烈焰棒
```
- **烈焰棒**：击败烈焰人掉落（需先到达下界要塞）

### 自然生成
- **雪屋地下室**：固定生成 1 个酿造台（带 1 瓶虚弱喷溅药水）
- **末地船**：船内的酿造室生成（带 2 瓶治疗药水 II）
- **村庄教堂**：牧师村民的工作站点

### 破坏
- 任何工具均可挖掘，掉落自身（保留内部物品）

## 酿造流程

### GUI 界面
```
┌─────────────────┐
│ [烈焰粉槽]       │  ← 燃料（每个烈焰粉 = 20 次）
│                 │
│ [ 材料槽  ]      │  ← 酿造材料
│                 │
│ [瓶1][瓶2][瓶3] │  ← 药水槽（3 个）
└─────────────────┘
```

### 基础流程
1. **放置水瓶** → 底部 3 个槽（玻璃瓶 + 水 = 水瓶）
2. **添加下界疣** → 顶部材料槽
3. 酿造 → **粗制药水**
4. **添加效果材料** → 各类功能药水
5. 可选：红石粉（延长）/ 荧石粉（增强）/ 火药（喷溅）/ 龙息（滞留）

## 红石信号

- 酿造台可通过比较器输出红石信号
- 信号强度 = 0~7（按药水槽中的水瓶数量 + 酿造进度计算）
- 可用于**自动酿造系统**

---

## 燃料

- **烈焰粉**：每个 = 20 次酿造操作
- 燃料槽最多堆叠 64 个烈焰粉（= 1280 次）
- 燃料是自动消耗的，无需手动填装每次

## 你知道吗

1. 🧪 雪屋地下室是唯一自然生成**已酿造的药水**的地方（虚弱喷溅药水），用于治疗僵尸村民。
2. 🔴 酿造台是少数自身**发光的方块**（亮度等级 1），在暗处可以看到它的微弱光芒。
3. 🤖 Java 版中，酿造台是唯一使用 **NBT 标签记录配方数据**的方块（记录了所有已发现的药水配方）。
4. ⏱️ 一次酿造周期为 **20 秒**（400 游戏刻），同时作用于底部所有 3 个药水槽。
