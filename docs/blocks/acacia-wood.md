# 金合欢木

![Acacia Wood](https://minecraft.wiki/images/Acacia_Wood_JE3_BE2.png?format=original)

> **稀有度**：常见 | **可再生**：是 | **可堆叠**：64 | **工具**：任何（斧最快）
> **硬度**：2 | **爆炸抗性**：2 | **可燃**：是
> **Java ID**：`acacia_wood` | **地图颜色**：`#AB6122`

金合欢木来自金合欢树（热带草原），树皮为独特的橙灰色，是所有木材中颜色最"温暖"的。金合欢树姿态奇特——树干弯曲分枝。

---

## 获取

- 热带草原、热带高原群系
- 村庄（热带草原变体）

---

## 特性

| 特性 | 描述 |
|------|------|
| 颜色 | 橙灰色树皮，内部橙红色 |
| 树形 | 不规则弯曲，多分枝 |
| 特别适合 | 暖色调建筑、非洲风格 |

## 你知道吗

- 🌅 金合欢木的橙色调让它在热带草原日落背景下格外美丽。
- 🏜️ 热带草原村庄大量使用金合欢木建造——整个村庄暖色调统一。
