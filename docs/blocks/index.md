# 方块

Minecraft 中的方块是组成游戏世界的基本单元。每个方块都有独特的属性——硬度、爆炸抗性、可再生性、透明度和采掘工具要求。

## 石头与矿物

### 基础石类
- [石头](stone.md) —— 主世界最丰富的固体方块，刷石机无限可再生
- [圆石](cobblestone.md) —— 挖掘石头掉落，建筑基石
- [花岗岩](granite.md) / [闪长岩](diorite.md) / [安山岩](andesite.md) —— 三类火成岩变体
- [深板岩](deepslate.md) —— Y=0 以下替代石头，硬度更高
- [凝灰岩](tuff.md) / [黑石](blackstone.md) / [玄武岩](basalt.md) —— 下界和深层地质

## 土壤与植被

- [草方块](grass-block.md) —— 主世界地表之"肤"，会向泥土蔓延
- [泥土](dirt.md) —— 最基础的土壤方块，农业基石
- [砂土](coarse-dirt.md) —— 草不会蔓延，适合道路
- [灰化土](podzol.md) —— 巨木针叶林特有，蘑菇在任何光照下生长
- [菌丝体](mycelium.md) —— 蘑菇岛专属，紫色孢子
- [缠根泥土](rooted-dirt.md) —— 苍郁洞穴杜鹃树下方
- [泥巴](mud.md) —— 1.19 沼泽方块，可脱水为黏土
- [沙子](sand.md) / [红沙](red-sand.md) —— 受重力影响，沙漠/恶地专属
- [沙砾](gravel.md) —— 受重力影响，掉落燧石
- [黏土](clay.md) —— 水底生成，破坏掉黏土球
