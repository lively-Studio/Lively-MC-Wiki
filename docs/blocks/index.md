# 方块

Minecraft 中的方块是组成游戏世界的基本单元。下面列出所有主要方块，点击名称查看详情。

## 分类
- **石头类**：石头、圆石、花岗岩、闪长岩、安山岩、深板岩、凝灰岩、黑石、玄武岩
- **矿物类**：煤矿石、铜矿石、铁矿石、金矿石、红石矿石、青金石矿石、钻石矿石、绿宝石矿石、下界石英矿石、远古残骸
- **土壤植被**：草方块、泥土、砂土、灰化土、菌丝体、缠根泥土、泥巴、沙子、红沙、沙砾、黏土
- **木质方块**：橡木、云杉木、白桦木、丛林木、金合欢木、深色橡木、红树木、樱花木、绯红菌柄、诡异菌柄、竹块
- **建筑方块**：砖块、石砖、混凝土、染色陶瓦、石英块、海晶石、末地石砖
- **功能性方块**：工作台、熔炉、高炉、烟熏炉、附魔台、铁砧、砂轮、切石机、锻造台、酿造台、箱子、木桶、潜影盒、末影箱
- **红石方块**：红石粉、红石火把、红石块、红石中继器、红石比较器、侦测器、活塞、黏性活塞、投掷器、发射器、漏斗、音符盒
- **特殊方块**：信标、潮涌核心、重生锚、磁石、避雷针
- **光源**：火把、灯笼、荧石、海晶灯、营火、南瓜灯
