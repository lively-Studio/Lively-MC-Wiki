# 深板岩

![Deepslate](../images/blocks/deepslate/0ad6413f.png)

> **稀有度**：常见 | **可再生**：否 | **可堆叠**：64 | **工具**：木镐或更好
> **硬度**：3 | **爆炸抗性**：6 | **发光**：否 | **透明**：否 | **可燃**：否
> **Java ID**：`deepslate` | **基岩 ID**：`deepslate` (数字ID: `-636`)
> **音符盒乐器**：低音鼓 | **地图颜色**：`#646464`

深板岩是 1.18 版本（洞穴与山崖第二部分）加入的深层岩石，取代了 Y=0 以下的石头。它标志着 Minecraft 地下世界的重大变革——更深、更暗、更危险。深板岩的硬度是石头的两倍，挖掘时间也更长。其深黑灰色调和板状纹理让深层洞穴充满了压抑的地质氛围。

---

## 获取方式

### 自然生成
深板岩在 Y=0 以下大量取代石头：
- **过渡带**：Y=8 到 Y=0，深板岩逐渐增多
- **深层区域**：Y=0 以下完全由深板岩构成（直到 Y=-64 基岩层）
- 深层矿石全部被**深板岩变体的矿石**取代（如深层钻石矿石）
- 化石在深层会替换为深板岩骨块

### 采集
- 用镐（无精准采集）→ 掉落**深板岩圆石**
- 用带**精准采集**附魔的镐 → 掉落**深板岩本身**
- 不用镐 → 无掉落
- 挖掘时间是石头的**两倍**

### 烧炼
- 深板岩圆石 + 燃料 → 深板岩

## 用途

### 合成（使用深板岩圆石）
- 深板岩圆石 → 石质工具（属性与石质工具相同）
- 4 深板岩圆石 → 磨制深板岩
- 4 深板岩圆石 → 深板岩砖
- 4 深板岩圆石（十字）→ 深板岩瓦
- 1 深板岩圆石（烧炼）→ 深板岩

### 切石机加工
深板岩及其变体可通过切石机加工为全套建筑方块：

| 原料 | 可加工为 |
|------|----------|
| 深板岩圆石 | 磨制深板岩、深板岩砖、深板岩瓦、錾制深板岩 |
| 磨制深板岩 | 深板岩砖、深板岩瓦、台阶、楼梯、墙 |
| 深板岩砖 | 裂纹深板岩砖、台阶、楼梯、墙 |
| 深板岩瓦 | 裂纹深板岩瓦、台阶、楼梯、墙 |

### 深板岩变体一览
- **深板岩** — 基础原料（精准采集获取）
- **深板岩圆石** — 最常见形式（普通镐掉落）
- **磨制深板岩** — 光滑表面
- **深板岩砖** → **裂纹深板岩砖**（烧炼）
- **深板岩瓦** → **裂纹深板岩瓦**（烧炼）
- **錾制深板岩** — 装饰用
- 上述所有变体的台阶、楼梯、墙

## 你知道吗

1. 🏔️ 深板岩最初命名为 "Grimstone"（阴森石），这个名字非常贴切地描述了深层洞穴的氛围——但 Mojang 后来觉得 "Deepslate" 更专业。
2. ⚒️ 深板岩挖掘时间是石头的两倍，这让深层挖矿变得更具挑战性。配合深层洞穴的巨大开放空间，早期探险尤为危险。
3. 🦇 深暗之域（Deep Dark）生成于深板岩层，这意味着从 Y=0 以下就开始进入潜在的监守者领地。
4. 🧱 深板岩是目前拥有**最多建筑变体的方块之一**：圆石、磨制、砖、瓦、錾制，以及各自的台阶/楼梯/墙——共计超过 20 种变体。
5. 🔄 深板岩圆石制作的工具与石质工具属性完全相同，仅外观不同——是纯粹的外观选择。
