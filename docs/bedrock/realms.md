# 领域 (Realms)

领域是 Mojang 官方提供的服务器托管服务，支持轻松创建和管理多人世界。

## 类型

### Realms（个人领域）
- 同时支持最多 3 名玩家（含主控）
- 提供 3 个世界存档位
- 仅好友可加入

### Realms Plus（高级领域）
- 同时支持最多 11 名玩家（含主控）
- 包含市场高级内容库（50+ 市场包）
- 定期轮换内容

## 功能

- **跨平台**：Java 版 Realms 仅 Java 玩家，基岩版 Realms 支持所有基岩平台
- **自动备份**：服务器自动定期备份世界
- **小游戏**：提供官方小游戏世界
- **邀请系统**：通过好友列表或邀请链接加入

## 定价

按月订阅制，Java 版与基岩版价格不同。基岩版 Realms Plus 价格高于普通 Realms。
