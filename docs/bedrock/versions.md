# 版本历史

基岩版版本历史从携带版（Pocket Edition）开始，经历了多次名称变更和版本统一。

## 版本演进

| 时代 | 版本范围 | 说明 |
|------|---------|------|
| Pocket Edition | 0.1.0 ~ 1.1.5 | 最初为移动设备专属 |
| 基岩版统一 | 1.2.0 | Better Together 更新，统一代码库 |
| 正式版 | 1.2+ | 跨平台统一版本号 |
