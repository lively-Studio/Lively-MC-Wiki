# 基岩版百科

## 与 Java 版的主要差异

- 跨平台多人游戏（好友系统、领域）
- 市场（Marketplace）支持地图/皮肤/模组下载
- 不同的红石机制和刷怪规则
- 部分方块和物品特性不同
- 支持 Add-Ons 系统（行为包 + 资源包）
