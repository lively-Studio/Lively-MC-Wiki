# 与 Java 版差异

基岩版与 Java 版虽然共享大部分内容，但存在许多重要差异。

## 红石机制

- **活塞行为**：基岩版活塞可推动箱子、熔炉等容器方块
- **半连接性**：基岩版没有 Java 版的 QC（Quasi-Connectivity）特性
- **更新顺序**：红石组件更新顺序在基岩版中具有不确定性（locality-based）

## 生物与刷怪

- **刷怪上限**：基岩版的地域刷怪上限、全局上限与 Java 版不同
- **生物行为**：部分生物 AI 存在差异，如村民繁殖规则不同
- **生物密度上限**：基岩版洞穴生物上限为 8，地表为 8；Java 版为 70

## 方块与物品

- **侦测器**：基岩版侦测器可检测更多方块状态变化
- **炼药锅**：基岩版炼药锅可储存药水、染料水，Java 版仅能存水（1.17+ 才支持细雪）
- **骨粉**：基岩版骨粉使用行为不同，如可催熟甘蔗

## 战斗系统

- 基岩版没有 Java 版 1.9 的攻击冷却机制
- 盾牌需蹲下才能激活（而非右键）

## 世界生成

- 基岩版种子生成算法与 Java 版完全不同
- 部分结构的生成位置和概率存在差异
