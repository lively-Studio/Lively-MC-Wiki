# 基岩版游戏机制

## 红石系统

基岩版红石与 Java 版有显著差异，以下为基岩版独立规则：

### 活塞行为
- 基岩版活塞**可以推动**容器方块（箱子、熔炉、发射器、投掷器）
- 基岩版活塞可推动**方向锁定**的方块（如侦测器）
- 基岩版没有 QC（Quasi-Connectivity）特性
- 活塞无法推动活塞（与 Java 版不同）

### 红石更新
- **更新顺序**：基岩版采用基于位置（locality-based）的更新顺序
- 同一 tick 内，不同位置的红石组件更新顺序具有不确定性
- 设计红石电路时不应依赖特定更新顺序

### 侦测器与比较器
- 基岩版侦测器可检测更多方块状态变化
- 比较器可检测炼药锅水位

## 战斗系统

| 特性 | 基岩版 | 说明 |
|------|--------|------|
| 攻击冷却 | 无 | 可连续快速攻击 |
| 盾牌激活 | 蹲下 (潜行) | 非右键触发 |
| 横扫攻击 | 无 | 剑没有范围攻击 |
| 斧伤害 | 较高 | 斧的伤害值高于 Java 版 |

## 命令系统

### 基岩版独有命令
| 命令 | 功能 |
|------|------|
| `/ability` | 设置玩家能力（飞行、建造等） |
| `/camerashake` | 屏幕震动效果 |
| `/event` | 触发实体事件 |
| `/music` | 控制背景音乐 |
| `/playanimation` | 播放实体动画 |
| `/structure` | 结构方块相关操作 |
| `/tickingarea` | 常加载区块 |
| `/gametest` | GameTest 框架 |

### 指令差异
- 不支持 `/scoreboard`（Java 版计分板）
- 不支持 NBT 数据标签
- `/execute` 子命令系统受限
- 目标选择器使用 `family` 而非 `type`

## 世界生成

- **种子**：与 Java 版完全不兼容
- **结构位置**：相同种子在两版本中生成结构位置不同
- **生物群系布局**：生物群系分布算法不同
- **洞穴生成**：基岩版洞穴分支更多、更密集

## 多人游戏

| 方式 | 说明 |
|------|------|
| 局域网 | 同一网络直接联机 |
| Xbox Live 好友 | 跨平台好友联机 |
| 领域 (Realms) | Mojang 官方服务器 |
| 领域 Plus | 包含市场内容库 |
| 第三方服务器 | 通过 IP 地址加入 |

## Add-Ons 系统

基岩版通过 Add-Ons 实现模组功能：
- **行为包 (Behavior Pack)**：JSON 定义实体、战利品、合成
- **资源包 (Resource Pack)**：纹理、音效、模型、UI
- **脚本 API**：1.19+ 支持 JavaScript 编程
- **市场发布**：认证创作者可在市场销售 Add-Ons
