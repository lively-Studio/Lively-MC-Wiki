# 市场 (Marketplace)

基岩版市场是内置在游戏中的数字商店，允许玩家购买由社区创作者制作的地图、皮肤、模组、材质包等内容。

## 货币

- **Minecoin**：市场专属虚拟货币
- 可通过真实货币购买 Minecoin 包
- 部分内容为免费下载

## 内容类型

| 类型 | 说明 |
|------|------|
| **世界** | 冒险地图、跑酷、小游戏等 |
| **皮肤包** | 角色皮肤、自定义外观 |
| **材质包** | 改变游戏材质风格 |
| **模组** | 通过 Add-Ons 系统添加新内容 |
| **混搭包** | 包含皮肤、材质、世界的综合包 |

## 创作者计划

市场内容由 Mojang 认证的创作者合作伙伴制作和发布，创作者可获得收入分成。

## 跨平台

购买的内容绑定到 Microsoft 账户，可在所有已登录该账户的设备上下载使用。
