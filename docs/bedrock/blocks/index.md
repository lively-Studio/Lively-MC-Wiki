## 炼药锅

基岩版炼药锅比 Java 版功能丰富：
- 储水（Java 版仅能储水，1.17+ 支持细雪）
- 储存药水（基岩版独有）
- 储存染料水（基岩版独有）
- 染色皮革盔甲
- 装填炼药锅（雨水/水瓶）
