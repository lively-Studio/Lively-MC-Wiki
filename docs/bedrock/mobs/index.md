## 基岩版刷怪差异

- **刷怪上限**：基岩版地域上限 8（地表）/ 8（洞穴），远低于 Java 版 70
- **村民繁殖**：需要足够数量的床，与 Java 版规则不同
- **铁傀儡生成**：需 20 张床 + 10 个村民
- **凋灵**：基岩版凋灵生命值固定 300（非随难度变化），且会召唤凋灵骷髅，有冲刺攻击
