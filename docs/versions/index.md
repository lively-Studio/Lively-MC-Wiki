# 版本历史

## 你知道吗

1. 版本号 `1.0` 跳过了 `Alpha 1.2.6` 和 `Beta 1.8`——Minecraft 在长达 2 年的公开测试后才正式发布。
2. 1.9 是第一个被玩家社区严重抗议的版本——导致了多年的 PvP 服务器分流。
3. 1.17/1.18 是第一个被拆分为两部分的更新——因为世界生成代码太复杂。
4. 基岩版最初是手机版（Pocket Edition），后在 1.2 版本统一了代码库。
