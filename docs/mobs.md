# 生物

生物（Mob）是 Minecraft 中具有自主行为能力的实体。截至最新版本，Minecraft 共有超过 80 种不同的生物。

## 友好生物

友好生物不会主动攻击玩家，即使玩家攻击它们也不会反击（狼除外）。

### 凋灵战斗策略

1. 建议在地下深处或末地召唤
2. 第一阶段：凋灵飞行并用凋灵之首轰炸
3. 第二阶段（半血以下）：获得护甲，免疫箭矢，近战破坏方块
4. 推荐装备：下界合金盔甲（保护 IV）、力量 V 弓、治疗 II 箭

### 末影龙战斗策略

1. 破坏黑曜石柱上的末影水晶以阻止龙回血
2. 使用弓射龙（建议力量 V，无限）
3. 当龙栖息在返回传送门上时近战攻击
4. 推荐装备：钻石/下界合金盔甲、缓降药水、大量食物和弓箭
