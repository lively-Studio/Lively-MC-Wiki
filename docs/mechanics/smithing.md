# 锻造系统

![Smithing Table](https://minecraft.wiki/images/Smithing_Table_JE4_BE3.png?format=original)

> **类别**：游戏机制 | **核心方块**：锻造台 | **用途**：装备升级、盔甲纹饰

锻造台是 1.14 村庄与掠夺更新引入的村民职业方块，后在 1.16 下界更新中获得核心功能——将钻石装备升级为下界合金装备。1.20 更新后又加入了盔甲纹饰锻造系统。

---

## 合成配方

```
[铁锭][铁锭]
[木板][木板]
```

---

## 锻造台界面

锻造台界面有 3 个槽位：
1. **左侧槽**：放入需要升级的装备（钻石盔甲/工具/武器）
2. **中间槽**：放入升级材料（下界合金锭）
3. **右侧槽**：输出升级后的下界合金装备

**关键优势**：升级**保留所有附魔、耐久度和自定义名称**！

## 下界合金升级

### 可升级物品清单
| 钻石装备 | → 下界合金装备 | 保留 |
|---------|---------------|------|
| 钻石剑 | 下界合金剑 | 附魔、名称 |
| 钻石镐 | 下界合金镐 | 附魔、名称 |
| 钻石斧 | 下界合金斧 | 附魔、名称 |
| 钻石锹 | 下界合金锹 | 附魔、名称 |
| 钻石锄 | 下界合金锄 | 附魔、名称 |
| 钻石头盔 | 下界合金头盔 | 附魔、名称 |
| 钻石胸甲 | 下界合金胸甲 | 附魔、名称 |
| 钻石护腿 | 下界合金护腿 | 附魔、名称 |
| 钻石靴子 | 下界合金靴子 | 附魔、名称 |

### 升级成本
- 每件装备：**1 个下界合金锭**
- 下界合金锭 = 4 个下界合金碎片 + 4 个金锭
- 远古残骸 → 熔炼 → 下界合金碎片

## 盔甲纹饰（1.20+）

### 纹饰材料
| 材料 | 纹饰颜色 | 材质效果 |
|------|---------|---------|
| 铁锭 | 浅灰 | 金属色 |
| 铜锭 | 橙色 | 铜色 |
| 金锭 | 金色 | 金色 |
| 青金石 | 深蓝 | 蓝色 |
| 绿宝石 | 绿色 | 绿色 |
| 钻石 | 浅蓝 | 闪亮钻石 |
| 下界合金锭 | 黑色 | 暗色 |
| 紫水晶碎片 | 紫色 | 紫色 |
| 红石粉 | 红色 | 红色 |
| 石英 | 白色 | 白色 |

### 纹饰模板
- 共有 **16 种**不同图案的纹饰模板
- 模板可在各地结构中找到（堡垒遗迹、末地城、远古城市等）
- 一次锻造消耗 1 个模板 + 1 份材料
- 模板可复制：模板 + 7 颗钻石 + 对应方块

## 你知道吗

1. 下界合金装备不会在熔岩中烧毁——掉入熔岩后会浮在表面等待拾取。
2. 锻造台最初只是工具匠村民的职业方块——没有任何实际功能。
3. 盔甲纹饰系统在 Minecraft Live 2022 中被首次展示，在当时引起了玩家社区的热烈反应。
4. 静音盔甲纹饰模板（Silence）是唯一以"沉默"为主题的纹饰——在远古城市中找到。
