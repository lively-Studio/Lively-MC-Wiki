# 进度系统

![Advancements](https://minecraft.wiki/images/Advancements_%28Java_Edition%29.png?format=original)

> **类别**：游戏机制 | **平台**：Java 版（进度）/ 基岩版（成就） | **用途**：引导游戏进程、记录成就

进度是 Minecraft Java 版的成就系统，它不仅是完成任务的标记，更是引导新玩家逐步了解游戏机制的教程。与基岩版的成就不同，进度可以通过数据包自定义，允许服务器或模组创建自己的进度树。

## 进度架构

### 标签页（5 个）

| 标签页 | 图标 | 主题 |
|--------|------|------|
| Minecraft | 草方块 | 游戏核心体验 |
| 下界 | 下界岩 | 下界探险 |
| 末地 | 末地石 | 末地挑战 |
| 冒险 | 空地图 | 探索与战斗 |
| 农牧业 | 胡萝卜 | 耕种与食物 |

### 进度类型
- **普通进度**（黄色弹窗）
- **目标进度**（黄色弹窗，与挑战类似）
- **挑战进度**（紫色弹窗，更困难的目标）

## 基岩版成就

基岩版的成就系统是独立的，与 Xbox/PlayStation 成就联动：
- 共约 **130 个**成就
- 支持 **Gamerscore** 积分
- 每个成就给予 10-100 分
- 较 Java 版进度更侧重于引导

## 你知道吗

1. "探索的时光"需要探索全部 42 个生物群系——这是最耗时的进度/成就之一。
2. "资深怪物猎人"要求玩家击杀约 **23 种不同的敌对生物**（含凋灵和末影龙）。
3. 自定义数据包可以创造任意进度——包括跨维度的链条目标。
4. 进度中的"来硬的"(Stone Age)英文原意是石器时代的双关。
