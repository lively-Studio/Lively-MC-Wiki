# Lively Minecraft Wiki

欢迎来到 **Lively MC WIKI**！这里是一个关于 Minecraft 的综合知识库。

## 什么是 Minecraft？

Minecraft 是由 Mojang Studios 开发的一款沙盒游戏，由 Markus "Notch" Persson 用 Java 语言创建。游戏于 2009 年 5 月 17 日作为首个公开版本发布，并于 2011 年 11 月 18 日正式发布。

在 Minecraft 中，玩家在一个由方块组成的 3D 世界中探索、建造、采集资源、战斗并生存。

## 游戏模式

| 模式 | 描述 |
|------|------|
| 生存模式 | 管理生命值与饥饿值，采集资源，建造和探索 |
| 创造模式 | 无限资源，可以飞行，方块瞬间破坏 |
| 冒险模式 | 类似生存模式，但方块需要正确工具才能破坏 |
| 旁观模式 | 可以飞越方块并观察世界，不可交互 |
| 极限模式 | 生存模式的变体，只有一条命，难度锁定为困难 |

## 游戏维度

### 主世界
- Minecraft 的主要维度，玩家出生于此
- 拥有丰富多样的生物群系：平原、森林、沙漠、丛林、海洋等
- 白天/黑夜循环，昼夜各约 10 分钟

### 下界
- 通过下界传送门进入的火热维度
- 坐标比例为 1:8（下界 1 格 = 主世界 8 格）
- 特色生物群系：下界荒地、绯红森林、诡异森林、灵魂沙峡谷、玄武岩三角洲

### 末地
- 通过要塞中的末地传送门进入
- 末影龙 Boss 战斗的场所
- 击败末影龙后可探索末地外岛，获取潜影贝、紫颂果和鞘翅
