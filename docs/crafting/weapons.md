# 武器合成配方

![Weapons](https://minecraft.wiki/images/Enchanted_Netherite_Sword_JE2_BE2.png?format=original)

> **类别**：合成系统 | **核心内容**：剑、弓、弩、三叉戟 | **合成方式**：工作台 3×3

武器是 Minecraft 生存的核心。从基础的木剑到下界合金剑，从木制弓到附魔三叉戟，选择正确的武器并熟悉其特性是生存的关键。

---

## 剑（Sword）

近战攻击武器，也是采集蜘蛛网的唯一工具。

### 合成配方
```
[材料]
[材料]
[木棍]
```

### 等级数据

| 等级 | 材料 | 耐久 | 攻击伤害 | 攻击速度 |
|------|------|------|---------|---------|
| 木 | 木板 | 59 | 4❤ | 1.6 |
| 石 | 圆石 | 131 | 5❤ | 1.6 |
| 铁 | 铁锭 | 250 | 6❤ | 1.6 |
| 钻石 | 钻石 | 1561 | 7❤ | 1.6 |
| 下界合金 | 下界合金锭 | 2031 | 8❤ | 1.6 |
| 金 | 金锭 | 32 | 4❤ | 1.6 |

### 横扫攻击（Java 版 1.9+）
- 攻击冷却 200% 时触发
- 对目标周围所有生物造成 1❤ 伤害
- 受横扫之刃附魔影响（最高 III）

## 弓（Bow）

远程攻击武器。

### 合成配方
```
[    ][木棍][线  ]
[木棍][    ][线  ]
[    ][木棍][线  ]
```

### 数据
- **耐久**：384
- **拉弓时间**：1 秒（满蓄力）
- **满蓄力伤害**：6❤（普通）/ 9❤（暴击）
- 拉弓时移动速度降低 50%

### 推荐附魔
| 附魔 | 效果 |
|------|------|
| 力量 V | 每级 +25% 伤害 |
| 无限 I | 无限普通箭（需要至少 1 支箭在物品栏） |
| 火矢 I | 点燃目标 5 秒 |
| 冲击 II | 击退距离 +8 格 |
| 经验修补 I | 用经验修复耐久 |

---

## 弩（Crossbow）

可预装填的远程武器。

### 合成配方
```
[木棍][铁锭][木棍]
[线  ][绊线钩][线]
[    ][木棍][    ]
```

### 数据
- **耐久**：326（Java）/ 464（BE）
- **装填时间**：1.25 秒
- **伤害**：6❤（普通）/ 9❤（暴击）
- 装填后可以存储，随时射击，不需要拉弓

### 特殊弹药
- **烟花火箭**：发射后造成爆炸伤害（最多 18❤）
- **穿透 IV 弩**：箭可穿透 4 个实体
- **多重射击 I**：发射 3 支箭

---

## 三叉戟

近战和远程两用武器。

### 获得方式
- **不能合成**——只能从**溺尸**掉落（8.5% 概率）
- 抢夺附魔可提高掉落率

### 数据
- **耐久**：250
- **近战伤害**：9❤
- **投掷伤害**：8❤
- 投掷后必须拾回才能再次使用

### 推荐附魔
| 附魔 | 效果 |
|------|------|
| 忠诚 III | 投掷后自动返回手中 |
| 激流 III | 在水中/雨中投掷 → 推动玩家前进（与忠诚/引雷冲突） |
| 引雷 I | 雷暴中击中生物 → 召唤闪电 |
| 穿刺 V | 对水生生物 +12.5❤ 伤害 |

## 你知道吗

1. 金剑的耐久度是所有工具中最差的——仅 32 次使用，但达到同级工具中最快的攻击速度。
2. 三叉戟是唯一不能通过合成获得的武器——必须在海洋中狩猎溺尸。
3. 弩发射烟花火箭在 Java 1.15 之后成为最受 PvP 欢迎的武器之一。
4. 盾牌在 Java 1.9 之前并不存在——在此之前，玩家只能用剑格挡。
