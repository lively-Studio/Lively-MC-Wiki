# 酿造工具与原料

药水酿造系统需要专门的设备和材料。以下是酿造所需的所有设备合成和关键材料。

---

## 核心设备

### 酿造台
```
[空]   [空]   [空]
[烈焰棒] [圆石] [烈焰棒]
[圆石] [圆石] [圆石]
```
需要：1 个烈焰棒 + 3 个圆石

- 燃料槽放入**烈焰粉**（1 根烈焰棒 → 2 个烈焰粉）
- 每次酿造消耗 1 份烈焰粉（约 20 次酿造量）
- 最多同时酿造 3 瓶

### 炼药锅
```
[空]   [空]   [空]
[铁锭] [空]   [铁锭]
[铁锭] [铁锭] [铁锭]
```
需要：7 个铁锭

- 储存水（最多 3 层，即 3 瓶水）
- 基岩版中可直接炼药锅 + 材料 → 药水
- 盛接雨水
- 清除盔甲染色和旗帜图案

### 玻璃瓶
```
[空]   [玻璃] [空]
[玻璃] [空]   [玻璃]
[空]   [玻璃] [空]
```
需要：3 个玻璃 → 3 个玻璃瓶

## 酿造材料

| 材料 | 获取方式 | 用途 |
|------|---------|------|
| 下界疣 | 下界要塞生长，灵魂沙上种植 | 制作粗制药水（基础） |
| 红石粉 | 红石矿石掉落或女巫掉落 | 延长药水持续时间 |
| 荧石粉 | 荧石或女巫掉落 | 提升药水效果等级 |
| 火药 | 苦力怕凋落 | 药水 → 喷溅药水 |
| 龙息 | 末影龙吐息收集 | 喷溅药水 → 滞留药水 |
| 发酵蛛眼 | 蜘蛛眼 + 糖 + 棕色蘑菇 | 腐化药水效果（正面 → 负面） |

## 你知道吗

1. 酿造台是 Minecraft 中唯一要求下界材料的制作方块——必须先进入下界获取烈焰棒。
2. 一个烈焰棒提供的燃料（2 个烈焰粉）可以完成约 20 次酿造操作。
3. 在基岩版中，炼药锅的酿造功能比 Java 版更方便——不需要水瓶，直接炼药锅 + 材料即可。
