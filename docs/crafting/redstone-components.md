# 红石组件合成

红石组件是 Minecraft 红石系统的基础模块。从简单开关到复杂计算机——都从这些基本组件开始构建。

---

## 信号源（输入）

| 组件 | 配方 | 说明 |
|------|------|------|
| 拉杆 | 1 木棍 + 1 圆石 | 持续开/关信号 |
| 按钮（石质） | 1 石头 | 1 秒脉冲（1 红石刻 = 0.1 秒） |
| 按钮（木质） | 1 木板 | 1.5 秒脉冲（也可被箭矢触发） |
| 压力板（石质） | 2 石头 | 仅实体触发 |
| 压力板（木质） | 2 木板 | 实体或掉落物触发 |
| 红石块 | 9 红石粉 | 持续 15 级信号 |
| 红石火把 | 1 红石粉 + 1 木棍 | 可反向（输入 → 熄灭） |
| 阳光探测器 | 3 玻璃 + 3 下界石英 + 3 木板 | 亮度 → 信号强度 |
| 侦测器 | 6 圆石 + 2 红石粉 + 1 下界石英 | 检测方块更新（脉冲输出） |

---

## 传输元件

| 组件 | 配方 | 说明 |
|------|------|------|
| 红石粉 | 红石矿石 + 精准采集（或烧炼） | 传输信号最多 15 格 |
| 红石中继器 | 2 红石火把 + 1 红石粉 + 3 石头 | 延长/延迟信号（1-4 红石刻） |
| 红石比较器 | 3 红石火把 + 1 下界石英 + 3 石头 | 比较/检测容器/减法模式 |

---

## 机械元件（输出）

| 组件 | 配方 | 说明 |
|------|------|------|
| 活塞 | 3 木板 + 4 圆石 + 1 铁锭 + 1 红石粉 | 推动最多 12 个方块 |
| 黏性活塞 | 1 活塞 + 1 黏液球 | 可拉回 1 个方块 |
| 发射器 | 7 圆石 + 1 弓 + 1 红石粉 | 使用物品效果（射箭、放水） |
| 投掷器 | 7 圆石 + 1 红石粉 | 向容器或世界投掷物品 |
| 漏斗 | 5 铁锭 + 1 箱子 | 收集/传输物品 |
| 红石灯 | 4 红石粉 + 1 荧石 | 接收信号时发光 |
| 音符盒 | 8 木板 + 1 红石粉 | 播放音阶声 |
| 铁门 | 6 铁锭（2×3） | 仅红石信号可打开 |
| 铁活板门 | 4 铁锭（2×2） | 仅红石信号可打开 |

## 你知道吗

1. 红石中继器是第一个可调节延迟的组件——1 红石刻 = 0.1 秒 = 2 游戏刻。
2. 侦测器在 1.11 探险更新中才被加入——在此之前玩家用 BUD（方块更新检测器）的 bug 来检测方块更新。
3. 漏斗是 Minecraft 最复杂的机械元件——它既是容器（5 格物品槽）也是传输器。
