# 游戏机制

Minecraft 拥有丰富的游戏机制，包括红石系统、附魔、酿造、战斗等。

## 红石系统

红石是 Minecraft 中的"电力"系统，能实现从简单开关到复杂计算机的各种装置。

### 红石基础

- **红石信号**：可以在红石粉中传输，最大距离 15 格
- **信号强度**：0 到 15，越远越弱（每格衰减 1）
- **充能状态**：方块可被弱充能或强充能
- **红石刻**：1 红石刻 = 0.1 秒 = 2 游戏刻

### 信号源

| 组件 | 输出方式 | 说明 |
|------|---------|------|
| 拉杆 | 持续开/关 | 最常用的手动开关 |
| 按钮 | 短暂脉冲 | 木质 1.5 秒，石质 1 秒 |
| 压力板 | 实体触发 | 木质可被物品触发，石质仅实体 |
| 红石块 | 持续强信号 | 15 级强信号输出 |
| 红石火把 | 持续信号 | 可反向（输入 → 熄灭） |
| 阳光探测器 | 随亮度变化 | 可切换到夜间模式 |
| 侦测器 | 方块更新脉冲 | 2 红石刻脉冲输出 |
| 音符盒 | 按下时输出 | 短暂激活下方方块 |
| 绊线钩 | 实体经过触发 | 需要线连接两个绊线钩 |

### 传输元件

| 组件 | 特点 |
|------|------|
| 红石粉 | 传输信号 15 格，可转向 |
| 红石中继器 | 延长信号至 15 格，可设置 1-4 红石刻延迟 |
| 红石比较器 | 比较/检测容器状态/减法模式 |
| 充能方块 | 被充能的方块可以中继信号 |

### 机械元件

| 组件 | 功能 |
|------|------|
| 活塞 | 推动最多 12 个方块 |
| 黏性活塞 | 可拉回 1 个方块 |
| 投掷器 | 向容器或世界投掷物品 |
| 发射器 | 使用物品效果（射箭、放水等） |
| 漏斗 | 收集和传输物品 |
| 红石灯 | 接收信号时发光 |
| TNT | 被红石激活后 40 红石刻爆炸 |
| 音符盒 | 发出音阶声 |
| 铁门/活板门 | 仅红石信号可打开 |
| 命令方块 | 执行命令（创造模式/管理员） |

### 常见红石电路

#### T 触发器（Toggle）

将按钮变为开关：每次脉冲切换输出状态。

基本方案：
- 黏性活塞 + 侦测器方案
- 投掷器 + 漏斗 + 比较器方案

#### 脉冲发生器

将持续信号变为短脉冲。常用侦测器或上升沿/下降沿检测器。

#### 脉冲延长器

将短脉冲延长。使用 2 个中继器互相锁定。

#### 时钟电路

产生循环脉冲。方案有：
- 2 个中继器环路（最简单）
- 漏斗时钟（可调节间隔）
- 比较器衰减时钟

## 附魔系统

### 附魔台

- 使用 30 级经验获取最佳附魔
- 附魔台周围放置书架（最多 15 个）以提升附魔等级
- 青金石每次消耗 1-3 个
- 书架与附魔台之间必须空一格

### 附魔兼容性

**冲突附魔**（不可共存）：
- 锋利 / 亡灵杀手 / 节肢杀手
- 保护 / 爆炸保护 / 火焰保护 / 弹射物保护
- 精准采集 / 时运
- 无限 / 经验修补
- 激流 / 忠诚 / 引雷
- 多重射击 / 穿透

### 铁砧机制

- 修复物品：材料 + 被修复物品
- 合并附魔书：附魔书 + 物品
- 重命名：始终消耗 1 级经验
- 经验消耗随操作次数增加（最多累计 39 级）
- "过于昂贵"限制：超过 39 级则无法操作

## 战斗机制

### 攻击冷却

Java 版 1.9+ 引入攻击冷却机制：
- 攻击后有冷却时间，满冷却（200%）造成全额伤害
- 剑冷却：0.625 秒；斧冷却：1.0-1.25 秒
- 横扫攻击（剑）：满冷却时对周围生物造成伤害

### 暴击

跳跃下落过程中攻击造成暴击（+50% 伤害），有暴击粒子效果。

### 伤害类型

| 类型 | 说明 | 防护附魔 |
|------|------|---------|
| 近战 | 直接攻击 | 保护 |
| 弹射物 | 弓/弩/恶魂火球 | 弹射物保护 |
| 爆炸 | TNT/苦力怕/床 | 爆炸保护 |
| 火焰 | 火/岩浆/烈焰人 | 火焰保护 |
| 魔法 | 药水/龙息/唤魔者 | 保护 |
| 摔落 | 从高处掉落 | 摔落缓冲 |
| 窒息 | 方块内窒息 | - |
| 溺水 | 水下缺氧 | 水下呼吸 |
| 凋零 | 凋灵/凋零玫瑰 | 保护 |
| 虚空 | Y=-64 以下 | - |

### 不死图腾

副手持不死图腾可防止死亡，触发后恢复 1 生命值，获得 40 秒生命恢复 II 和 5 秒伤害吸收 II。
